Rename or Delete an XS Advanced Application

Rename or delete an XS advanced application.

Prerequisites

To access the Application Monitor tool, a user requires the authorization scopes defined in the roles grouped together in one of the following role collections:

  • XS_CONTROLLER_ADMIN

    Full access: no access restrictions

  • XS_CONTROLLER_USER

    Authorized to perform all tasks within an assigned organization

  • XS_CONTROLLER_AUDITOR

    Authorized to only view the details in an organization or space

Tip Role collections can be assigned to an SAP HANA user in SAP HANA studio by means of user parameters, for example, XS_RC_XS_CONTROLLER_ADMIN or XS_RC_XS_CONTROLLER_USER, or XS_RC_XS_CONTROLLER_AUDITOR.

Context

You can use the XS advanced administration tools to change the name of an application running in the XS advanced run-time environment or delete the application completely. It is also possible to delete particular instances of an application, for example, instances that have stopped or crashed.

To rename or delete an XS advanced application, perform the following steps:

Procedure

  1. Start the SAP HANA cockpit.
  2. Locate the XS Advanced Administration tools catalog.
  3. Start the Administration and Monitoring tools.
  4. Start the Application Monitor tool.
  5. Display a list of the applications running in the XS advanced run-time environment.
  6. Locate the application you want to manage and choose the desired action.

    In the Actions column of the list of available applications, choose more options (the list icon) and select one of the following actions:

    • Rename

      You will need to enter the new name of the application in the Rename Application dialog and choose OK.

    • Delete

      The Delete action requires user interaction; choose Yes to confirm deletion of the selected XS advanced application.

  7. Refresh the content of the application list.
    Tip You can enable the Auto Refresh feature to ensure that the latest application status is displayed.
