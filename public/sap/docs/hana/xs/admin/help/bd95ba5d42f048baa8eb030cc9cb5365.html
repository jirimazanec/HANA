Managing Hosts in XS Advanced

View and maintain system hosts in XS advanced.

The XS advanced administrator tools include a Host Management tool that enables you to perform the following tasks:

  • View a list of the SAP HANA hosts that are pinned to XS advanced applications or spaces

  • View the XS advanced applications to which SAP HANA hosts are pinned

  • View the spaces to which SAP HANA hosts are pinned

Note You can also search the list of pinned hosts for a specific host name.
