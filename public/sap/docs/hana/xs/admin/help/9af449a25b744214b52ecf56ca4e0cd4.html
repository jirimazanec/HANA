Database Configuration Details in XS Advanced

A list and description of the details required to configure a logical database in XS advanced.

You can use SAP HANA Logical Database Setup tool to prepare a new logical database for use with XS advanced. The following table indicates the information displayed for the logical database

Table 1: Logical Database Details

UI Element

Description

Example

Name

The name of the logical database.

TenantDB1

MyDatabaseName

Enable for XS Advanced

The status of the logical database, for example, Enable.
  • Disable

    The selected logical database is being used with XS advanced; choose Disable to make the logical database unavailable for use with XS advanced.

  • Enable

    The selected logical database is not yet ready for use with XS advanced; choose Enable to make the logical database available for use with XS advanced.

Status

The current status of the logical database.

Running

Delete

Use this option to delete a logical database.
