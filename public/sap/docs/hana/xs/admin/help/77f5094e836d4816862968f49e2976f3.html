Managing Trusted Certificates in XS Advanced

View and maintain certificates used for trusted connections in XS advanced.

The XS advanced administrator tools include a Trusted Certificates tool that enables you to perform the following tasks:

  • Display a list of the certificates that XS advanced uses for trusted connections

  • Display details of a specific XS advanced trusted certificate

  • Add a certificate to the list of trusted XS advanced certificates

  • Remove a certificate from the list of trusted XS advanced certificates
