Delete an SAML Identity Provider

Remove an SAML Identity Provider (IDP) from the list of available identity providers.

Prerequisites

To access the SAML Identity Provider Configuration tool, you require the authorization scopes defined in the roles grouped together in one of the following role collections:

  • XS_AUTHORIZATION_ADMIN

    Full access: no access restrictions

  • XS_AUTHORIZATION_DISPLAY

    Read-only access SAML Identity Provider Configuration tool

Context

You can delete an SAML Identity Provider from the IDP list by using the SAML Identity Provider Configuration tool.

Procedure

  1. Start the SAP HANA cockpit.
  2. Locate the XS Advanced Administration tools catalog.
  3. Start the Administration and Monitoring tools.
  4. Start the SAML Identity Provider Configuration tool.
  5. Choose the SAML IDP.

    In the SAML Identity Provider List, choose the SAML IDP that you want to remove from the list of identity providers.

  6. Remove the SAML IDP.

    Choose Delete to remove the SAMP IDP.
