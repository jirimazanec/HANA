Create a Logical Database in XS Advanced

Creating a logical database for use in SAP HANA XS advanced.

Prerequisites

To create a new logical database, you require the authorization scope defined in the following role collection:
  • XS_CONTROLLER_ADMIN

    Full access: no access restrictions

Context

You create a logical database for usage of applications developed in XS advanced.

Procedure

  1. Start the SAP HANA cockpit.
  2. Locate the XS Advanced Administration tools catalog.
  3. Start the Administration and Monitoring tools.
  4. Start the SAP HANA Logical Database Setup tool.
  5. Choose New to display the New Logical Database dialog.
  6. Enter the details and choose OK.
  7. In the System Database Credential dialog, enter the logon credentials for the database user “System”.

Results

The new logical database is created with the status Creating. Eventually, the status automatically changes to Running or Not Running.
