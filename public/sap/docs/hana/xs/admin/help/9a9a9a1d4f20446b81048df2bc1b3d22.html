Monitoring the SAP HANA XS Advanced Model Run-time Environment

Monitor how system resources are used by the applications running in the XS advanced model run-time environment.

The Application Monitor enables you to view the system resources that are being consumed by the individual application instances running in the SAP HANA XS advanced model run-time environment. For example, you can see how much memory is allocated and how long the application has been running for.
