Delete a Logical Database

Remove a logical database.

Prerequisites

To delete a logical database, you require the authorization scope defined in the following role collection:
  • XS_CONTROLLER_ADMIN

    Full access: no access restrictions

Context

You can delete a logical database using the SAP HANA Logical Database Setup tool.

Procedure

  1. Start the SAP HANA cockpit.
  2. Locate the XS Advanced Administration tools catalog.
  3. Start the Administration and Monitoring tools.
  4. Start the SAP HANA Logical Database Setup tool.
  5. Locate the Logical Database that you want to remove.
  6. In the Delete column, choose Delete.

Results

The logical database is deleted.
