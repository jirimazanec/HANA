Sort and Group Audit Logs

Sort and group the viewed audit logs by different criteria.

Prerequisites

Access to the XS Advanced Admin Tools Audit Log UI is only granted to those users who are assigned the required role collection. For more details, see Related Information.

Procedure

  1. Start the SAP HANA cockpit.
  2. Locate the XS Advanced Administration tools catalog.
  3. Start the Administration and Monitoring tools.
  4. Start the Audit Logs tool.

    The UI shows the history of all audit logs.

  5. Choose (Settings).
    • To sort your audit logs by specific criteria, choose the Sort tab.
    • To view the audit logs per group, select the respective group.
