Trusted Certificates Details in XS Advanced

Display a list of the certificates used to establish trusted connections between XS advanced hosts.

Trusted Certificates Overview

The Trusted Certificates tool displays a list of the certificates that have been configured and exchanged and includes the following information:

Table 1: Trusted Certificate Overview

UI Element

Description

Example

Alias

A name representing the trusted certificate

myCert

Usage

The type of use for which the certificate is intended, for example, “HTTP” or “JDBC”. To set up trust with the SAP HANA database, use JDBC. The JDBC certificate establishes a secure and encrypted communication channel between SAP HANA XS applications and the SAP HANA database.

HTTP

Issuer

Details about the authority that issued the named certificate

Valid Until

The date and time at which the certificate expires
