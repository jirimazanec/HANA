SAP HANA XS Advanced Administration Tools

SAP HANA XS advanced model includes a Web-based administration tool that enables you to maintain important elements of the XS advanced application-development environment, for example, security and authentication methods.

The XS Advanced Administration tools include a selection of Web-based applications that enable you to configure and maintain the basic administration-related elements of the application-development process for the XS advanced run-time environment. The tools included in XS Advanced Administration cover the tasks and areas listed in the following table:

Table 1: Administration Tools for the XS Advanced Run-Time Environment
Tool Name Description Scope

Application Monitor

Monitor the system resources used by the applications running in the XS advanced model run time environment.
  • Management of application security
  • Monitoring application use of system resources

Organization and Space Management

Create, list, or delete user organizations and spaces in the XS advanced model run time.
  • Management of organizations
  • Management of spaces
  • Management of users in organizations and spaces
  • Management of XS advanced business-user roles for organizations and spaces

SAML Identity Provider

Configure an SAML identity provider for use by XS advanced applications that need to authenticate the XS advanced business users signing in by means of SSO.
  • Management of SAML identity-providers, including IDP meta data, certificates, and destinations

Application Role Builder

Maintain and manage roles and role collections for XS advanced business users.
  • Management of XS advanced business user role collections
  • Creation of roles and role collections

User Management

Maintain and manage database users for SAP HANA XS advanced
  • Creation of XS advanced business users

SAP HANA Logical Database Setup

Manage, maintain, and configure SAP HANA logical databases for use with SAP HANA XS advanced model applications.

  • Management of logical databases in XS advanced
  • Creation of logical databases in XS advanced
  • Preparation of logical databases in XS advanced

SAP HANA Service Broker Configuration

Manage mapping of SAP HANA logical databases to an organization or a space.
  • Mapping of logical databases to an organization or a space in XS advanced

Host Management

Manage the SAP HANA hosts that are pinned to SAP HANA XS advanced applications or spaces.
  • Displaying a list of the SAP HANA hosts that are pinned to XS advanced applications or spaces

Trusted Certificates

Manage and maintain the certificates used to establish secure and trusted connections between SAP HANA systems and SAP HANA XS advanced applications.
  • Adding certificates
  • Displaying a list of certificates and details

Job Scheduler Service Dashboard *

Create, schedule, and manage long-running operations jobs in the SAP HANA XS advanced model run-time environment.
  • Enabling of job scheduler services
  • Monitoring of job-schedule status
  • Display and maintenance of job schedule's run-time configuration
  • Creation of schedules for (or deletion from) an XS job
