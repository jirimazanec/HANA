Enable a Logical Database for use with XS Advanced

Step-by-step instructions for enabling a logical database for use in SAP HANA XS advanced.

Context

To make a logical database available for use in XS advanced, you must enable the logical database, for example, using the SAP HANA Logical Database Setup tool, as described in the following steps:

Procedure

  1. Start the SAP HANA cockpit.
  2. Locate the XS Advanced Administration tools catalog.
  3. Start the Administration and Monitoring tools.
  4. Start the SAP HANA Logical Database Setup tool.
  5. Locate the logical database that you want to “prepare” for use in XS advanced.
  6. In the Enable for XS Advanced column, choose Enable.

    The Database Credential dialog appears requesting logon credentials for the logical database.

  7. Enter a password for the logical database.
  8. During the enabling of the logical database, the status Enabling is displayed. When the logical database is available for use in XS advanced, in the Enable for XS Advanced column you will see Disable.

Results

The selected logical database is “enabled” and available for use in XS advanced.
