Logical Database Details

Defines logical database details

You can define details of a logical database using XS advanced administration tools.
Table 1: Logical Database Details

UI Element

Description

Name

Name of the logical database

Internal Port
This is an optional field. A specific port can be set for the logical database. The port must adhere to anyone of the following rules:
  • Accepts a value range and meets the condition (<port number> - (30000+40)) % 3 == 0. For example, the range can vary from 30040 to 30100.
    Note The port value (3<instance number>00) varies with the instance number. If the instance number is 08, the port value will be 30800. Confirm the instance number on your system and accordingly provide the port details.
  • Accepts port number of the default index server

Host

This is an optional field. If the logical database is required to be hosted on a specific system then provide the host details.

Logical Database Password

Type a password to access the new logical database. The password must contain a minimum of eight alphanumeric characters.

Confirm Password

Type the password again for confirmation.
