Host Management Details

Display an overview of the hosts where XS advanced applications are running and, if required, more specific details of the applications and spaces pinned to a selected host.

The Host Management tool displays information about the following areas:

  • Host Management Overview

    Overview of the all managed XS advanced hosts

  • General

    Overview of the details available for a selected host

  • Applications

    A list of XS advanced applications pinned to the selected host

  • Spaces

    A list of the spaces to which a host is pinned

Host Management Overview

The Host Management tool displays an overview of the hosts that have been pinned to XS advanced applications or spaces and includes the following information:

Table 1: General Host Details

UI Element

Description

Example

Host ID

Display all information by application in alphabetic order, starting with the letter “A”

host.acme.com

Exclusive Pin

Indicates (Yes/No) if the host-pinning is exclusive to the pinned applications or spaces. If the host is set to exclusive pin, only the pinned applications and spaces will be deployed on the host.

No

Pinned Applications

Indicates if any applications have been pinned to the selected host

<application name>

Pinned Spaces

Indicates if any the selected host is pinned to any spaces.

<space name>
Tip For more information about which applications and spaces are pinned to a particular host, click the host name or choose > for a selected host.

Applications

The Applications tab in the XS Advanced administration Host Management tool displays the following information:

Table 3: General Host Management Details

UI Element

Description

Name

The name of the application that is pinned to a specific host

Pinned Host

The name of the host to which the XS advanced application is pinned. If the pin type is “strict”, the host must be available for the application to run.

Pin Type

The type of pin configured for the application, for example:

  • “Strict”

    Application can only run on the pinned host

  • “Relaxed”

    Application can run on another host, if the pinned host is not available
Tip Use the Application Monitor tool to pin XS advanced applications to a specific host, and set the pinning type.
