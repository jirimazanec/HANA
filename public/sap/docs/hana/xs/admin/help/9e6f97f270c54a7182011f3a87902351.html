Filter Audit Logs

Use the filters of the Audit Log UI to see specific audit logs.

Prerequisites

Access to the XS Advanced Admin Tools Audit Log UI is only granted to those users who have the required role collection assigned. For more details, see Related Information.

Context

You can filter audit logs by the following criteria:

  • Time

    By using the time criteria, you can view audit logs before, after, and between specific times and days. For example, you can view the audit logs between October 10, 2016, 9 a.m. and October 20, 2016, 6 p.m.

  • Org

    Set filters for the name of the org.

  • Space

    Set filters for the name of the space, which is located in an org.

  • App

    Set filters for the name of the application, which is scoped to a space.

  • User

    Set filters for the name of the user that logs on to the application.

  • Category

    Select a category that corresponds to the audit log event.

Procedure

  1. Start the SAP HANA cockpit.
  2. Locate the XS Advanced Administration tools catalog.
  3. Start the Administration and Monitoring tools.
  4. Start the Audit Logs tool.

    The UI shows the history of all audit logs.

  5. If necessary, specify the fields to filter by.

    By default all fields are displayed, but you can customize the filters by choosing  Filters  Change Filters .

  6. Specify the criteria for your filtering by populating the appropriate filter fields with the desired information.

    You can use the fields either on the Audit Logs page or on the Filters dialog.

    The Audit Log UI displays only those audit logs allowed by the current filtering.
