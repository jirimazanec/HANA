hdbalm dependencies Command

Use this command to display and analyze dependencies of SAP HANA delivery units.

Use the following syntax for the dependencies command:
hdbalm [<general options>] dependencies [<command option>]* [<source du>] [<source du vendor>] [<target du>] [<target du vendor>]
Command options
The following command options exist:
Command Option Description
-f, --full Shows the full dependency view and analysis.
-r, --references Shows object references between delivery units.

This command requires the name and vendor of the source and the target delivery units.
-n, --nirvana Shows nirvana references for a delivery unit.

This option displays object references for objects that are part of a delivery unit to objects which are not part of a delivery unit. If a delivery unit contains objects with these references, it cannot be imported into another system.
Note If you do not specify any command options, a list of delivery unit dependencies is displayed. Each line of the output lists a delivery unit followed by a colon and a comma-separated list of referenced delivery units.
Example To display the dependencies of the delivery unit HANA_XS_LM, you use the following command:
hdbalm dependencies HANA_XS_LM sap.com
The delivery unit HANA_XS_LM has references to the delivery units SAPUI5_1 and HANA_XS_BASE. The output appears as follows:
HANA_XS_LM(sap.com): SAPUI5_1(sap.com), HANA_XS_BASE(sap.com)
