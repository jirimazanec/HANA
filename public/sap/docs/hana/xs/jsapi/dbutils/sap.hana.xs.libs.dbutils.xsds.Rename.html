Class: Rename

.xsds. Rename

new Rename()

This class provides various predefined renaming functions for mapping column names to field names.

Methods

<static> $camelcase()

Convert name to camel case.
Example
$camelcase("XS_data__service") => "XsDataService"

<static> $id()

Identity function; leave name as-is.

<static> $lowercase()

Convert name to lower case.
Example
$lowercase("XS_Data__Service") => "xs_data__service"

<static> $normalcase()

Capitalize first character of name only.
Example
$normalcase("xs_Data__Service") => "Xs_data__service"

<static> $uppercase()

Convert name to upper case.
Example
$uppercase("XS_Data__Service") => "XS_DATA__SERVICE"
