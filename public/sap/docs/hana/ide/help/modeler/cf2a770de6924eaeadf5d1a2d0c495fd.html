Orphan Nodes

For orphan nodes in a hierarchy, SAP HANA modeler provides different options to handle them. For example, you can treat orphan nodes as root nodes or treat them as errors.

Options

Description

Root Node

Treat orphan nodes as root nodes.

Error

Stop processing and hierarchy and show an error.

Ignore

Ignore orphan nodes.

Step Parent

Put orphan nodes under a step parent node.
