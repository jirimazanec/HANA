Replace a Data Source in Calculation Views

Replace a data source in a calculation view with another data source in the catalog object without performing a delete operation, and retain the semantic information of the changed node.

Context

For example, in the below calculation view, if you want to replace the data source, Projection_1 of Union_1 with another data source in the catalog object, then you can do it by executing the below procedure.

Procedure

  1. Open required calculation view in view editor.
  2. Select a data source you want to replace.
  3. In the context menu of the data source, choose Replace With Data Source.
  4. In the Find Data Sources dialog, enter the name of the new data source.
  5. Select the data source from the list.
  6. Choose Next.
  7. Manage the source and target mappings.
  8. Choose Finish.
