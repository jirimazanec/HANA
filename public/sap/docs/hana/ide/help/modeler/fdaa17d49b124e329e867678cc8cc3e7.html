SAP BusinessObjects Design Studio

Use SAP BusinessObjects Design Studio to view output data with rich visualizations, and to perform advanced analysis using the labels and value axis. For example, to analyze the sales on a country basis, add Country to the labels axis and Sales to the value axis.

In SAP BusinessObjects Design Studio you can preview calculation view output in different charts such as the column charts, pie charts, and so on. You can also apply filters on attributes and visualize filtered data.

For more information on SAP BusinessObjects Design Studio, see SAP BusinessObjects Design Studio documentation.
