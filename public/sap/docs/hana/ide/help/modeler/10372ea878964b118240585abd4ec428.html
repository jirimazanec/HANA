Restricted Column Properties

After creating a restricted column, you can view its properties or change them based on your business requirements.

Modeler displays the following properties for restricted columns in the Semantics node.

Properties

Description

Data Type

The value of this property specifies the data type of the restricted column.

Hidden

The value of this property determines whether the restricted column is hidden in reporting tools.

Display Folder

If the restricted measure is grouped in any of the display folder, the value of this property specifies the display folder that was used to group related measures. For more information, see Group Related Measures.
