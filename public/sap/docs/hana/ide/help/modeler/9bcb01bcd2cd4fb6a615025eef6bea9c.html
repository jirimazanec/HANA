Counter Properties

After creating a counter, you can view its properties or change them based on your business requirement.

Modeler displays the following properties for counter in the Semantics node.

Properties

Description

Data Type

The value of this property specifies the data type of the counter.

Semantic Type

The value of this property specifies the semantics assigned to the counter. For more information, see Assign Semantics

Hidden

The value of this property determines whether the counter column is hidden in reporting tools.

Exception Aggregation Type

The value of this property specifies the exception aggregation type used for creating counters. SAP HANA modeler supports only the COUNT_DISTINCT exception aggregation type for counters. This exception aggregation type counts the distinct occurrences of values for a set off attribute columns.

Display Folder

If the counter is grouped in any of the display folder, the value of this property specifies the display folder that was used to group related measures. For more information, see Group Related Measures.

Columns

The attribute columns used in the counter. Modeler counts the distinct combinations of these columns in the data source.
