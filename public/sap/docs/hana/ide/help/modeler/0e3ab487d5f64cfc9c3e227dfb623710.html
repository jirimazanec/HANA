Supported Calendar Types to Generate Time Data

SAP HANA modeler supports generating time dimension data using the Fiscal or Gregorian calendar types.

The table below provides more information on each of the calendar types.

Calendar Type

Description

Gregorian

Use the Gregorian calendar type, if your financial year is same as the calendar year, for example, January to December.

Fiscal

Use the Fiscal calendar type, if your financial year is not same as the calendar year, for example, March to April.
