Enable or Disable Attributes for Drilldown in Reporting Tools

By default, the tool lets you drilldown the attributes or calculated attributes in the reporting tools. You can disable this behavior for selected attributes.

Procedure

  1. Open the required calculation view in the view editor.
  2. Select the Semantics node.
  3. Select an attribute.
  4. In the Drill Down Enablement dropdown list, select a value.

    Set the value to blank if you want to disable drilldown for the selected attribute.
