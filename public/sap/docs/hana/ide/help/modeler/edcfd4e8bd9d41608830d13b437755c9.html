Default Client Values

Assign a default client to an calculation view and filter data at runtime based on the default client value. The table below lists the default client value types you can assign and their description.

Default Client Value

Description

Session Client

If you use session client as the default client value, then at run time, the tool filters the table data according to the value you specify as the session client in the user profile.

Cross Client

If you use cross client as the default client value, the tool does not filter the table data against any client and you see values relevant to all clients.

Fixed Client

If you want to use a fixed client value, for example, 001, then the tool filters the table data for this client value.
