Maintain Comments for Modeler Objects

When you are modeling a calculation view, you can also maintain comments for the view or for its objects such as parameters, hierarchies, view nodes and so on. The comments can include, for example, information that provides more clarity on the calculation view or its objects for data modelers accessing the same view or its objects.

Context

Maintaining comments helps your store more information related to the calculation view or to store and provide reference information for other data modelers working on the same calculation view. You can also use the comments for documentation purposes.
  • Columns in the semantics node
  • View Node
  • Input Parameters and Variables
  • Hierarchies
  • Calculated columns in underlying view nodes
Note

You cannot translate comments that you maintain for the modeler objects.

Procedure

  1. Open the SAP HANA Web-based Development Workbench Editor.

    The Web-based Editor tool is available on the SAP HANA XS Web server at the following URL: http://<WebServerHost>:80<SAPHANAinstance>/sap/hana/ide/editor

  2. Open the graphical calculation view for which you want to maintain comments in view editor.
  3. Maintain comments for the information view at the view level.
    1. Select the Semantics node.
    2. Choose the View Properties tab.
    3. In the General section, choose .
    4. Enter a new comment or edit an existing comment.
  4. Maintain comments for columns in the semantics node.
    1. Select the Semantics node.
    2. Choose the Columns tab.
    3. Select the column for which you want to maintain comments.
    4. Choose .
    5. Enter a new comment or edit an existing comment.
  5. Maintain comments for view nodes.
    1. Select the view node other than the Semantics node and the default view node.
    2. Choose .
    3. Enter a new comment or edit an existing comment.
  6. Maintain comments for input parameters and variables.
    1. Select the Semantics node.
    2. Choose the Parameters/Variables tab.
    3. Select an input parameter or variable for which you want to maintain comments.
    4. Choose .
    5. Enter a new comment or edit an existing comment.
  7. Maintain comments for hierarchies.
    1. Select the Semantics node.
    2. Choose the Hierarchies tab.
    3. Select a hierarchy for which you want to maintain comments.
    4. Choose .
    5. Enter a new comment or edit an existing comment.
  8. Maintain comments for calculated columns.
    1. Select a view node with a calculated column.
    2. Choose the Calculated Columns tab.
    3. Select a calculated column for which you want to maintain comments.
    4. Choose .
    5. Enter a new comment or edit an existing comment.
  9. Maintain comments for restricted columns.
    1. Select a default aggregation node.
    2. Choose the Restricted Columns tab.
    3. Select a restricted column for which you want to maintain comments.
    4. Choose .
    5. Enter a new comment or edit an existing comment.
