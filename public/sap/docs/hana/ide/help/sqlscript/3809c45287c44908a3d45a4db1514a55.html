Procedure Parameters

Parameter Modes

The following table lists the parameters you can use when defining your procedures.

Table 1: Parameter modes
Mode Description
IN An input parameter
OUT An output parameter
INOUT Specifies a parameter that passes in and returns data to and from the procedure
Note This is only supported for scalar values. The parameter needs to be parameterized if you call the procedure, for example CALL PROC ( inout_var=>?). A non-parameterized call of a procedure with an INOUT parameter is not supported.

Supported Parameter Types

Both scalar and table parameter types are supported. For more information on datatypes, see Datatype Extension
