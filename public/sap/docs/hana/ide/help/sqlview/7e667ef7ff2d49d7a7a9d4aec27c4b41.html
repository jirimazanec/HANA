M_CUSTOMIZABLE_FUNCTIONALITIES System View

Provides information about the enablement status of restricted features in databases. The full content of this view is only accessible from the system database.

Structure

Column name

Data type

Description

NAME

VARCHAR (64)

Customizable functionality name

DESCRIPTION

VARCHAR (256)

Customizable functionality description

IS_ENABLED

VARCHAR (5)

Customizable functionality is enabled: TRUE/FALSE
