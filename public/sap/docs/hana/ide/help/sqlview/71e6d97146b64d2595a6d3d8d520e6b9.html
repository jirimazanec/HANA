LAST_ACCESS_TIME TIMESTAMP The timestamp lastly accessed on the result cache entry by a user
ACCESS_COUNT BIGINT The number of access on the cache entry
RUNTIME_MISS_COUNT BIGINT The number of cache miss count decide at runtime with parameter values
