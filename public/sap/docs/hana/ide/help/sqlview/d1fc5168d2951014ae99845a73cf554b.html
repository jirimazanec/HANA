ACCESSIBLE_VIEWS System View

Provides information about accessible views for a given user.

Structure

Column name

Data type

Description

NVARCHAR(256)

Name of the user who is authorized to access the view

SCHEMA_NAME

NVARCHAR(256)

Schema name of the view

VIEW_NAME

NVARCHAR(256)

Name of the view

ANALYTICAL_PRIVILEGE_NEEDED

VARCHAR(5)

Specifies if this view is secured by means of analytical privileges
