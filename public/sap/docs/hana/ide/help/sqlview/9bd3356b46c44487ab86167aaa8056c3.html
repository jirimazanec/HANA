HOST_LIVECACHE_PROCEDURE_STATISTICS System View (Embedded Statistics Service)

LiveCache procedure statistics per host. This table contains information only for the last 21 days. The collection interval is 30 minutes.

Structure

Column name

Data type

Description

SNAPSHOT_ID TIMESTAMP Timestamp (UTC) identifying when the statistics service performed a data collection. Data collections scheduled for the same time have the same snapshot ID.
SERVER_TIMESTAMP TIMESTAMP Timestamp (local server time) identifying when a data collection actually occurred on the server.
OBJECT_NAME VARCHAR(128) Object name
METHOD_NAME VARCHAR(128) Method name
CALL_COUNT BIGINT Number of calls of the procedure
SUM_RUN_TIME BIGINT Sum runtime of the procedure
SUM_DEREF_COUNT BIGINT Number of OID derefs
SUM_DEREF_BASE_COUNT BIGINT Number of OID derefs against basis (incl. OMS version)
SUM_ITER_BASE_COUNT BIGINT Number of objects read via an OID-iterator from the basis (incl. OMS version)
SUM_DEREF_KEYED_OBJECT_COUNT BIGINT Number of key derefs
SUM_DEREF_KEYED_OBJECT_BASE_COUNT BIGINT Number of key derefs against basis (incl. OMS version)
SUM_ITER_KEYED_OBJECT_BASE_COUNT BIGINT Number of objects read via a key-iterator from the basis (incl. OMS version)
SUM_DEREF_VAR_OBJECT_BASE_COUNT BIGINT Number of derefs of var objects against basis (incl. OMS version)
SUM_NEW_OBJECT_COUNT BIGINT Number of newly created objects (STD+KEY+VAR)
SUM_NEW_OBJECT_IN_VERSION_COUNT BIGINT Number of newly created objects (STD+KEY+VAR) in an OMS version
SUM_DELETE_OBJECT_COUNT BIGINT Number of delete calls on objects (STD+KEY+VAR)
SUM_DELETE_OBJECT_IN_VERSION_COUNT BIGINT Number of delete calls on objects (STD+KEY+VAR) in an OMS version
SUM_LOCK_EXCLUSIVE_BASE_COUNT BIGINT Number of exclusive lock requests on objects (STD+KEY+VAR) in the basis
SUM_LOCK_SHARE_BASE_COUNT BIGINT Number of share lock requests on objects (STD+KEY+VAR) in the basis
EXCEPTION_COUNT BIGINT Number of dbp-exceptions thrown (exclusive OutOfDate,OutOfMemory,Timeout,OmsTerminate exceptions)
OUT_OF_DATE_EXCEPTION_COUNT BIGINT Number of out-of-date-exceptions thrown
OUT_OF_MEMORY_EXCEPTION_COUNT BIGINT Number of out-of-memory-exceptions thrown
TIMEOUT_EXCEPTION_COUNT BIGINT Number of timeout-exceptions thrown
OMS_TERMINATE_COUNT BIGINT Number of OMS terminate calls
SUM_USER_DELTA_MAX_SIZE BIGINT Peak delta of the user heap consumption during procedure execution (total)
SUM_OMS_DELTA_SIZE BIGINT Delta of the OMS heap consumption between start and end of the procedure execution (total)
SUM_VERSION_DELTA_SIZE BIGINT Delta of the OMS version heap consumption between start and end of the procedure execution (total)
SUM_HEAP_DELTA_SIZE BIGINT Delta of the overall heap consumption (user + OMS + OMS version) between start and end of the procedure execution (total)
SUM_STREAM_COMMUNICATION_TIME BIGINT Stream communication time (total)
SUM_INTERNAL_SQL_TIME BIGINT Internal SQL prepare+execute+fetch time (total)
