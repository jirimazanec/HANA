HOST_DELTA_MERGE_STATISTICS System View (Embedded Statistics Service)

Table delta merge statistics per host. The table contains information only for the last 42 days.

Structure

Column name

Data type

Unit

Description

SNAPSHOT_ID

TIMESTAMP

Datetime

Timestamp (UTC) identifying when the statistics service performed a data collection. Data collections scheduled for the same time have the same snapshot ID.

INDEX

NVARCHAR(5000)

  

Timestamp (local server time) identifying when a data collection actually occurred on the server.

SERVER_TIMESTAMP

TIMESTAMP

Datetime

Unique identifier for a row of the result set that a particular snapshot of a data collection represents.

ERROR_DESCRIPTION

NVARCHAR(2000)

Description of the last error that occurred during merge. A failing merge does not necessarily indicate a problem

EXECUTION_TIME

BIGINT

Millisecond

Execution duration

HOST

VARCHAR(64)

  

Host name

LAST_ERROR

INTEGER

  

Error code of the last error that occurred. Can be used to explain why a merge did not succeed. See ERROR_DESCRIPTION for details.

MEMORY_MERGE

VARCHAR(6)

Boolean

Flag to indicate table was merged in memory only: 'TRUE'/'FALSE'

MERGED_DELTA_RECORDS

INTEGER

Counter

Number of documents in delta

MOTIVATION

VARCHAR(8)

  

HARD, SMART, AUTO, FORCE

PART_ID

INTEGER

Partition number (0 for non-partitioned tables and 1 through number of partitions for partitioned tables. -1 if the whole table is unloaded

PASSPORT

VARCHAR(256)

  

External identifier for table merge called by an application

PORT

INTEGER

Internal port

SCHEMA_NAME

NVARCHAR(256)

  

Schema name

START_TIME

TIMESTAMP

Execution start time

SUCCESS

VARCHAR(5)

Boolean

Call success flag; depends on field TYPE; HINT: application merge hint was accepted/rejected; MERGE/SPARSE: delta merge/optimize compression was completed with or without success: 'TRUE'/'FALSE'

TABLE_NAME

NVARCHAR(256)

  

Table name

TYPE

VARCHAR(8)

  

MERGE (table delta merge), HINT (application merge hint), SPARSE (optimize compression), FACT (fact table compression), RECLAIM (table delta garbage collection)
