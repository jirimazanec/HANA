HOST_BLOCKED_TRANSACTIONS System View (Embedded Statistics Service)

Information on currently blocked transactions per host. The table contains information only for the last 7 days.

Structure

Column name

Data type

Description

SNAPSHOT_ID TIMESTAMP Timestamp (UTC) identifying when the statistics service performed a data collection. Data collections scheduled for the same time have the same snapshot ID.
SERVER_TIMESTAMP TIMESTAMP Timestamp (local server time) identifying when a data collection actually occurred on the server.
INDEX INTEGER Unique identifier for a row of the result set that a particular snapshot of a data collection represents.
WAITING_MINUTES DECIMAL(25,6) Waiting time in minutes
BLOCKED_TIME TIMESTAMP Blocked timestamp
BLOCKED_TRANSACTION_ID INTEGER Transaction object ID of the transaction waiting for a lock
BLOCKED_UPDATE_TRANSACTION_ID BIGINT Write transaction ID of the write transaction waiting for the lock
HOST VARCHAR(64) Host name
LOCK_MODE VARCHAR(32) Lock mode: 'SHARED'/'EXCLUSIVE'/'INTENTIONAL EXCLUSIVE'
LOCK_OWNER_TRANSACTION_ID INTEGER Transaction object ID of the transaction holding the lock
LOCK_OWNER_UPDATE_TRANSACTION_ID BIGINT Write transaction ID of the write transaction holding the lock

PORT INTEGER Internal port
WAITING_OBJECT_NAME NVARCHAR(256) Name of the object on which the lock is currently placed
