STATISTICS_SCHEDULE System View (Embedded Statistics Service)

Statistics schedule per host. This table contains information only for the last 7 days.

Structure

Column name

Data type

Description

ID SMALLINT Unique ID of check, collector or special function.
STATUS NVARCHAR(256) Scheduling status of the check, collector or special function. The following values are possible: Idle, Scheduled, Manual, Disabled, Inactive.
STATUSREASON NVARCHAR(1024) Reason if STATUS is Disabled. This column is empty for other scheduling statuses.
INTERVALLENGTH INTEGER Number of seconds between 2 scheduled calls of the check, collector, or special function
MINIMALINTERVALLENGTH INTEGER Minimum number of seconds between 2 scheduled calls. This value cannot be changed.
RETENTION_DAYS_CURRENT INTEGER Number of days for which data in statistics data tables is retained. Older data collected is deleted. This value is used for collectors only.
RETENTION_DAYS_DEFAULT INTEGER Default number of days for which data in statistics data tables is retained. See RETENTION_DAYS_CURRENT.
EXECUTION_TIME_OF_DAY TIME Time (UTC) used as the basis for calculating the execution times of checks, collectors, or special functions with INTERVALLENGTH of 1 hour, 6 hours, 12 hours, or 24 hours.
LATEST_START_SNAPSHOT_ID TIMESTAMP Time (UTC) at which the scheduler last actually called the check, collector, or special function
LATEST_START_SERVERTIME TIMESTAMP Time (local server time) at which the scheduler last actually called the check, collector, or special function (convenience field)
LATEST_DURATION BIGINT Duration in milliseconds of the most recently scheduled call of the check, collector, or special function
LATEST_START_SERVERTIME_MANUAL TIMESTAMP Time (local server time) at which the user last called the check, collector, or special function (convenience field)
START_COUNT INTEGER Number of times the check, collector, or special function has timed out.
SKIP_INTERVAL_ON_DISABLE INTEGER Length of time that a check or collector stays in status ‘Disabled’. This time is expressed as a multiple of interval length.
