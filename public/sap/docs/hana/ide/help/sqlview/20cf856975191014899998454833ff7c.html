SESSION_COOKIES System View

Shows information about available session cookies.

Structure

Column name

Data type

Description

USER_NAME

NVARCHAR(256)

The name of the user that this session cookie was created for

CLIENT_HOST

NVARCHAR(64)

The host name of client machine

CLIENT_PID

BIGINT

The client process ID

CREATE_TIME

TIMESTAMP

The time that the session cookie was created

LAST_CONNECT_TIME

TIMESTAMP

The last time that the session cookie was used for successful reconnection

SESSION_END_TIME

TIMESTAMP

The time that the main connection ended
