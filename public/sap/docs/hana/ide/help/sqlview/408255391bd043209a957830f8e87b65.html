UNSET PSE Statement (System Management)

Unsets the purpose for a PSE store.

Syntax
UNSET PSE <pse_name> PURPOSE <purpose>
Syntax Elements
<pse_name>

Specifies the name of the PSE store.

<pse_name> ::= <identifier>
<purpose>
<purpose> ::= 
 JWT
 | SAML
 | SAP LOGON
 | SSL
 | X509
Description

For the SSL purpose, this command can be executed by users with the SSL ADMIN privilege. For all other purposes, TRUST ADMIN is required.

Examples
UNSET PSE examplepse PURPOSE SSL
