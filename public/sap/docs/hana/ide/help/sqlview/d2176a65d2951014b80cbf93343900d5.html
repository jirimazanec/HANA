M_SERVICE_THREAD_SAMPLES System View

Displays detailed information about locks held by threads.

Structure

Column name

Data type

Description

HOST

VARCHAR(64)

Specifies the host name.

PORT

INTEGER

Specifies the internal port.

SERVICE_NAME

VARCHAR(32)

Specifies the service name. See the M_SERVICE_TYPES system view for all known service names.

HIERARCHY

VARCHAR(128)

Specifies the thread grouping information. This column contains the connection ID, the update transaction ID, and the transaction ID. The column is empty if the threads are inactive.

TIMESTAMP

TIMESTAMP

Specifies the timestamp of the record.

THREAD_ID

BIGINT

Specifies the thread ID.

THREAD_TYPE

VARCHAR(128)

Specifies the thread type.

THREAD_METHOD

VARCHAR(256)

Specifies the thread method.

THREAD_DETAIL

NVARCHAR(256)

Specifies the thread detail (truncated).

THREAD_STATE

VARCHAR(32)

Specifies the thread state.

IS_ACTIVE

VARCHAR(5)

Specifies whether the thread is active.

DURATION

BIGINT

Specifies the duration of the operation.

CALLER

VARCHAR(256)

Specifies the service which called this thread.

CALLING

VARCHAR(256)

Specifies the service which the thread calls.

STATEMENT_ID

VARCHAR(20)

Specifies the statement ID.

STATEMENT_HASH

VARCHAR(32)

Specifies the unique identifier for an SQL string.

CONNECTION_ID

INTEGER

Specifies the connection ID.

TRANSACTION_ID

INTEGER

Specifies the transaction ID.

UPDATE_TRANSACTION_ID

BIGINT

Specifies the write transaction ID (this number increments).

USER_NAME

NVARCHAR(256)

Specifies the SQL user name.

APPLICATION_NAME

NVARCHAR(256)

Specifies the name of the application.

APPLICATION_USER_NAME

NVARCHAR(256)

Specifies the application user name.

APPLICATION_SOURCE

NVARCHAR(256)

Specifies that the application can define which source file SAP HANA is called from. The usage is up to the application. This value is also displayed in M_PREPARED_STATEMENTS.APPLICATION_SOURCE.

CPU_TIME_SELF

BIGINT

Specifies the active CPU time of the thread (in millisecnds).

CPU_TIME_CUMULATIVE

BIGINT

Specifies the active CPU time of the thread and any associated children (in milliseconds).

CLIENT_IP

VARCHAR(45)

Specifies the IP of the client machine.

CLIENT_PID

BIGINT

Specifies the client process ID.

STATEMENT_EXECUTION_ID

BIGINT

Specifies the statement execution ID.

LOCK_WAIT_COMPONENT

VARCHAR(32)

Specifies the waiting time for the lock component.

LOCK_WAIT_NAME

VARCHAR(128)

Specifies the waiting time for the lock ID (can be joined with STATISTICS_NAME from, for example, M_MUTEXES).

LOCK_OWNER_THREAD_ID

BIGINT

Specifies the waiting time for the thread ID.

LOCKS_OWNED

VARCHAR(256)

Specifies the locks currently owned by the thread, including the lock type (shared/exclusive). Not all locks are monitored and show up here.

PRIORITY

INTEGER

Specifies the effective statement priority.

STATEMENT_THREAD_LIMIT

INTEGER

Specifies the effective statement thread limit.

STATEMENT_MEMORY_LIMIT

INTEGER

Specifies the effective statement memory limit.

PASSPORT_ROOTCONTEXT_ID

VARCHAR(16)

Specifies the passport root context ID.

PASSPORT_TRANSACTION_ID

VARCHAR(32)

Specifies the passport transaction ID.

PASSPORT_CONNECTION_ID

VARCHAR(16)

Specifies the passport connection ID.

PASSPORT_CONNECTION_COUNTER

BIGINT

Specifies the passport connection counter.

PASSPORT_COMPONENT_NAME

NVARCHAR(32)

Specifies the passport component name.

PASSPORT_ACTION

NVARCHAR(40)

Specifies the passport action.
