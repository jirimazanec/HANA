ALTER VIEW CACHE REFRESH Statement (System Management)

Refreshes all view cache entries that exist for the view with the specified view name.

Syntax
ALTER VIEW CACHE <view_name> REFRESH
Syntax Elements
<view_name>

Specifies the view for which a cache is defined.

<view_name> ::= <identifier>
Description

There is no subsequent SELECT statement required.
