M_STREAMING_PROJECT_SUBSCRIBERS System View [Smart Data Streaming]

Contains information and statistics about streaming subscribers for each SAP HANA streaming project, including external adapters.

Caution This information applies only for the SAP HANA option cited in the title of this topic. You must have purchased the license for it in order to use the information contained in this topic.

Only users with a MONITOR role can view the content of this system view. The view is empty for all other users.

Structure
Column name Data type Unit Description
SCHEMA_NAME NVARCHAR(256)   The schema in which the project is deployed.
PROJECT_NAME NVARCHAR(256)   The name given to the project. It corresponds to the APPLICATION_NAME in the M_STREAMING_APPLICATIONS System View for APPLICATION_TYPE "PROJECT" or "HA_PROJECT".
PROJECT_INSTANCE INTEGER   The instance number of the project.. It corresponds to the APPLICATION_INSTANCE in the M_STREAMING_APPLICATIONS System View for APPLICATION_TYPE "PROJECT" or "HA_PROJECT".
STREAMING_CONNECTION_ID BIGINT   A unique integer ID for the connection.
STREAM_NAME NVARCHAR(256)   The name of the stream subscribed to.
USER_NAME NVARCHAR(256)   The user name provided by the client during connection establishment, displayed once the user is authenticated.
CLIENT_HOST NVARCHAR(256)   The symbolic host name of the client machine, if available. If not available, host is the IP address of the client machine.
CLIENT_IP VARCHAR(64)   The IP address of the client machine, as a string.
CLIENT_PORT INTEGER   The TCP port number from which the connection originates.
CONNECTION_TAG NVARCHAR(256)   The user-set symbolic connection tag name. If not set by the user, the connection tag is NULL.
IS_SUBSCRIBED VARCHAR(5)   The status of a subscription to a stream.TRUE if subscribed, FALSE if not subscribed.
TRANSACTION_THROUGHPUT REAL MONITOR_UNIT_RATE The client's performance, in transactions per second, received by the client since the last update.
ROW_THROUGHPUT REAL MONITOR_UNIT_RATE The client's performance, in data rows per second, received by the client since the last update.
INCREMENTAL_TRANSACTION_COUNT BIGINT MONITOR_UNIT_COUNTER The number of transactions, envelopes, and messages received by the client since the last update.
INCREMENTAL_ROW_COUNT BIGINT MONITOR_UNIT_COUNTER The number of data rows received by the client since the last update.
TOTAL_TRANSACTION_COUNT BIGINT MONITOR_UNIT_COUNTER The total number of transactions, envelopes, and messages received by the client.
TOTAL_ROW_COUNT BIGINT MONITOR_UNIT_COUNTER The total number of data rows received by the client.
TOTAL_ROWS_DROPPED_COUNT BIGINT MONITOR_UNIT_COUNTER The total number of data rows dropped in the gateway because they were not read quickly enough by the client. For lossy subscriptions.
ACCUMULATOR_SIZE INTEGER   The number of rows collected in the accumulator to be sent in the next pulse. For pulsed subscriptions.
QUEUE_SIZE INTEGER   The number of rows queued for transmission to the client.
QUEUE_UTILIZATION REAL MONITOR_UNIT_PERCENT The subscriber queue size, as a percentage, relative to the queue size limit. If queue utilization reaches 100 percent, any future attempts to post data to this client are blocked, propagating the flow control back to the source of the post.
WORK_QUEUE_SIZE INTEGER   The number of rows queued for transmission to the client. These rows have been transferred from the internal queue to the socket buffer. The rows can be regrouped by envelopes.
CPU_CORE_UTILIZATION REAL MONITOR_UNIT_PERCENT Total CPU usage for the client thread, as a percentage of a single CPU core.
CPU_UTILIZATION REAL MONITOR_UNIT_PERCENT Total CPU usage for the client, as a percentage of all CPU cores on the machine. Total CPU usage equals system CPU usage plus user CPU usage.
CPU_SYSTEM_UTILIZATION REAL MONITOR_UNIT_PERCENT System CPU usage for the client, as a percentage of all CPU cores on the machine.
CPU_USER_UTILIZATION REAL MONITOR_UNIT_PERCENT User CPU usage for the client, as a percentage of all CPU cores on the machine.
TOTAL_CPU_TIME BIGINT MONITOR_UNIT_US Total CPU time since the creation of the client, in microseconds. Total CPU time equals system CPU time plus user CPU time.
TOTAL_CPU_SYSTEM_TIME BIGINT MONITOR_UNIT_US Total system CPU time, in microseconds, since the creation of the client thread.
TOTAL_CPU_USER_TIME BIGINT MONITOR_UNIT_US Total user CPU time, in microseconds, since the creation of the client thread .
CONNECT_TIME TIMESTAMP   The time when the server accepts (but does not authenticate) the connection.
SUBSCRIBER_DURATION BIGINT MONITOR_UNIT_US Duration of lapsed real time since the creation of the client thread.
LAST_UPDATE_TIME TIMESTAMP   The most recent time that statistics were updated for the subscriber.
