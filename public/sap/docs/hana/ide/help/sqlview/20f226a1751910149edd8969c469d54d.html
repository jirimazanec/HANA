TO_VARCHAR Function (Data Type Conversion)

Converts a given <value> to a VARCHAR character data type.

Syntax
TO_VARCHAR (<value> [, <format>])
Description

If the <format> specifier is omitted, then the conversion is performed using the date format model.

Example

The following example converts the value 2009-12-31 to a date value with the format YYYY/MM/DD. It then converts it again to a VARCHAR type and returns the value 2009/12/31

SELECT TO_VARCHAR (TO_DATE('2009-12-31'), 'YYYY/MM/DD') "to varchar" FROM DUMMY;
