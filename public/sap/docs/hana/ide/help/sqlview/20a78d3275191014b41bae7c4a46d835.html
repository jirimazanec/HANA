SQL Error Codes

Each SAP HANA error has a numeric error code. The M_ERROR_CODES system view contains information about the error codes.

  • To get a complete list of error codes and their descriptions, execute the following query:

    SELECT * FROM M_ERROR_CODES ORDER BY CODE ASC;

  • To get information about a specific error code, execute the following query:

    SELECT * FROM M_ERROR_CODES WHERE CODE=<error-code-number>;

    For example:

    SELECT * FROM M_ERROR_CODES WHERE CODE=14;

  • To get information for a specific error code string, execute the following query:

    SELECT * FROM M_ERROR_CODES WHERE CODE_STRING='<error-code-string>';

    For example:

    SELECT * FROM M_ERROR_CODES WHERE CODE_STRING='ERR_DATA';

  • To get information for a range of errors, execute the following query:

    SELECT * FROM M_ERROR_CODES WHERE CODE BETWEEN <error-code-number> AND <error-code-number> ORDER BY CODE ASC;

    For example:

    SELECT * FROM M_ERROR_CODES WHERE CODE BETWEEN 2 AND 200 ORDER BY CODE ASC;
