M_STREAMING_PROJECT_STREAMS System View [Smart Data Streaming]

Provides information and statistics about each stream in an SAP HANA streaming project.

Caution This information applies only for the SAP HANA option cited in the title of this topic. You must have purchased the license for it in order to use the information contained in this topic.

Only users with a MONITOR role can view the content of this system view. The view is empty for all other users.

Structure
Column name Data type Unit Description
SCHEMA_NAME NVARCHAR(256)   The schema in which the project is deployed.
PROJECT_NAME NVARCHAR(256)   The name given to the project. It corresponds to the APPLICATION_NAME in the M_STREAMING_APPLICATIONS System View for APPLICATION_TYPE "PROJECT" or "HA_PROJECT".
PROJECT_INSTANCE INTEGER   The instance number of the project.. It corresponds to the APPLICATION_INSTANCE in the M_STREAMING_APPLICATIONS System View for APPLICATION_TYPE "PROJECT" or "HA_PROJECT".
STREAM_NAME NVARCHAR(256)   The name of the stream
STREAM_TYPE VARCHAR(32)   The type of the stream.
  • STREAM
  • DELTASTREAM
  • WINDOW
  • METADATA
VISIBILITY VARCHAR(32)   The visibility of the stream.
  • INPUT
  • OUTPUT
  • LOCAL
  • INTERMEDIATE
TARGET_NAME VARCHAR(256)   Denotes that this stream is an intermediate stream used to populate this target.
GD_SUPPORT_LEVEL VARCHAR(32)   Indicates whether guaranteed delivery (GD) is enabled for the stream or window.
  • NOT_SUPPORTED
  • SUPPORTED
  • SUPPORTED_WITH_CHECKPOINT
SUPPORTED_WITH_CHECKPOINT is available only when the server is running in consistent recovery mode, or the auto checkpoint option is enabled.
TRANSACTION_THROUGHPUT REAL MONITOR_UNIT_RATE The stream's performance, in transactions per second, since the last update.
ROW_THROUGHPUT REAL MONITOR_UNIT_RATE The stream's performance, in rows per second, since the last update.
INCREMENTAL_TRANSACTION_COUNT BIGINT MONITOR_UNIT_COUNTER The number of transactions processed by the server since the last update.
INCREMENTAL_ROW_COUNT BIGINT MONITOR_UNIT_COUNTER The number of rows processed by the server since the last update.
QUEUE_SIZE INTEGER   The stream's input queue size.
STORE_ROW_COUNT BIGINT   The current number of records in the stream's store. Only applicable for windows.
SEQUENCE_NUMBER BIGINT   The sequence number of the current update.
CPU_CORE_UTILIZATION REAL MONITOR_UNIT_PERCENT Total CPU usage for the stream thread, as a percentage of a single CPU core.
CPU_UTILIZATION REAL MONITOR_UNIT_PERCENT Total CPU usage for the stream, as a percentage of all CPU cores on the machine. Total CPU usage is equal to the sum of the system CPU usage plus the user CPU usage.
CPU_SYSTEM_UTILIZATION REAL MONITOR_UNIT_PERCENT System CPU usage for the stream, as a percentage of all CPU cores on the machine.
CPU_USER_UTILIZATION REAL MONITOR_UNIT_PERCENT User CPU usage for the stream, as a percentage of all CPU cores on the machine.
TOTAL_CPU_TIME BIGINT MONITOR_UNIT_US Total CPU time since the creation of the stream, in microseconds. Total CPU time is equal to the sum of the system CPU time plus the user CPU time.
TOTAL_CPU_SYSTEM_TIME BIGINT MONITOR_UNIT_US Total system CPU time, in microseconds, since the creation of the stream.
TOTAL_CPU_USER_TIME BIGINT MONITOR_UNIT_US Total user CPU time, in microseconds, since the creation of the stream.
LAST_UPDATE_TIME TIMESTAMP   The most recent time that statistics were updated for the stream.
