ALTER DATABASE Statement (Remote Data Sync)

Alters an existing database by adding or removing the rdsyncserver service.

Syntax
Caution This information applies only for the SAP HANA option cited in the title of this topic. You must have purchased the license for it in order to use the information contained in this topic.
Note While SAP HANA and remote data sync share much of the same base SQL syntax, there are some differences in supported and nonsupported clauses and parameters. The following syntax applies only to remote data sync.
ALTER DATABASE <database_name>
{ ADD 'rdsyncserver' [ AT [LOCATION] '<hostname>[:<port_number>]' ]
  | REMOVE 'rdsyncserver' AT [LOCATION] '<hostname>[:<port_number>]' }
Syntax Elements
<database_name> ::= <identifier>

This parameter specifies the name of the database to which the change will be applied.

<hostname> ::= <identifier>

This parameter specifies the name of the host where the remote data sync server runs. If you specify the AT clause and there is a remote data sync server on the host, then the specified tenant is associated with the given remote data sync server. If you do not specify the AT clause and a remote data sync server has not yet been associated with the host, then a randomly chosen remote data sync server is associated with the host.

<port_number> ::= <identifier>

This parameter specifies port number that the remote data sync server uses on the host.

Description

There must be a host in the HANA landscape in the rdsync role that has not yet been associated with a tenant database for you to execute this statement. Before synchronizations can begin, you must initialize the rdsyncserver service by executing an ALTER SYSTEM INITIALIZE statement.

Execute this statement as the SYSTEM user in SYSTEMDB.

Examples

The following example shows you how to add the service rdsyncserver to the database my_database at hostA on port 30321.

ALTER DATABASE my_database ADD 'rdsyncserver'  AT LOCATION 'hostA:30321'

The following example shows you how to remove the service rdsyncserver from the database my_database at hostA on port 30321.

ALTER DATABASE my_database REMOVE 'rdsyncserver'  AT LOCATION 'hostA:30321'
