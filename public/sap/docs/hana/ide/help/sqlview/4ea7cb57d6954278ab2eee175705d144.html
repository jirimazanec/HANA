HOST_MVCC_OVERVIEW System View (Embedded Statistics Service)

MVCC overview per host. This table contains information only for the last 7 days.

Structure

Column name

Data type

Description

SNAPSHOT_ID TIMESTAMP Timestamp ID (UTC)
SERVER_TIMESTAMP TIMESTAMP Processing Timestamp (local time)
INDEX VARCHAR(95) Index
ACQUIRED_LOCK_COUNT BIGINT Number of Acquired Records Locks
GLOBAL_MVCC_TIMESTAMP BIGINT Current Global MVCC timestamp
HOST VARCHAR(64) Host name
MAX_VERSIONS_PER_PARTITION BIGINT Max number of versions per partition
MIN_MVCC_SNAPSHOT_TIMESTAMP BIGINT Minimum MVCC timestamp which at least one transaction holds
MIN_READ_TRANSACTION_ID BIGINT All active transactions can see changes of TRANSACTION IDs less than or equal to MIN_READ_TRANSACTION_ID
MIN_WRITE_TRANSACTION_ID BIGINT All write transactions with TRANSACTION ID less than or equal to MIN_WRITE_TRANSACTION_ID are closed
NEXT_WRITE_TRANSACTION_ID BIGINT Maximum TRANSACTION ID which will be assigned to next write transaction
PORT INTEGER Internal port
VERSION_COUNT BIGINT Number of all versions per service
