VIEWS System View

Lists available views.

Structure

Column name

Data type

Description

SCHEMA_NAME

NVARCHAR(256)

Specifies the schema name

VIEW_NAME

NVARCHAR(256)

Specifies the view name

VIEW_OID

BIGINT

Specifies the object ID of the view

IS_UNICODE

VARCHAR(5)

Specifies whether the query string contains Unicode: TRUE, FALSE

IS_READ_ONLY

VARCHAR(5)

Specifies whether this view is read-only or updatable: TRUE, FALSE

HAS_CHECK_OPTION

VARCHAR(5)

Specifies whether this view has an updatable view condition: TRUE, FALSE

HAS_COLUMN_ALIASES

VARCHAR(5)

Specifies whether the view has a columns alias: TRUE, FALSE

DEFINITION

NCLOB

Specifies the view definition

COMMENTS

NVARCHAR(5000)

Specifies the view description

IS_COLUMN_VIEW

VARCHAR(5)

Specifies whether this view is a column view: TRUE, FALSE

VIEW_TYPE

VARCHAR(16)

Specifies the type of view: ROW, OLAP, JOIN, HIERARCHY, CALC

IS_TENANT_SHARED_METADATA

VARCHAR(5)

Specifies whether the table is a global table or not: TRUE, FALSE

IS_VALID

VARCHAR(5)

Specifies whether the view is valid or not. This value is FALSE when its base objects are changed or dropped: TRUE, FALSE

HAS_STRUCTURED_PRIVILEGE_CHECK

VARCHAR(5)

Returns TRUE if the view is registered for a structured privilege check, else FALSE

HAS_MASKED_COLUMNS

VARCHAR(5)

Returns TRUE if the view has a mask definition for at least one column, else FALSE

HAS_CACHE

VARCHAR(36)

Specifies whether the result cache is enabled for the view: STATIC, FULL/STATIC, PARTIAL (SELECTED COLUMNS)/STATIC, PARTIAL (FILTERED)/STATIC, PARTIAL (SELECTED COLUMNS, FILTERED)/DYNAMIC, FULL/DYNAMIC, PARTIAL (FILTERED)/NONE

CACHE_RETENTION

INTEGER

Specifies the cache refresh interval in minutes

CACHE_FILTER

NCLOB

Specifies the filter condition specified to reduce the cache size

IS_CACHE_FORCED

VARCHAR(5)

Indicates whether the result cache is forced or cachability is checked

CREATE_TIME

TIMESTAMP

Specifies the creation time
