Full description

This view has a resettable counterpart. This means, you can also see the values since the last reset in the dependent view M_LIVECACHE_PROCEDURE_STATISTICS_RESET. To reset the values, use the SQL command: ALTER SYSTEM RESET MONITORING VIEW SYS.M_LIVECACHE_PROCEDURE_STATISTICS_RESET or the management console command: monitor reset -n M_LIVECACHE_PROCEDURE_STATISTICS_RESET

For each liveCache procedure, which has been called already since the last restart, statistics are shown.

Caution

This view can only be used if liveCache is enabled.
