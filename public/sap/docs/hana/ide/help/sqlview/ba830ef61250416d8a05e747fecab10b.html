ABSTRACT_SQL_PLANS System View

Provides information about Abstract SQL Plans (ASPs).

Structure

Column name

Data type

Description

ID

BIGINT

Number that uniquely identifies the ASP

STATEMENT_STRING

NCLOB

Target statement string

CACHE_KEY

NVARCHAR(500)

Key for the ASP in JSON format

ABSTRACT_ SQL_PLAN

NCLOB

ASP in JSON format

IS_ENABLED

VARCHAR(5)

Whether the ASP is enabled (TRUE) or disabled (FALSE)

IS_VALID

VARCHAR(5)

Whether the ASP is valid (TRUE) or invalid (FALSE)

NOTES

VARCHAR(5000)

Management information about the ASP

LAST_ENABLE_TIME

TIMESTAMP

Timestamp for the last time the ASP was enabled

LAST_ENABLE_USER

NVARCHAR(256)

ID of the user who last enabled the ASP

LAST_DISABLE_TIME

TIMESTAMP

Whether the ASP is enabled (TRUE) or disabled (FALSE)

LAST_DISABLE_USER

NVARCHAR(256)

ID of the user who last disabled the ASP
