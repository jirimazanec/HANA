SEQUENCES System View

Provides information about available sequences.

Structure

Column name

Data type

Description

SCHEMA_NAME

NVARCHAR(256)

Specifies the schema name of the sequence

SEQUENCE_NAME

NVARCHAR(256)

Specifies the name of the sequence

SEQUENCE_OID

BIGINT

Specifies the object ID of the sequence

START_NUMBER

BIGINT

Specifies the start number

MIN_VALUE

BIGINT

Specifies the minimum value of the sequence

MAX_VALUE

BIGINT

Specifies the maximum value of the sequence

INCREMENT_BY

BIGINT

Specifies the incremental value

IS_CYCLED

VARCHAR(5)

Specifies whether the sequence starts with MIN_VALUE after having reached MAX_VALUE in case INCREMENT_BY is greater than 0 or start with MAX_VALUE after having reached MIN_VALUE in case INCREMENT_BY is less than 0: 'TRUE', 'FALSE'

RESET_BY_QUERY

NCLOB

The reset by query string for the sequence

CACHE_SIZE

SMALLINT

Specifies the cache size of the sequence

CREATE_TIME

TIMESTAMP

Specifies the creation time of the sequence
