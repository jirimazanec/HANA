M_JOINENGINE_STATISTICS System View

Provides statistics about join engine runtime objects used for column store join operations.

Structure

Column name

Data type

Unit

Description

HOST

VARCHAR(64)

Specifies the host name.

PORT

INTEGER

Specifies the internal port.

TRANSLATION_TABLE_COUNT

BIGINT

Specifies the number of translation tables.

TRANSLATION_TABLE_MEMORY_SIZE

BIGINT

Byte

Specifies the memory used by translation tables.

TRANSLATION_TABLE_UNLOAD_COUNT

BIGINT

Specifies the number of translation table unloads.
