M_ES_DBSPACE_FILES System View [Dynamic Tiering]

Shows information of all dbspace files in the extended storage.

Caution This information applies only for the SAP HANA option cited in the title of this topic. You must have purchased the license for it in order to use the information contained in this topic.

This monitoring view provides information for the extended storage corresponding to the connected HANA database.

Only users with CATALOG READ, DATA ADMIN, or EXTENDED STORAGE ADMIN system privilege can view the content of the M_ES_DBSPACE_FILES view. The view is empty for all other database users.

Structure
Column name Data type Description
HOST VARCHAR(64) Host name.
PORT INTEGER Internal port.
VOLUME_ID INTEGER Volume ID.
DBSPACE_NAME VARCHAR(128) Name of the dbspace using this file.
DBSPACE_FILE_NAME VARCHAR(128) Name of the dbspace file.
STATUS VARCHAR(7) Status of the file:
  • OFFLINE
  • ONLINE
MODE VARCHAR(9) Mode of the file:
  • READONLY
  • READWRITE
'
USAGE DOUBLE Percent of file in use by the dbspace.
PATH VARCHAR(512) Physical path of the file.
TOTAL_SIZE BIGINT Total size of the file specified in bytes.
RESERVED_SIZE BIGINT Reserved space in bytes that can be added to the fil.e
DISK_STRIPE_SIZE BIGINT Disk stripe size specified in bytes.
IS_DROP_ALLOWED VARCHAR(5) TRUE if file can be dropped, else FALSE.
