M_CONVERTER_STATISTICS System View

Provides converter statistics.

Full description

This view has a resettable counterpart. This means, you can also see the values since the last reset in the dependent view M_CONVERTER_STATISTICS_RESET. To reset the values, use the SQL command: ALTER SYSTEM RESET MONITORING VIEW SYS.M_CONVERTER_STATISTICS_RESET or the management console command: monitor reset -n M_CONVERTER_STATISTICS_RESET

This view contains information about the converter, which administers logical page numbers and maps them to physical pages within the DataVolume(s).
