NVARCHAR(256)

NVARCHAR(256)

OBJECT_OID

NVARCHAR(256)

STRUCTURED_PRIVILEGE_NAME

NVARCHAR(256)

Name of a certain structured privilege created for that particular object

STRUCTURED_PRIVILEGE_ID

BIGINT

ID of the structured privilege

STRUCTURED_PRIVILEGE_FILTER

NVARCHAR(5000)

Filter condition provided by that particular structured privilege

STRUCTURED_PRIVILEGE_DEFAULT_SCHEMA_NAME

NVARCHAR(256)

Default schema name of this structured privilege (used for unqualified tables and views involved in a complex filter)

STRUCTURED_PRIVILEGE_STATUS

NVARCHAR(64)

Status of a particular structured privilege (APPLIED, NOT GRANTED, NO MATCHING ATTRIBUTE, NO FILTER VALUES FOUND)
