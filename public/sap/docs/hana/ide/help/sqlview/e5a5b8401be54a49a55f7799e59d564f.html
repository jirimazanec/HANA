M_DYNAMIC_RESULT_CACHE_EXCLUSIONS System View

Lists cache exclusions of the dynamic result cache.

Structure

Column name

Data type

Description

SCHEMA_NAME

NVARCHAR(256)

Schema name that the dynamic result cache entry belongs to

OBJECT_NAME

NVARCHAR(256)

Object name that the dynamic result cache entry belongs to

DETAILS

NVARCHAR(5000)

Lists all constituents of the excluded cache key

MEMORY_SIZE

BIGINT

Size, in bytes, of the memory occupied by the excluded cache entry

RECORD_COUNT

BIGINT

Number of records in the excluded cache entry

EXCLUDE_TIME

TIMESTAMP

Timestamp when the dynamic result cache entry was disabled
Description

Cache entries that exceed configurable size (defined in the .ini file) are automatically removed and added to M_DYNAMIC_RESULT_CACHE_EXCLUSION. If the view is marked to be excluded, then redefine the view definition rather than increasing the configurable size.
