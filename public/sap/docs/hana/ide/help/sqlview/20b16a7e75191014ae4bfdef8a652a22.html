Column name

Data type

Description

VARCHAR(256)

VARCHAR(16)

TENANT_NAME

VARCHAR(256)

Deprecated.

HOST

VARCHAR(64)

Host name, if layer is HOST

VARCHAR(128)

Configuration section name

KEY

VARCHAR(128)

Configuration key name

VALUE

VARCHAR(5000)

Configuration value
