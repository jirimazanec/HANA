TRIGGERS System View

Provides information about triggers that are defined for tables.

Structure

Column name

Data type

Description

SCHEMA_NAME

NVARCHAR(256)

The schema name of the trigger

TRIGGER_NAME

NVARCHAR(256)

The trigger name

TRIGGER_OID

BIGINT

The object ID of the trigger

OWNER_NAME

NVARCHAR(256)

The name of the trigger owner

OWNER_OID

BIGINT

The object ID of the trigger owner

SUBJECT_TABLE_SCHEMA

NVARCHAR(256)

The schema name of the table, the trigger is defined for

SUBJECT_TABLE_NAME

NVARCHAR(256)

The table name of the table, the trigger is defined for

TRIGGER_ACTION_TIME

VARCHAR(10)

The time the trigger is executed: BEFORE, AFTER, INSTEAD OF the specified event

TRIGGER_EVENT

VARCHAR(20)

The event, the trigger is defined for: combination of DELETE, INSERT, UPDATE

TRIGGERED_ACTION_LEVEL

VARCHAR(9)

The level of the event, where the triggered action happens: ROW, STATEMENT

DEFINITION

NCLOB

The query string of the trigger

IS_VALID

VARCHAR(5)

Specifies whether the trigger is valid or not. This becomes FALSE when its base objects are changed or dropped: TRUE, FALSE

IS_ENABLED

VARCHAR(5)

Specifies whether the trigger is enabled or not: TRUE, FALSE

CREATE_TIME

TIMESTAMP

The time that the trigger was created
