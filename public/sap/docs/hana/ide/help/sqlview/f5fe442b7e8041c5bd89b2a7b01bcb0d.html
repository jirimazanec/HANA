BACKUP DATA DROP SNAPSHOT Statement (Backup and Recovery)

Drops one or more database snapshots.

Syntax
BACKUP DATA [ FOR FULL SYSTEM ] DROP SNAPSHOT
Syntax elements
FOR FULL SYSTEM

Specifies that the drop operation is performed on every database in the current system, including SystemDB and all tenant databases. Omitting FOR FULL SYSTEM performs the operation only on the connected database.

Description

In the case of an error during BACKUP DATA...CLOSE SNAPSHOT, executing BACKUP DATA DROP SNAPSHOT removes the latest database snapshot completely from persistence.

Example

Drop the system-wide database snapshot.

BACKUP DATA FOR FULL SYSTEM DROP SNAPSHOT;
