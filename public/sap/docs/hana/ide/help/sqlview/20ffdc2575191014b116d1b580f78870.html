Data type

Description

The number of filter to be able to combine all operators/ operands belonging to one filter

FILTER_TYPE

VARCHAR(16)

The type of filter: STATIC, DYNAMIC

VARCHAR(5)

Indicates whether the operator was negated in filter; TRUE, FALSE

OPERATOR

VARCHAR(16)

The type of operator: CONTAINS PATTERN, BETWEEN, EQUAL, IN, LESS THAN, LESS EQUAL, GREATER THAN, GREATER EQUAL

OPERAND_ORDER

INTEGER

The sequence of operands per filter ID

OPERAND

NVARCHAR(5000)

The value that the operator is compared to
