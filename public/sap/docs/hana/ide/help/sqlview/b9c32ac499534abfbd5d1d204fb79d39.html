M_ES_TABLES System View [Dynamic Tiering]

Shows information for all extended tables in extended storage.

Caution This information applies only for the SAP HANA option cited in the title of this topic. You must have purchased the license for it in order to use the information contained in this topic.

An extended table is a table created in extended storage. This monitoring view provides information for the extended storage corresponding to the connected HANA database.

Users with DATA ADMIN system privilege can view information about all extended tables in the M_ES_TABLES view. Users without this system privilege can see information only for extended tables owned by that user or for extended tables to which they have SELECT permission.

Structure
Column name Data type Description
SCHEMA_NAME NVARCHAR(256) Schema name for extended table.
TABLE_NAME NVARCHAR(256) Table name for extended table.
PART_ID INTEGER Logical partition ID. Value is 0 for extended tables.
IS_DELTA_ENABLED VARCHAR(5) TRUE if it is a RLV table, otherwise FALSE.
TABLE_SIZE BIGINT Size of compressed data in bytes.
RECORD_COUNT BIGINT Row count in extended table.
LAST_MODIFIED_TIME TIMESTAMP Timestamp when table was last modified.
