ALTER DATABASE Statement (Tenant Database Management)

Makes changes to a tenant database.

Syntax
ALTER DATABASE <database_name> 
 (ADD '<servicetype>' [AT [LOCATION] '<hostname>[:<port_number>]' ]) 
 | ( REMOVE '<servicetype>' AT [LOCATION] '<hostname>:<port_number>' ) 
 | OS USER '<username>' OS GROUP '<groupname>'
 | FINALIZE REPLICA [DROP SOURCE DATABASE]
 | CANCEL REPLICA
 | SYSTEM USER PASSWORD <password>
Syntax Elements
<database_name>

Specifies the name of the database to which the change is applied.

<database_name> ::= <identifier>
<servicetype>

Specifies the service type to add. See the SAP HANA Administration Guide for more details.

<servicetype> ::= 'indexserver' | 'xsengine' | 'scriptserver' | 'dpserver'
<hostname>

Specifies the host name where the database is located.

<port_number>

Specifies the port number for the host name where the database is located.

<username>

Changes the user name.

<username> ::= <identifier>
<groupname>

Changes the OS user and OS group.

<groupname> ::= <identifier>

The database must be offline to change the user and group. The user and group must already exist already. If you specify OS USER '' OS GROUP '', it is interpreted as OS USER 'sidadm' OS GROUP 'sapsys'.

FINALIZE REPLICA

Stops a replication and finalizes the copy of the database. Use this syntax when the replication status is ACTIVE.

DROP SOURCE DATABASE

Drops the original tenant database in the source system.

CANCEL REPLICA

Cleans up the system. Use this syntax when you cannot create a new replica because the replication status of the source database is REPLICATING even though the target database has been dropped. You must execute this statement from the source database (the one that is being replicated).

SYSTEM USER PASSWORD <password>

Specifies a new system user password for the tenant database.

Description

The use of this statement requires the DATABASE ADMIN privilege.

Resetting of the system user password is an audited action on both systemdb and on the tenant. Also, the tenant database must be shutdown before resetting the system user password.

This feature is used with restrictions and/or is extended by the following SAP HANA options:

  • Accelerator for SAP ASE
  • Dynamic Tiering
  • Remote Data Sync
  • Smart Data Streaming
Examples

Add the service scriptserver to the database my_database at hostA, port 30303.

ALTER DATABASE my_database ADD 'scriptserver' AT LOCATION 'hostA:30303';

The following example changes the system user password on fictitious database my_database to fhu37656:

ALTER DATABASE my_database SYSTEM USER PASSWORD fhu37656;
