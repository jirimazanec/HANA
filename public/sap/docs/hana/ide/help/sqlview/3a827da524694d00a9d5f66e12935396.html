PINNED_SQL_PLANS System View

Provides information on currently pinned SQL plans and corresponding hints.

Structure

Column name

Data type

Description

PINNED_PLAN_ID

BIGINT

The pinned ID

VOLUME_ID

INTEGER

The volume ID

USER_NAME

NVARCHAR(256)

The name of the user who prepared the plan

SESSION_USER_NAME

NVARCHAR(256)

The SESSION_USER name who prepared the plan. The same value for M_CONNECTIONS.USER_NAME

SCHEMA_NAME

NVARCHAR(256)

The schema name that the SQL plan belongs to

STATEMENT_STRING

NCLOB

The statement string

STATEMENT_HASH

VARCHAR(32)

The MD5 hash value for STATEMENT_STRING

IS_INTERNAL

TINYINT

The value is TRUE if the plan is executed from a database internal connection and FALSE if it is executed from a remote connection

ABAP_VARCHAR_MODE

TINYINT

The value is TRUE if ABAP VARCHAR mode is enabled, and FALSE otherwise

PIN_TIME

TIMESTAMP

The timestamp when the plan is pinned

LAST_MODIFY_TIME

TIMESTAMP

The timestamp when the plan is modified

HINT_STRING

NCLOB

The text string to indicate applied hint

SESSION_PROPERTIES

NVARCHAR(500)

The composed variables from session context
