CERTIFICATES System View

Provides information about certificates useable in PSEs.

Structure

Column name

Data type

Description

CERTIFICATE_ID

BIGINT

Unique identification of the certificate

SUBJECT_COMMON_NAME

NVARCHAR(5000)

Distinguished name of the X.509 certificate subject

ISSUER_COMMON_NAME

NVARCHAR(5000)

Common name of the X.509 certificate issuer

VALID_FROM

TIMESTAMP

Start time of certificate's validity

VALID_UNTIL

TIMESTAMP

End time of certificate's validity

COMMENT

NVARCHAR(1024)

Description for the certificate

SUBJECT_DISTINGUISHED_NAME

NVARCHAR(5000)

Distinguished name of the X.509 certificate subject

ISSUER_DISTINGUISHED_NAME

NVARCHAR(5000)

Distinguished name of the X.509 certificate issuer

BASIC_CONSTRAINTS

NVARCHAR(32)

Basic constraints

SUBJECT_ALT_NAME

NVARCHAR(5000)

Comma-separated list of alternative names of the X.509 certificate subject

PUBLIC_KEY_ALGORITHM

NVARCHAR(256)

Public key algorithm

PUBLIC_KEY_LENGTH

SMALLINT

Public key length

KEY_USAGE

NVARCHAR(256)

Comma-separated list of key usage strings as specified in RFC 5280

SIGNATURE_ALGORITHM

NVARCHAR(256)

Signature algorithm

VERSION

TINYINT

Version of the certificate format

SERIAL_NUMBER

NVARCHAR(64)

Serial number as assigned from the certificate issuer

FINGERPRINT

NVARCHAR(256)

The hash of the entire certificate. It is used as a unique identifier in the certificate store.

CERTIFICATE

NCLOB

The certificate as given during SQL
