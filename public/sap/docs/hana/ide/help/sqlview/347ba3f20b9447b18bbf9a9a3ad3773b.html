HOST_COLUMN_TABLES_PART_SIZE System View (Embedded Statistics Service)

Information on column tables partition size per host. The table contains information only for the last 42 days.

Full description

Only tables where MEMORY_SIZE_IN_TOTAL (the amount of space the table uses in a specific partition) > 1,000,000 bytes are listed.

The table contains information only for the last 30 days.
