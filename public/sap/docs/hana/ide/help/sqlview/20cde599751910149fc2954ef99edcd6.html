SCHEMAS System View

Shows available schemas.

Structure

Column name

Data type

Description

SCHEMA_NAME

NVARCHAR(256)

Specifies the schema name

SCHEMA_OWNER

NVARCHAR(256)

Specifies the schema owner

HAS_PRIVILEGES

VARCHAR(5)

Shows if the user is the schema owner or has any privileges for the schema or any object within the schema: TRUE, FALSE

CREATE_TIME

TIMESTAMP

Specifies the creation time of the schema
