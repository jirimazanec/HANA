M_DYNAMIC_RESULT_CACHE System View

Lists statistics for the dynamic result cache.

Structure

Column name

Data type

Description

CACHE_ID

BIGINT

Unique ID for each dynamic result cache entry

SCHEMA_NAME

NVARCHAR(256)

Schema name that the dynamic result cache entry belongs to

OBJECT_NAME

NVARCHAR(256)

Object name that the dynamic result cache entry belongs to

OBJECT_TYPE

VARCHAR(32)

Object type that the dynamic result cache entry belongs to

DETAILS

NVARCHAR(5000)

Details (all constituents of cache key)

LAST_MVCC_SNAPSHOT_TIMESTAMP

BIGINT

Last cached MVCC snapshot timestamp

MEMORY_SIZE

BIGINT

Size, in bytes, of the memory occupied by the dynamic result cache entry

RECORD_COUNT

BIGINT

Number of records in the cache entry

DELTA_RECORD_COUNT

BIGINT

Number of records in the cache entry created by a delta refresh

IS_REFRESHING

VARCHAR(5)

The status of asynchronous cache entry refresh job. It will be TRUE if refresh job is currently running

CURRENT_REFRESH_DURATION

BIGINT

The duration (in milliseconds) of the currently running cache entry refresh job. The value is -1 when IS_REFRESHING is FALSE.

CURRENT_REFRESH_REASON

VARCHAR(32)

The reason why the refresh job was triggered. The value is empty when IS_REFRESHING is FALSE

CREATE_TIME

TIMESTAMP

Timestamp when the dynamic result cache entry was created

LAST_REFRESH_TIME

TIMESTAMP

Timestamp when the last cache entry refresh job finished.

LAST_REFRESH_DURATION

BIGINT

The duration (in milliseconds) of the last cache entry refresh job (finished)

LAST_REFRESH_REASON

VARCHAR(32)

The reason why the last refresh job was triggered

LAST_DELTA_REFRESH_TIME

TIMESTAMP

Timestamp when the last delta refresh on the cache entry finished.

REFRESH_COUNT

BIGINT

Number of refreshes on the cache entry

DELTA_REFRESH_COUNT

BIGINT

Number of delta refreshes on the cache entry
HOST

VARCHAR(64)

Host name

PORT

INTEGER

Internal port
