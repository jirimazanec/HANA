SQLSCRIPT_TRACE System View

Contains the names of the local temporary tables created by the SQLScript TRACE operator.

Structure

Column name

Data type

Description

TRACE_TIMESTAMP

TIMESTAMP

The timestamp that was generated when the TRACE was performed

TRANSACTION_ID

INTEGER

The transaction object ID

STATEMENT_ID

BIGINT

The logical ID of the outmost user statement, which triggers the procedure execution (direct or indirect)

PROCEDURE_OID

BIGINT

The procedure object ID

VARIABLE_NAME

NVARCHAR(256)

The internal variable name used for the variable to which the TRACE operator is assigned

TABLE_NAME

NVARCHAR(256)

The name of the local temporary table containing the table result
