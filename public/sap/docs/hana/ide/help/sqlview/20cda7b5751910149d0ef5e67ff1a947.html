SAML_PROVIDER System View

Shows available SAML providers.

Structure

Column name

Data type

Description

SAML_PROVIDER_NAME

NVARCHAR(256)

The provider name

SUBJECT_NAME

NVARCHAR(512)

The subject of the X.509 certificate of the SAML provider

ISSUER_NAME

NVARCHAR(512)

The issuer of the X.509 certificate of the SAML provider

IS_USER_CREATION_ENABLED

VARCHAR(5)

Specifies whether the SAML provider is allowed to create a user implicitly: TRUE, FALSE
