M_PLUGIN_STATUS System View

Provides status for plugins.

Structure

Column name

Data type

Description

PLUGIN_NAME

VARCHAR

A subdirectory, containing executables and/or plugins, where the files of an AFL area are located. There can be several areas per subdirectory.

ERROR_TEXT

NVARCHAR

An error message (in case a package/AFL cannot be registered)

AREA_NAME

NVARCHAR

Name of an area

AREA_STATUS

VARCHAR

Status of an area. Possible values are: REGISTRATION SUCCESSFUL, REGISTRATION FAILED, or REGISTRATION SKIPPED.

PACKAGE_NAME

NVARCHAR

Name of a package (internal representation of an AFL). There can be several packages per area.

PACKAGE_STATUS

VARCHAR

Status of a package. Possible values are: REGISTRATION SUCCESSFUL, REGISTRATION FAILED, or REGISTRATION SKIPPED.
