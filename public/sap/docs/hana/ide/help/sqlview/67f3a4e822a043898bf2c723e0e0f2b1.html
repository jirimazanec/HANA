M_SERIES_TABLES System View

Statistics on the physical contents of series tables.

Structure

Column name

Data type

Description

SCHEMA_NAME

NVARCHAR(256)

Schema name.

TABLE_NAME

NVARCHAR(256)

Column table name.

PART_ID

INTEGER

Partition number. This value is zero for non-partitioned tables.

SERIES_KEY_COUNT

BIGINT

Number of distinct series identified by the SERIES KEY columns.

PERIOD_COUNT

BIGINT

The number of distinct periods represented in the period column..

EQUIDISTANT_RECORD_COUNT

BIGINT

Number of rows optimized to be in a piecewise equidistant segment.

NOT_EQUIDISTANT_RECORD_COUNT

BIGINT

Number of rows that have not been optimized to be in a piecewise equidistant segment.

REORGANIZED_RECORD_COUNT

BIGINT

Number of rows that have been optimized by the ALTER TABLE SERIES REORGANIZE statement.

NOT_REORGANIZED_RECORD_COUNT

BIGINT

Number of rows that have not been processed by the ALTER TABLE SERIES REORGANIZE statement.
