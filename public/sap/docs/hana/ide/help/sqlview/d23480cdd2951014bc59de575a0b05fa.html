ST_GEOMETRY_COLUMNS System View

Provides information about spatial columns.

Structure

Column name

Data type

Description

NVARCHAR(256)

NVARCHAR(256)

The name of the table containing the spatial column

NVARCHAR(256)

The name of the spatial column

SRS_ID

INTEGER

The numeric identifier (SRID) for the spatial reference system (SRS) associated with the spatial column

SRS_NAME

NVARCHAR(256)

The name of the SRS associated with the spatial column

DATA_TYPE_NAME

VARCHAR(16)

The spatial data type
