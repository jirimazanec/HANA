M_ES_DBSPACES System Views [Dynamic Tiering]

Shows the dbspace configured in extended storage.

Caution This information applies only for the SAP HANA option cited in the title of this topic. You must have purchased the license for it in order to use the information contained in this topic.

System and user data for the extended storage is managed in dbspaces, which can contain one or more files. This monitoring view provides information for the extended storage corresponding to the connected HANA database.

Only users with CATALOG READ, DATA ADMIN, or EXTENDED STORAGE ADMIN system privilege can view the content of the M_ES_DBSPACE view. The view is empty for all other database users.
