M_OUT_OF_MEMORY_EVENTS System View

Monitors events that run out of memory.

Structure

Column name

Data type

Description

HOST

VARCHAR ( 64 )

Host name

PORT

INTEGER

Internal port

TIME

TIMESTAMP

Time of OOM

CONNECTION_ID

INTEGER

Connection ID

STATEMENT_ID

VARCHAR ( 20 )

Statement ID

STATEMENT_HASH

VARCHAR ( 32 )

Identifier for an SQL string

HEAP_MEMORY_CATEGORY

VARCHAR ( 128 )

Allocator name in case of a heap memory failure

MEMORY_REQUEST_SIZE

BIGINT

The size in bytes of the failed memory allocation

MEMORY_USED_SIZE

BIGINT

The size in bytes currently allocated, depends on EVENT_REASON

EVENT_REASON

VARCHAR ( 32 )

Reason for OOM, e.g. GLOBAL ALLOCATION LIMIT, PROCESS ALLOCATION LIMIT, STATEMENT MEMORY LIMIT

MEMORY_LIMIT_SIZE

BIGINT

The size of the limit, depends on EVENT_REASON

TRACEFILE_NAME

VARCHAR ( 256 )

OOM trace file name
