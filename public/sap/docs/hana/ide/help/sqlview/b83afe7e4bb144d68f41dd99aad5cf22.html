M_DATABASE_REPLICAS System View

Provides source and target information for databases involved in replication.

Structure

Column name

Data type

Description

SOURCE_DATABASE_NAME

NVARCHAR(256)

Name of source database

SOURCE_SYSTEMDB_HOST

VARCHAR(64)

Host of source system database

SOURCE_SYSTEMDB_PORT

INTEGER

Port of source system database

TARGET_DATABASE_NAME

NVARCHAR(256)

Name of target database

TARGET_SYSTEMDB_HOST

VARCHAR(64)

Host of target system database

TARGET_SYSTEMDB_PORT

VARCHAR(12)

Port of target system database

REPLICATION_STATUS

Aggregated replication status of database services
