LCM_SOFTWARE_COMPONENTS System View

Provides all installed software component versions.

Structure

Column name

Data type

Description

VENDOR_NAME

VARCHAR(30)

Vendor, for example, sap.com

COMPONENT_NAME

VARCHAR(30)

Technical name of the software component version

VERSION

VARCHAR(30)

Version of the Delivery unit (=SWC), for example, 1

VERSION_SP

VARCHAR(16)

Number of the SP, for example, 3

VERSION_PATCH

VARCHAR(16)

Number of the patch, for example, 37

PPMS_ID

VARCHAR(20)

Technical ID of the software component version, for example, the one identifying it in PPMS

COMPONENT_DESCRIPTION

VARCHAR(64)

(Human-readable) name of the software component version

SP_NAME

VARCHAR(30)

Technical name of the support package, for example, SAPK-10000INATEST2 or SP000

SP_DESCRIPTION

VARCHAR(64)

(Human-readable) name of the support package

SP_PPMS_ID

VARCHAR(20)

20-digit PPMS number of the installed support package
