Full description

This view has a resettable counterpart. This means, you can also see the values since the last reset in the dependent view M_SAVEPOINT_STATISTICS_RESET. To reset the values, use the SQL command: ALTER SYSTEM RESET MONITORING VIEW SYS.M_SAVEPOINT_STATISTICS_RESET or the management console command: monitor reset -n M_SAVEPOINT_STATISTICS_RESET.

This view shows information about executed savepoints.

Columns START_TIME, STATE, VERSION and LAST_* relate to the last executed or currently-executing savepoint. Other columns contain aggregated values. Refer to the M_SAVEPOINTS system view for further information about various counters.
