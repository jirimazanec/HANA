Embedded Statistics Service Views

Tables and views from the _SYS_STATISTICS schema.

The following tables and views are from the _SYS_STATISTICS schema. They are used when you migrate to the embedded statistics service available with SAP HANA on Intel architecture and initial shipment stack on Power architecture.

For more information about how to migrate to the new mechanism after an upgrade, see SAP Note 1917938.

As of revision 93, the embedded statistics service is enabled by default after installation or upgrade. For more information, see SAP Note 2091313.
