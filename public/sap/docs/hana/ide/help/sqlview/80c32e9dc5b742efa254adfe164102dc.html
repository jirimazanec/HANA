M_EXPENSIVE_STATEMENT_EXECUTION_LOCATION_STATISTICS System View

Provides location statistics for expensive statements.

Structure

Column name

Data type

Description

HOST

VARCHAR(64)

Specifies the host name.

PORT

INTEGER

Specifies the internal port.

STATEMENT_EXECUTION_ID

BIGINT

Specifies the execution ID of the statement.

START_TIME

TIMESTAMP

Specifies the execution start time of the statement.

EXECUTION_HOST

VARCHAR(64)

Specifies the host where the statement was executed.

EXECUTION_PORT

INTEGER

Specifies the port of the host where the statement was executed.

MEMORY_SIZE

BIGINT

Specifies the actual memory size of each statement execution location.
