M_ADMISSION_CONTROL_STATISTICS System View

Provides the overall statistics values of the Session-Wise Admission Control feature.

Structure

Column name

Data type

Unit

Description

TOTAL_ADMIT_COUNT

BIGINT

Counter

Specifies the accumulated request admission count.

TOTAL_REJECT_COUNT

BIGINT

Counter

Specifies the accumulated request rejection count.

TOTAL_ENQUEUE_COUNT

BIGINT

Counter

Specifies the accumulated request queued count.

TOTAL_DEQUEUE_COUNT

BIGINT

Counter

Specifies the accumulated request dequeued count (the executed request count).

TOTAL_TIMEOUT_COUNT

BIGINT

Counter

Specifies the accumulated request dequeued count due to timeout (the rejected request count).

CURRENT_QUEUE_SIZE

BIGINT

Counter

Specifies the current waiting request queued count.

LAST_CPU_USAGE_RATIO

INTEGER

Ratio

Specifies the last measured value of the CPU usage ratio.

LAST_CPU_USAGE_MEASURE_TIME

TIMESTAMP

Specifies the time at which the last CPU usage ratio was measured.

LAST_MEMORY_SIZE

BIGINT

Byte

Specifies the last measured memory size in GB.

LAST_MEMORY_RATIO

BIGINT

Ratio

Specifies the last measured value of the memory ratio (last_memory_size/memory_allocation_limit).

LAST_MEMORY_MEASURE_TIME

TIMESTAMP

Specifies the time at which the last memory size was measured.

LAST_WAIT_TIME

BIGINT

Microsecond

Specifies the last wait time of the request in the queue.

AVG_WAIT_TIME

BIGINT

Microsecond

Specifies the average wait time of the request in the queue.

MAX_WAIT_TIME

BIGINT

Microsecond

Specifies the maximum wait time of the request in the queue.

MIN_WAIT_TIME

BIGINT

Microsecond

Specifies the minimum wait time of the request in the queue.

SUM_WAIT_TIME

BIGINT

Microsecond

Specifies the total wait time of the request in the queue.

HOST

VARCHAR(64)

Specifies the host name.

PORT

INTEGER

Specifies the internal port.
