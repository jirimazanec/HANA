ALTER SYSTEM START DATABASE Statement (Tenant Database Management)

Starts the specified database.

Syntax
ALTER SYSTEM START DATABASE <database_name>
Syntax Elements
<database_name>

Specifies the database to start.

<database_name> ::= <identifier>
Description

The use of this statement requires the DATABASE ADMIN privilege.

Example

Start the my_database database.

ALTER SYSTEM START DATABASE my_database;
