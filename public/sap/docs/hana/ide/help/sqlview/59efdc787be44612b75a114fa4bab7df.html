LCM_PRODUCTS System View

Provides information about all installed product versions.

Structure

Column name

Data type

Description

VENDOR_NAME

VARCHAR(30)

Vendor, for example, sap.com

PRODUCT_NAME

VARCHAR(30)

Technical name of the product

VERSION

VARCHAR(30)

Version of the PPMS product version, for example, 1.0

PPMS_ID

VARCHAR(30)

Technical ID of the product version, for example, the one identifying it in PPMS.

PRODUCT_DESCRIPTION

VARCHAR(30)

(Human-readable) name of the product

SP_STACK_ID

VARCHAR(30)

SP stack ID of the PPMS product version

SP_STACK_DESCRIPTION

VARCHAR(30)

(Human-readable) name/description of SP stack
