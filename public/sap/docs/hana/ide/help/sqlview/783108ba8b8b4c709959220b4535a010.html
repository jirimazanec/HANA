M_BACKUP_PROGRESS System View

Provides the progress of the most recent backup.

Structure

Column name

Data type

Description

BACKUP_ID

BIGINT

Unique ID of a data backup. All backup files of a single data backup share the same BACKUP_ID

HOST

VARCHAR(64)

Host name

PORT

INTEGER

Internal port

SERVICE_NAME

VARCHAR(32)

Name of the service

ENTRY_TYPE_NAME

VARCHAR(64)

Classification of the type of backup. Currently only 'complete data backup' is supported

SYS_START_TIME

TIMESTAMP

The local server time the backup started

UTC_START_TIME

TIMESTAMP

The time the backup started

SYS_END_TIME

TIMESTAMP

The local server time the backup was terminated

UTC_END_TIME

TIMESTAMP

The time the backup was terminated

STATE_NAME

VARCHAR(64)

Current state of the backup

BACKUP_SIZE

BIGINT

Total amount of data

TRANSFERRED_SIZE

BIGINT

Amount of data transferred
