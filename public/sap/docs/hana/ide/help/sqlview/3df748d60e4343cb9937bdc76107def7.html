JWT_PROVIDERS System View

Lists all of the JWT providers configured in the SAP HANA database.

Structure

Column name

Data type

Description

JWT_PROVIDER_NAME

NVARCHAR(256)

The name of the JWT provider

ISSUER_NAME

NVARCHAR(512)

The issuer name provided in the iss claim of the JWT tokens issued by this JWT provider

EXTERNAL_IDENTITY_CLAIM

NVARCHAR(256)

The claim provided in the JWT tokens to use for mapping the SAP HANA user to an external user name
