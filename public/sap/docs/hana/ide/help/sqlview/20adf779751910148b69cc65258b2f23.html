M_DATA_VOLUME_SUPERBLOCK_STATISTICS System View

Provides FreeBlockManager Superblock statistics.

Full description

This view shows information about the number and distribution of Superblocks inside DataVolumes. SUPERBLOCK_COUNT is the total number of Superblocks and USED_SUPERBLOCK_COUNT is the number of Superblocks currently occupied by at least one used or shadow page. FILL_RATIO gives the ratio of the number of used superblocks versus the total number of superblocks. The fill ratio of the superblocks themselves is not part of this formula.
