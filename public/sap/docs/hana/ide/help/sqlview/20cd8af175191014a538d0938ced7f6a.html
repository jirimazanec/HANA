ROLES System View

Shows available roles.

Structure

Column name

Data type

Description

ROLE_SCHEMA_NAME

NVARCHAR(256)

The name of the schema of the role granted

ROLE_NAME

NVARCHAR(256)

The role name

ROLE_ID

BIGINT

The role ID

ROLE_MODE

VARCHAR(5)

The mode of the role: LOCAL

GLOBAL_IDENTITY

NVARCHAR(256)

The identity specified for the role with ROLE_MODE GLOBAL

CREATOR

NVARCHAR(256)

The name of the user who created the role

CREATE_TIME

TIMESTAMP

The time that the role was created

CONTEXT

NVARCHAR(2048)

The context that the role is valid for
