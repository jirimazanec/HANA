M_STREAMING_PROJECTS System View [Smart Data Streaming]

Provides information and statistics about each streaming project deployed in the SAP HANA streaming cluster.

Caution This information applies only for the SAP HANA option cited in the title of this topic. You must have purchased the license for it in order to use the information contained in this topic.

Only users with a MONITOR role can view the content of this system view. The view is empty for all other users.

Structure
Column name Data type Unit Description
SCHEMA_NAME NVARCHAR(256)   Schema in which the project is deployed.
PROJECT_NAME NVARCHAR(256)   Name given to the project. It corresponds to the APPLICATION_NAME in the M_STREAMING_APPLICATIONS system view for APPLICATION_TYPE "PROJECT" or "HA_PROJECT".
PROJECT_INSTANCE INTEGER   Instance number of the project. It corresponds to the APPLICATION_INSTANCE in the M_STREAMING_APPLICATIONS system view for APPLICATION_TYPE "PROJECT" or "HA_PROJECT".
CPU_UTILIZATION REAL MONITOR_UNIT_PERCENT Total CPU usage, as a percentage, used by the project since the last update. Total CPU usage is equal to the sum of the system CPU usage plus the user CPU usage. Valid values range from 00 to 100.00. On multi-core machines, the percentage is relative to the total number of available cores. For example, a value of 100.00 indicates a usage of 100% on all cores on the machine.
CPU_SYSTEM_UTILIZATION REAL MONITOR_UNIT_PERCENT System CPU usage (Kernel usage on Windows), as a percentage, used by the project since the last update. Valid values range from 0.0 to 100.00. On multi-core machines, the percentage is relative to the total number of available cores. For example, a value of 100.00 indicates a usage of 100% of all cores on the machine.
CPU_USER_UTILIZATION REAL MONITOR_UNIT_PERCENT User CPU usage, as a percentage, used by the project since the last update. On multi-core machines, the percentage is relative to the total number of available cores. For example, a value of 100.00 indicates a usage of 100% on all cores on the machine.
TOTAL_CPU_TIME BIGINT MONITOR_UNIT_US Total CPU time for the project, in microseconds. Total CPU time is equal to the sum of the system CPU time plus the user CPU time.
TOTAL_CPU_SYSTEM_TIME BIGINT MONITOR_UNIT_US Total system CPU time for the project, in microseconds.
TOTAL_CPU_USER_TIME BIGINT MONITOR_UNIT_US Total user CPU time for the project, in microseconds.
PROJECT_DURATION BIGINT MONITOR_UNIT_US Duration of lapsed real time since the project was started, in microseconds.
VIRTUAL_MEMORY_SIZE BIGINT MONITOR_UNIT_BYTE Total amount of virtual memory, in bytes, used by the project at the time of the update.
RESIDENT_MEMORY_SIZE BIGINT MONITOR_UNIT_BYTE Total amount of system memory (RSS), in bytes, used by the project at the time of the update.
THREAD_COUNT INTEGER MONITOR_UNIT_COUNTER Total number of threads used by the project at the time of the update.
MONITORING_INTERVAL INTEGER MONITOR_UNIT_SEC Specifies, in seconds, how often stream and client statistics are gathered. A value of 0 indcates that no statistics are gathered for the streams and clients.
IS_ACTIVE_ACTIVE_ENABLED VARCHAR(5) MONITOR_UNIT_BOOL Indicates whether active-active high availability mode is enabled for the project.
ACTIVE_ACTIVE_ROLE VARCHAR(16)   Indicates whether the project instance plays the primary or secondary role in active-active high availability mode.
  • PRIMARY
  • SECONDARY
  • UNKNOWN (a transitional state)
LAST_UPDATE_TIME TIMESTAMP   Most recent time that statistics were updated for the project.
