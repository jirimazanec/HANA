M_BACKUP_SIZE_ESTIMATIONS System View

Provides the estimated size of the next data backup.

Structure

Column name

Data type

Description

HOST

VARCHAR(64)

Host name

PORT

INTEGER

Internal port

SERVICE_NAME

NVARCHAR(32)

Name of the service

ENTRY_TYPE_NAME

NVARCHAR(64)

Classification of the type of backup. The following types are supported: complete data backup, data snapshot, log backup, log missing, differential and incremental backups

ESTIMATED_SIZE

BIGINT

The estimated size of the backup in bytes

Note The actual backup size may differ from the estimation shown in this view.

The ENTRY_TYPE_NAME column is equivalent to the column in the M_BACKUP_CATALOG view. The table contains records for each service with a volume for all the supported backup types. If a complete data backup has not yet been written, a differential or incremental backup will not be possible. If no differential or incremental backups have been created, this table will not contain any records for differential or incremental backups.
