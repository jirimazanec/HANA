FIRST_VALUE Function (Aggregate)

Returns the value of the first element of an expression.

Syntax
FIRST_VALUE (<expression> ORDER BY <column>)
Syntax Elements
<expression>
Specifies the expression that the function returns the first element value of.
ORDER BY
Orders the result set according to the specified <column>,
Description

Returns the value of the first element in <expression> as ordered by <column>.

Null is returned if the value is null or if <expression> is empty.

The output of the FIRST_VALUE function can be non-deterministic among tie values.

Example

The example below returns the first value in the COL1 column when the table is ordered by COL2:

CREATE TABLE T (COL1 DOUBLE, COL2 DOUBLE);

INSERT INTO T VALUES(9, 1);
INSERT INTO T VALUES(4, 5);
INSERT INTO T VALUES(7, 3);

SELECT FIRST_VALUE (COL1 ORDER BY COL2) FROM T;

The query returns 9.
