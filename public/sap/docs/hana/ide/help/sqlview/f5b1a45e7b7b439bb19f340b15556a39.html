HOST_RS_MEMORY System View (Embedded Statistics Service)

RS memory per host. This table contains information only for the last 42 days.

Structure

Column name

Data type

Description

SNAPSHOT_ID TIMESTAMP Timestamp ID (UTC)
SERVER_TIMESTAMP TIMESTAMP Processing Timestamp (local time)
INDEX VARCHAR(204) Index
ALLOCATED_SIZE BIGINT Allocated memory size of the module
CATEGORY VARCHAR(128) Module name
FREE_SIZE BIGINT Free memory size of the module
HOST VARCHAR(64) Host name
PORT INTEGER Internal port
USED_SIZE BIGINT Used memory size of the module
