Hierarchy Functions

Hierarchy functions allow you to work with hierarchical data such as tables with rows arranged in a tree or directed graph.

Hierarchy functions behave similarly to conventional user-defined or built-in table functions at runtime, and their input tables or subqueries are allowed to have any result set signature.

Supported functions

Hierarchy functions are grouped into several categories: generation, navigation, and scalar. Generator functions create hierarchy structures. Navigation and generator functions are table functions, they are usually applied in a FROM clause instead of a table or view reference. Whereas scalar functions like HIERARCHY_COMPOSITE_ID is a scalar function, and are normally used in a SELECT clause projection list. Scalar functions return a single (calculated) column instead of a tabular result.
