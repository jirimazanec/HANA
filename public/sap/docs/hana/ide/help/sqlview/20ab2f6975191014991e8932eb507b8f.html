M_CONDITIONAL_VARIABLES System View

Provides semaphore statistics.

Full description

This view has a resettable counterpart. This means, you can also see the values since the last reset in the dependent view M_CONDITIONAL_VARIABLES_RESET. To reset the values, use the SQL command: ALTER SYSTEM RESET MONITORING VIEW SYS.M_CONDITIONAL_VARIABLES_RESET or the management console command: monitor reset -n M_CONDITIONAL_VARIABLES_RESET

This view contains information about single conditional variable objects or groups of conditional variable objects. It does not contain information about all conditional variables.
