DROP JWT PROVIDER Statement (Access Control)

Drops a JWT provider in the SAP HANA database.

Syntax
DROP JWT PROVIDER <jwt_provider_name>
Syntax Elements
<jwt_provider_name>

Specifies which JWT provider to drop.

<jwt_provider_name> ::= <simple_identifier>
Description

Only database users that have the USER ADMIN system privilege can drop a JWT provider.

If the JWT provider is in use by an SAP HANA database user, the JWT provider cannot be dropped.

Examples

Drops a JWT provider, my_jwt_provider.

DROP JWT PROVIDER my_jwt_provider;
