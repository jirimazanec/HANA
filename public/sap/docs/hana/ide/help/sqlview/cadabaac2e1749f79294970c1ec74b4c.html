STATISTICS_OBJECTS System View (Embedded Statistics Service)

Statistics of objects per host. This table contains information only for the last 7 days.

Structure

Column name

Data type

Description

ID SMALLINT Unique ID of installed procedure
NAME NVARCHAR(1024) Catalog name in catalog of installed procedure
TYPE NVARCHAR(256) Type of installed procedure. The following values are possible: Alert, Collector, Shared, SpecialFunction, Preamble, Appendix.
VERSION SMALLINT Version of installed procedure
REACHED_AT TIMESTAMP Time at which the current version was reached (in UTC)
