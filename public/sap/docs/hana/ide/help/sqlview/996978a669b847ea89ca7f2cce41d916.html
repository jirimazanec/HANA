CREATE WORKLOAD MAPPING Statement (Workload Management)

Defines workload mappings.

Syntax
CREATE WORKLOAD MAPPING  <mapping_name> WORKLOAD CLASS <wc_name> <property_list>
Syntax Elements
<mapping_name>

Specifies the workload mapping to be created.

<mapping_name> ::= <identifier>
<wc_name>

Specifies the workload class name.

<wc_name> ::= <identifier>
<property_list>

Defines the properties of a workload mapping. 

<property_list> ::=
 [SET | UNSET] '<key>' = '<value>' [...]
<key> ::= 
 APPLICATION USER NAME 
 | CLIENT 
 | APPLICATION NAME 
 | USER NAME }

value ::= <string_literal>

The properties have the format of key-value pairs.

Description

This statement defines workload mappings. The use of this statement requires the WORKLOAD_ADMIN privilege.

Examples

Create a new entry for the workload mapping “MyWorkloadMapping” for the database user ‘ABCADM’ and the application ‘BW’.

CREATE WORKLOAD MAPPING "MyWorkloadMapping" WORKLOAD CLASS "MyWorkloadClass" SET 'USER NAME' = 'ABCADM', 'APPLICATION NAME' = 'BW';
