SEARCH_RULE_SET_CONDITIONS System View

Shows conditions to available search rule sets.

Structure

Column name

Data type

Description

SEARCH_RULE_SET_SCHEMA_NAME

NVARCHAR(256)

The name of the runtime schema of the search rule set

SEARCH_RULE_SET_PACKAGE_ID

NVARCHAR(256)

The name of the repository package of the search rule set

SEARCH_RULE_SET_NAME

NVARCHAR(256)

The name of the search rule set

RULE_NUMBER

INTEGER

The number of the rule that contains the column

RULE_ID

NVARCHAR(256)

The unique ID/name of the rule that contains the column

SEARCHED_COLUMN_NAME

NVARCHAR(256)

The name of a column that is used by a search rule set

SEARCHED_COLUMN_CONDITION

NVARCHAR(20)

The condition. Possible values are EQUALS, NOT EMPTY, and MISSING

SEARCHED_COLUMN_VALUE

NVARCHAR(5000)

The value the user input is compared to, if condition is EQUALS

SEARCHED_COLUMN_VALUE_REPLACED_BY

NVARCHAR(5000)

The user input is replaced by this value if the action is REPLACE

ACTION

NVARCHAR(20)

The action that is performed when the condition is true. Possible values are SKIP COLUMN, SKIP RULE, and REPLACE
