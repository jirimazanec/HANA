M_LOAD_HISTORY_INFO System View

Load history KPI description.

Structure
Column name Data type Description
VIEW_NAME NVARCHAR(256) View name where the KPI can be found
COLUMN_NAME NVARCHAR(256) Column name where the KPI can be found
IS_CUMULATIVE VARCHAR(5) Cumulative KPIs return a relative value since the previous sample. Noncumulative KPIs return an absolute value for the current point in time
SQL_SOURCE VARCHAR(256) Equivalent SQL statement
DESCRIPTION VARCHAR(256) KPI description
SAMPLE_UNIT VARCHAR(16) Sample unit
DISPLAY_HIERARCHY VARCHAR(8) Recommended display hierarchy / sorting criteria
DISPLAY_NAME VARCHAR(32) Recommended display name
DISPLAY_UNIT VARCHAR(16) Recommended display unit
DISPLAY_LINE_COLOR INTEGER Recommended display line color as RGB
DISPLAY_LINE_STYLE TINYINT Recommended display line style: 1=solid, 2=dotted, 3=dashed
DISPLAY_DIVIDER INTEGER Divider to convert SAMPLE_UNIT in DISPLAY_UNIT
DISPLAY_Y_SCALE TINYINT KPIs with same value should be shown with same Y scale in a chart. Special value 1 is used for CPU KPIs with fixed Y scale of 100
