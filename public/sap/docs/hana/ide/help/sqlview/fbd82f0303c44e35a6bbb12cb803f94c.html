HOST_RESOURCE_UTILIZATION_STATISTICS System View (Embedded Statistics Service)

Host resource utilization. The table contains information only for the last 42 days.

Structure

Column name

Data type

Description

SNAPSHOT_ID TIMESTAMP Timestamp (UTC) identifying when the statistics service performed a data collection. Data collections scheduled for the same time have the same snapshot ID.
SERVER_TIMESTAMP TIMESTAMP Timestamp (local server time) identifying when a data collection actually occurred on the server.
INDEX VARCHAR(64) Unique identifier for a row of the result set that a particular snapshot of a data collection represents.
TOTAL_CPU_IDLE_TIME_DELTA BIGINT Total CPU idle time
TOTAL_CPU_SYSTEM_TIME_DELTA BIGINT Total CPU system time
TOTAL_CPU_USER_TIME_DELTA BIGINT Total CPU user time
TOTAL_CPU_WIO_TIME_DELTA BIGINT Total IO time
SNAPSHOT_DELTA BIGINT Time elapsed
ALLOCATION_LIMIT BIGINT Allocation limit for all processes
FREE_PHYSICAL_MEMORY BIGINT Free physical memory on the host
FREE_SWAP_SPACE BIGINT Free swap memory on the host
HOST VARCHAR(64) Host name
INSTANCE_CODE_SIZE BIGINT Code size, including shared libraries of SAP HANA processes
INSTANCE_SHARED_MEMORY_ALLOCATED_SIZE BIGINT Shared memory size of SAP HANA processes
INSTANCE_TOTAL_MEMORY_ALLOCATED_SIZE BIGINT Size of the memory pool for all SAP HANA processes
INSTANCE_TOTAL_MEMORY_USED_SIZE BIGINT Amount of memory from the memory pool that is currently in actual use by SAP HANA processes
USED_PHYSICAL_MEMORY BIGINT Used physical memory on the host
USED_SWAP_SPACE BIGINT Used swap memory on the host
