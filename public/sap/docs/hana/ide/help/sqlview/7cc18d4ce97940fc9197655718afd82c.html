M_STREAMING_APPLICATIONS System View [Smart Data Streaming]

Provides a list of streaming applications running in the SAP HANA streaming cluster.

Caution This information applies only for the SAP HANA option cited in the title of this topic. You must have purchased the license for it in order to use the information contained in this topic.

Only users with a MONITOR role can view the content of this system view. The view is empty for all other users.

Structure
Column name Data type Unit Description
SCHEMA_NAME NVARCHAR(256)   The schema in which the application is deployed.
APPLICATION_NAME NVARCHAR(256)   The name of the streaming application.
APPLICATION_TYPE VARCHAR(32)   Application type. For example PROJECT, HA_PROJECT, TOOLKIT_ADAPTER
APPLICATION_INSTANCE INTEGER   The instance number of the streaming application. A schema may have multiple instances of an application running. As, for example, in HA or distributed processing.
APPLICATION_INSTANCE_COUNT INTEGER   The number of instances that are running or will be started.
SERVICE_ALIAS NVARCHAR(256)   Name of the service on which the application is running.
REQUESTED_STATUS VARCHAR(32)   The status the application should be in, as requested by the client.
  • STOPPED
  • STARTED_RUNNING
CURRENT_STATUS VARCHAR(32)   The status the application is currently in.
  • START
  • STARTING
  • STARTED
  • STARTED_RUNNING
  • STOPPING
  • STOPPED
  • STOPPED_FAILED
  • STOPPED_FAILED_START
IS_FAILOVER_ENABLED VARCHAR(5)   Indicates whether the project should be started on another node if the project or the node fails.
  • TRUE
  • FALSE
FAILURE_COUNT INTEGER   The number of failures that have occurred during the failure interval.
FAILURE_INTERVAL INTEGER   The interval, in seconds, during which, if the failure count reaches the maximum failures per interval, the application is shut down
MAX_FAILURES_PER_INTERVAL INTEGER   The number of failures that can occur in the specified failure interval before the application is shut down.
