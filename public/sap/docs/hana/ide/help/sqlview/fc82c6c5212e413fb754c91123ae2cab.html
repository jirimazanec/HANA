M_TIMEZONE_ALERTS System View

Alerts relating to status of SAP HANA internal timezone conversion.

Structure

Column name

Data type

Description

HOST

VARCHAR(64)

Host name

PORT

INTEGER

Internal port

STATUS

VARCHAR(32)

Status to report: TABLES NOT FOUND, TABLES EMPTY, NO SYSTEM TIMEZONE, SYSTEM TIMEZONE DISABLED, CONVERSION MISMATCH

TIMEZONE_NAME

NVARCHAR(64)

Name of timezone to which status message applies or NULL if status does not apply to a specific timezone

MISMATCH_BEGIN

TIMESTAMP

UTC Time at which first mismatch of mismatch period was found to occur or NULL if no mismatch applies

MISMATCH_END

TIMESTAMP

UTC Time at which last mismatch of mismatch period was found to occur or NULL if no mismatch applies

DETAILS

NVARCHAR(256)

Detail information
