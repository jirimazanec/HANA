M_VOLUME_FILES System View

Provides information about volume files.

Structure

Column name

Data type

Description

HOST

VARCHAR(64)

Specifies the host name.

PORT

INTEGER

Specifies the internal port.

VOLUME_ID

INTEGER

Specifies the persistence volume ID.

FILE_TYPE

VARCHAR(16)

Specifies the type of file (DATA, LOG, and so on).

FILE_NAME

VARCHAR(256)

Specifies the file name.

USED_SIZE

BIGINT

Specifies the size of used data within the file.

TOTAL_SIZE

BIGINT

Specifies the total file size.
Full description

Specifies information about files in the volume directories. All files in these directories are shown, but only registered files (files currently used by the database) have a file type. TOTAL_SIZE is the size as reported by the file system.

The meaning of USED_SIZE depends on the file type:

  • DATA: Specifies the size of used and shadow pages in the data volume file.

  • LOG: Specifies that the used size equals the TOTAL_SIZE.
