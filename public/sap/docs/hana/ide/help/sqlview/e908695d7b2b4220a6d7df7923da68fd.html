HOST_LIVECACHE_SCHEMA_STATISTICS System View (Embedded Statistics Service)

LiveCache schema statistics per host. This table contains information only for the last 21 days. The collection interval is 30 minutes.

Structure

Column name

Data type

Description

SNAPSHOT_ID TIMESTAMP Timestamp (UTC) identifying when the statistics service performed a data collection. Data collections scheduled for the same time have the same snapshot ID.
SERVER_TIMESTAMP TIMESTAMP Timestamp (local server time) identifying when a data collection actually occurred on the server.
OMS_SCHEMA_HANDLE INTEGER OMS schema handle
OMS_SCHEMA_NAME VARCHAR(96) OMS schema name
CONTAINER_COUNT BIGINT Container count
OBJECT_COUNT BIGINT Object count (incl. deleted objects)
OBJECT_SIZE_SUM BIGINT Transient object size sum (incl. deleted objects)
GC_PENDING_OBJECT_DELETE_COUNT BIGINT Count of objects that are deleted and not yet freed by garbage collection
GC_PENDING_OBJECT_DELETE_SIZE_SUM BIGINT Transient size sum of objects that are deleted and not yet freed by garbage collection
HISTORY_OBJECT_COUNT BIGINT Count of transient history object versions that are not yet freed by garbage collection
HISTORY_SIZE_SUM BIGINT Size sum of transient history object versions that are not yet freed by garbage collection
PAGE_SIZE_SUM BIGINT Persistent page size sum
HEAP_USAGE BIGINT Memory that is currently used by the container
OBJECT_CREATE_COUNT BIGINT Number of object creations since restart
OBJECT_UPDATE_COUNT BIGINT Number of object updates since restart
OBJECT_DELETE_COUNT BIGINT Number of object deletions since restart
