PROCEDURE_ROUTES System View

Provides information about the procedure being routed. This view is for internal use only.

Structure

Column name

Data type

Description

PROCEDURE_SCHEMA

NVARCHAR(256)

Specifies the schema name of the stored procedure.

PROCEDURE_NAME

NVARCHAR(256)

Specifies the name of the stored procedure.

OBJECT_TYPE

VARCHAR(13)

Specifies the object type. Can be specified as: REMOTE_SOURCE, VOLUME, or TABLE.

OBJECT_SCHEMA

NVARCHAR(256)

Specifies the schema of the object, if applicable.

OBJECT_NAME

NVARCHAR(256)

Specifies the object name.

VOLUME_ID

INTEGER

Specifies the volume ID.
