GROUPING Function (Miscellaneous)

Determines whether a specified column is used in grouping.

Syntax
GROUPING(<column_name>)
Description

The GROUPING function can be used with GROUPING SETS, ROLLUP, or CUBE, which return multiple levels of aggregation in a single result set. The function returns 1 if the specified column is used in grouping, and 0 otherwise. The column of GROUPING must be an element of the GROUPING SETS.

Example

The following statement uses the fictitious table guided_navi_tab and the result set indicates whether the results are grouped by customer, year, and/or product.

SELECT customer, year, product, SUM(sales), 
   GROUPING(customer), GROUPING(year), GROUPING(product)
   FROM guided_navi_tab
   GROUP BY GROUPING SETS (
   (customer, year, product),
   (customer, year),
   (customer, product),
   (year, product),
   (customer),
   (year),
   (product));
