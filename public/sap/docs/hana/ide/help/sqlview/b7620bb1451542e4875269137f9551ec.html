HOST_LONG_SERIALIZABLE_TRANSACTION System View (Embedded Statistics Service)

Transactions created by users that are started and running. The table contains information only for the last 42 days.

Structure

Column name

Data type

Unit

Description

SNAPSHOT_ID TIMESTAMP Datetime Timestamp (UTC) identifying when the statistics service performed a data collection. Data collections scheduled for the same time have the same snapshot ID.
SERVER_TIMESTAMP TIMESTAMP Datetime Timestamp (local server time) identifying when a data collection actually occurred on the server.
INDEX VARCHAR(21) Unique identifier for a row of the result set that a particular snapshot of a data collection represents.
TOTAL_TIME BIGINT Millisecond Time elapsed
CLIENT_HOST VARCHAR(64) Host name of client machine
CLIENT_PID BIGINT Client Process ID
CONNECTION_STATUS VARCHAR(128) Connection Status: 'RUNNING'/'IDLE'
LOGICAL_CONNECTION_ID INTEGER Logical connection ID in statement routing
HOST VARCHAR(64) Host name
MIN_MVCC_SNAPSHOT_TIMESTAMP BIGINT Logical timestamp of the first executed statement in the transaction
PORT INTEGER Internal port
TRANSACTION_ID INTEGER Transaction object ID (This number is reused after a transaction is closed)
UPDATE_TRANSACTION_ID BIGINT Write transaction ID (This number is ever increasing)
