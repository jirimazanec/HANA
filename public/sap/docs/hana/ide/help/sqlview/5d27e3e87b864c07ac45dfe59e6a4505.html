HOST_LONG_IDLE_CURSOR System View (Embedded Statistics Service)

Long idle cursor on SQL statement. The table contains information only for the last 42 days.

Structure

Column name

Data type

Description

SNAPSHOT_ID TIMESTAMP Timestamp (UTC) identifying when the statistics service performed a data collection. Data collections scheduled for the same time have the same snapshot ID.
SERVER_TIMESTAMP TIMESTAMP Timestamp (local server time) identifying when a data collection actually occurred on the server.
INDEX VARCHAR(267) Unique identifier for a row of the result set that a particular snapshot of a data collection represents.
IDLE_TIME BIGINT Idle time (in seconds)
HOST VARCHAR(64) Host name
PORT INTEGER Internal port
START_MVCC_TIMESTAMP BIGINT Internal MVCC timestamp of the transaction start time
STATEMENT_STATUS VARCHAR(128) Status of SQL statement
STATEMENT_STRING NCLOB SQL statement
CLIENT_HOST VARCHAR(64) Host name of client machine
CLIENT_PID BIGINT Client Process ID
CONNECTION_STATUS VARCHAR(128) Connection Status: 'RUNNING'/'IDLE'
LOGICAL_CONNECTION_ID INTEGER Logical connection ID in statement routing
APPLICATION NVARCHAR(512) Value of a session context variable
APPLICATIONVERSION NVARCHAR(512) Value of a session context variable
APPLICATIONSOURCE NVARCHAR(512) Value of a session context variable
APPLICATIONUSER NVARCHAR(512) Value of a session context variable
