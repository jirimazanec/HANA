ALTER SYSTEM {ENABLE | DISABLE} SYSTEM HINT Statement (System Management)

Enables or disables hints from a target query.

Syntax
ALTER SYSTEM ENABLE STATEMENT HINT ( FOR <target_query> | ALL );
| ALTER SYSTEM DISABLE STATEMENT HINT ( FOR <target_query> | ALL );
Syntax Elements
<target_query>

Specifies the target query.

ALL
Applies the action to both user-defined hints and system hints that are associated with the specified statement. If ALL is not specified, then the action is only applied on the user-defined hints.
Description

Both statements (ENABLE, DISABLE) also evict all cached plans for the target SQL statements from the SQL Plan Cache.
