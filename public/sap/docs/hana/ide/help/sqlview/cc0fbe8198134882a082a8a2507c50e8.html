M_ADMISSION_CONTROL_QUEUES System View

Provides detailed information regarding queued session requests by Session-Wise Admission Control.

Structure

Column name

Data type

Description

MESSAGE_ID

BIGINT

Specifies the request ID of the client packet.

CONNECTION_ID

INTEGER

Specifies the session connection ID.

MESSAGE_TYPE

VARCHAR(32)

Specifies the request type.

STATEMENT_TYPE

VARCHAR(8)

Specifies the statement type (for example, DDL or DML).

ENQUEUE_TIME

TIMESTAMP

Specifies the request queue time.

TIMEOUT_TIME

TIMESTAMP

Specifies the time when the request should be timed-out.

HOST

VARCHAR(64)

Specifies the host name.

PORT

INTEGER

Specifies the internal port.
