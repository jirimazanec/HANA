GRAPH_WORKSPACES System View

Provides information about graph workspaces in the database.

Column name

Data type

Description

SCHEMA_NAME

NVARCHAR ( 256 )

Name of the schema the workspace resides in

WORKSPACE_NAME

NVARCHAR ( 256 )

Workspace name

CREATE_TIMESTAMP

Workspace creation timestamp

USER_NAME

NVARCHAR ( 256 )

Workspace creator user name

EDGE_SCHEMA_NAME

NVARCHAR ( 256 )

The schema name of workspace edge table

EDGE_TABLE_NAME

NVARCHAR ( 256 )

Workspace edge table name

EDGE_SOURCE_COLUMN_NAME

NVARCHAR ( 256 )

Workspace source column name in the edge table

EDGE_TARGET_COLUMN_NAME

NVARCHAR ( 256 )

Workspace target column name in the edge table

EDGE_KEY_COLUMN_NAME

NVARCHAR ( 256 )

Workspace key column name in the edge table

VERTEX_SCHEMA_NAME

NVARCHAR ( 256 )

The schema name of workspace vertex table

VERTEX_TABLE_NAME

NVARCHAR ( 256 )

Workspace vertex table name

VERTEX_KEY_COLUMN_NAME

NVARCHAR ( 256 )

Workspace key column name in the vertex table

IS_VALID

VARCHAR ( 5 )

Specifies whether this workspace is valid: TRUE/FALSE
