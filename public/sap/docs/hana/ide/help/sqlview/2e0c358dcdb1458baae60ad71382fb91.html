STATISTICS_ALERT_INFORMATION System View (Embedded Statistics Service)

Statistics alerts per host. This table contains information only for the last 7 days.

Structure

Column name

Data type

Description

ALERT_ID INTEGER Unique check ID
ALERT_NAME NVARCHAR(64) Short name of check
ALERT_DESCRIPTION NVARCHAR(1024) Check description
ALERT_USERACTION NVARCHAR(1024) Recommended action to be taken by an administrator when the check issues an alert
ALERT_CATEGORY TINYINT Category ID of the check
