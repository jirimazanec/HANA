ALTER SYSTEM STOP DATABASE Statement (System Management)

Stops a database.

Syntax
ALTER SYSTEM STOP DATABASE  <database_name> [IMMEDIATE [WITH COREFILE]]
Syntax Elements
<database_name>

Specifies the name of the database to stop.

IMMEDIATE

Immediately stops the database without waiting for a regular shutdown.

WITH COREFILE

Writes a core dump file.

Description

Executing this statement stops , all of the services that belong to the database on all hosts. This statement can only be issued from the SYSTEM database (systemDB) and is only available for SAP HANA tenant databases. You need the DATABASE ADMIN privilege to use this statement.

Example

Stop the database named MyHANADB. 

ALTER SYSTEM STOP DATABASE 'MyHANADB';
