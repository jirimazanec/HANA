CS_ALL_COLUMNS System View

Provides information from all columns of column tables, including internal ones.

Structure

Column name

Data type

Description

SCHEMA_NAME

NVARCHAR(256)

Schema name

TABLE_NAME

NVARCHAR(256)

Table name

COLUMN_NAME

NVARCHAR(256)

Available table columns

POSITION

INTEGER

Ordinal position of the parameter

GENERATION_TYPE

VARCHAR(22)

ALWAYS AS if the column is generation column, ALWAYS AS IDENTITY/BY DEFAULT AS IDENTITY if the column is an identity column whose values are always generated or generated by default, NULL otherwise

GENERATED_ALWAYS_AS

NVARCHAR(1000)

Specifies the expression of the column created by GENERATED ... AS

INTERNAL_ATTRIBUTE_TYPE

VARCHAR(32)

Attribute type for internal columns, otherwise NULL
