M_SQL_PLAN_CACHE_EXECUTION_LOCATION_STATISTICS System View

Provides statistics for hosts where plans were executed.

This view has a resettable counterpart. This means, you can also see the values since the last reset in the dependent view M_SQL_PLAN_CACHE_EXECUTION_LOCATION_STATISTICS_RESET. To reset the values, use the SQL command: ALTER SYSTEM RESET MONITORING VIEW SYS.M_SQL_PLAN_CACHE_EXECUTION_LOCATION_STATISTICS_RESET or the management console command: monitor reset -n M_SQL_PLAN_CACHE_EXECUTION_LOCATION_STATISTICS_RESET.
