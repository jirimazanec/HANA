HOST_LIVECACHE_OMS_VERSIONS System View (Embedded Statistics Service)

LiveCache OMS versions per host status information. This table contains information only for the last 21 days. The collection interval is 30 minutes.

Structure

Column name

Data type

Description

SNAPSHOT_ID TIMESTAMP Timestamp (UTC) identifying when the statistics service performed a data collection. Data collections scheduled for the same time have the same snapshot ID.
SERVER_TIMESTAMP TIMESTAMP Timestamp (local server time) identifying when a data collection actually occurred on the server.
AGE_IN_MINUTES INTEGER Age of the OMS version in minutes
VERSION_ID VARCHAR(22) ID of the OMS version
CREATE_DATE TIMESTAMP Timestamp of the creation of the OMS version
LAST_OPEN_DATE TIMESTAMP Timestamp, when the OMS version was opened last
IS_OPEN VARCHAR(5) Flag indicating whether the version is currently open: 'TRUE'/'FALSE'
CONNECTION_ID INTEGER ID of the connection/session the version is bound to
LIVECACHE_MVCC_SNAPSHOT_TIMESTAMP BIGINT Consistent view mvcc snapshot timestamp
HEAP_USAGE BIGINT Memory that is currently used by the version
VERSION_DESCRIPTION VARCHAR(512) Description
