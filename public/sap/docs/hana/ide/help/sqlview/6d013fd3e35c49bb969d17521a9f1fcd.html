M_STREAMING_PROJECT_ADAPTERS System View [Smart Data Streaming]

Contains information about internal adapters attached to each SAP HANA streaming project

Caution This information applies only for the SAP HANA option cited in the title of this topic. You must have purchased the license for it in order to use the information contained in this topic.

Only users with a MONITOR role can view the content of this system view. The view is empty for all other users.

Structure
Column name Data type Unit Description
SCHEMA_NAME NVARCHAR(256)   Schema in which the project is deployed
PROJECT_NAME NVARCHAR(256)   Name given to the project. Corresponds to the APPLICATION_NAME in the M_STREAMING_APPLICATIONS System View for APPLICATION_TYPE "PROJECT" or "HA_PROJECT".
PROJECT_INSTANCE INTEGER   Instance number of the project..Corresponds to the APPLICATION_INSTANCE in the M_STREAMING_APPLICATIONS System View for APPLICATION_TYPE "PROJECT" or "HA_PROJECT".
ADAPTER_NAME NVARCHAR(256)   The name of the adapter, as defined in the project.
ADAPTER_TYPE VARCHAR(32)   The adapter type defined in the ATTACH ADAPTER statement.
STREAM_NAME NVARCHAR(256)   The name of the stream on which the adapter is defined.
IS_INPUT VARCHAR(5) MONITOR_UNIT_BOOL Indicates whether the connection is an input adapter or an output adapter. Values are TRUE for input connection and FALSE for output connection.
STARTUP_GROUP_NAME NVARCHAR(256)   The start up group where this connector belongs.
ADAPTER_STATE VARCHAR(32)   The state of the adapter.
  • READY – ready to be started
  • INITIAL – performing start-up and initialization
  • CONTINUOUS – continuously receiving real-time data
  • IDLE – currently not receiving data but attempting to re-connect the to the data source or link
  • DONE – no remaining input or output data; the adapter is about to exit
  • DEAD – the adapter thread has exited. The adapter remains in this state until explicitly requested to restart.
TOTAL_ROW_COUNT BIGINT MONITOR_UNIT_COUNTER The total number of data records recognized in the input data.
SUCCESSFUL_ROW_COUNT BIGINT MONITOR_UNIT_COUNTER The number of data records successfully processed.
UNSUCCESSFUL_ROW_COUNT BIGINT MONITOR_UNIT_COUNTER The number of data records that experienced errors.
MEMORY_USAGE BIGINT MONITOR_UNIT_BYTE The total memory used by the adapter, in bytes.
LAST_ERROR_TIME TIMESTAMP   The last time that an error occurred.
LAST_ERROR_MESSAGE NVARCHAR(1024)   The complete text of the error message as written to the log.
LATENCY BIGINT MONITOR_UNIT_US The latency introduced by the adapter. For an input adapter, this is the amount of time it takes the adapter to receive data from its source and publish the data to the stream. For an output adapter, this is the amount of time it takes for the adapter to receive a message from the stream and publish the data to its destination.
