HOST_LONG_RUNNING_STATEMENTS System View (Embedded Statistics Service)

Information on long running system operations per host. The table contains information only for the last 7 days.

Structure

Column name

Data type

Unit

Description

SNAPSHOT_ID TIMESTAMP Datetime Timestamp (UTC) identifying when the statistics service performed a data collection. Data collections scheduled for the same time have the same snapshot ID.
SERVER_TIMESTAMP TIMESTAMP Datetime Timestamp (local server time) identifying when a data collection actually occurred on the server.
INDEX VARCHAR(95) Unique identifier for a row of the result set that a particular snapshot of a data collection represents.
APPLICATION_USER_NAME NVARCHAR(256) Application user name
CONNECTION_ID INTEGER Connection ID
DURATION BIGINT Millisecond Thread Duration
HOST VARCHAR(64) Host name
PORT INTEGER Internal port
THREAD_DETAIL NCLOB Thread detail
THREAD_ID BIGINT Thread ID
USER_NAME NVARCHAR(256) SQL user name
AUTO_COMMIT VARCHAR(5) Commit mode of the current transaction: TRUE if the current connection is in auto-commit mode, FALSE otherwise
CLIENT_HOST VARCHAR(64) Host name of client machine
CLIENT_IP VARCHAR(16) IP of client machine
CLIENT_PID BIGINT Client Process ID
CONNECTION_START_TIME TIMESTAMP Datetime Connected Time
TRANSACTION_ID INTEGER Transaction object ID
UPDATE_TRANSACTION_ID BIGINT Number Write transaction ID (This number is ever increasing)
