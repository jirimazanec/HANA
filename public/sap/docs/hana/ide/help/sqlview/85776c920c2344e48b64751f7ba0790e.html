M_ES_DELTA_MEMORY System View [Dynamic Tiering]

Delta memory utilization inSAP HANA dynamic tiering.

Caution This information applies only for the SAP HANA option cited in the title of this topic. You must have purchased the license for it in order to use the information contained in this topic.
Structure
Column name Data type Description
SCHEMA_NAME NVARCHAR(256) Schema name.
TABLE_NAME NVARCHAR(256) Table name.
FRAGMENT_COUNT INTEGER Number of store fragments for this table.
TOTAL_SIZE INTEGER Total delta memory used in MB by this table.
DATA_SIZE INTEGER Delta memory used in MB to store the table data.
DICTIONARY_SIZE INTEGER Delta memory used in MB to store the dictionaries for this table.
BITMAP_SIZE INTEGER Delta memory used in MB to store table level bitmaps.
RIDSPACE_INDEX_SIZE BIGINT Size of the RID space index.
TRANSACTION_ID BIGINT Associated transaction ID.
