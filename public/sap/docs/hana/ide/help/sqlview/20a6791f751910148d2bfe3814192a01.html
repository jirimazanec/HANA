SQL Statements

SAP HANA supports many SQL statements to allow you to do such tasks as create database objects, administer your system, and manipulate data.


    Alphabetical List of Statements
    Access Control Statements
    Access control statements enable database administrators to create, alter, and drop access to the SAP HANA database.
    Backup and Recovery Statements
    Backup and recovery statements allow users to perform backup and recovery operations.
    Data Definition Statements
    Data definition statements define structures in the SAP HANA database.
    Data Import Export Statements
    The following statements import and export data to and from the SAP HANA database.
    Data Manipulation Statements
    The following statements enable you to manipulate data within the SAP HANA database.
    JSON Document Store Statements
    The following statements enable you to use the SAP HANA database as a JSON document store.
    Procedural Statements
    Procedural statements manage system and user-defined procedures for the SAP HANA database.
    Session Management Statements
    The following SQL statements manage database sessions.
    System Management Statements
    The following statements enable you to manage system configuration settings in your SAP HANA database.
    Tenant Database Management Statements
    The following SQL statements manage SAP HANA tenant databases.
    Transaction Management Statements
    The following SQL statements manage transactions in the SAP HANA database.
    Workload Management Statements
    The following SQL statements manage workload classes and mappings.
