M_WORKLOAD_REPLAYS System View

Provides information about workload replays.

Structure

Column name

Data type

Description

CAPTURE_ID

BIGINT

The unique ID of the captured workload.

CAPTURE_SYSTEM_ID

VARCHAR (3)

The system ID in which the capture has been done.

CAPTURE_DATABASE_NAME

NVARCHAR (256)

The name of the database in which the capture has been done.

REPLAY_ID

BIGINT

The unique ID of the replayed workload.

REPLAY_NAME

NVARCHAR(256)

The user-specified name of the replayed workload.

REPLAY_DESCRIPTION

NVARCHAR(5000)

The user-specified description of the replayed workload.

REPLAY_START_TIME

TIMESTAMP

The start timestamp of the replay.

REPLAY_UTC_START_TIME

TIMESTAMP

The UTC start timestamp of the replay.

REPLAY_END_TIME

TIMESTAMP

The end timestamp of the replayed workload.

REPLAY_UTC_END_TIME

TIMESTAMP

The UTC end timestamp of the replayed workload.

REPLAY_HOST

VARCHAR (64)

The host name in which the replay has been done.

REPLAY_PORT

INTEGER

The service port number in which the replay has been done.

REPLAY_SYSTEM_ID

VARCHAR (3)

The system ID in which the replay has been done.

REPLAY_DATABASE_NAME

NVARCHAR (256)

The name of the database in which the replay has been done.

STATUS

VARCHAR (16)

The status of workload replays.

PROGRESS

DOUBLE

A measure of how much the replayer has made its progress.

REPLAY_SESSION_COUNT

BIGINT

The number of current or replayed logical sessions

REPLAY_STATEMENT_COUNT

BIGINT

The number of current or replayed statements

REPLAY_FETCH_COUNT

BIGINT

The number of current or replayed fetch operations

REPLAY_COMMITTED_TRANSACTION_COUNT

BIGINT

The number of current or replayed committed transactions

ERROR_CODE

INTEGER

The error code generated by the system during capturing.

ERROR_MESSAGE

NVARCHAR (5000)

The error message generated by the system during capturing.
