M_HA_DR_PROVIDERS System View

Provides information about loaded HA/DR Providers.

Structure
Column name Data type Description
PROVIDER_NAME VARCHAR(64) HA/DR Provider name; class name of the provider
PROVIDER_COMPANY VARCHAR(64) HA/DR Provider company name, from provider's about() method
PROVIDER_DESCRIPTION VARCHAR(2000) HA/DR Provider description, from provider's about() method
PROVIDER_VERSION VARCHAR(32) HA/DR Provider version from provider's about() method
PROVIDER_TYPE VARCHAR(32) HA/DR Provider type
PROVIDER_PATH VARCHAR(512) HA/DR Provider path
EXECUTION_ORDER INTEGER HA/DR Provider execution order
