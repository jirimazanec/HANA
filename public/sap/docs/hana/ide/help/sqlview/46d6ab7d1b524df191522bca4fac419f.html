Smart Data Streaming


    M_STREAMING_APPLICATIONS System View [Smart Data Streaming]
    Provides a list of streaming applications running in the SAP HANA streaming cluster.
    M_STREAMING_PROJECTS System View [Smart Data Streaming]
    Provides information and statistics about each streaming project deployed in the SAP HANA streaming cluster.
    M_STREAMING_PROJECT_ADAPTERS System View [Smart Data Streaming]
    Contains information about internal adapters attached to each SAP HANA streaming project
    M_STREAMING_PROJECT_ADAPTER_STATISTICS System View [Smart Data Streaming]
    Contains custom statistics for both external and internal adapters.
    M_STREAMING_PROJECT_GD_SUBSCRIPTIONS System View [Smart Data Streaming]
    Contains information about active and inactive GD (Guaranteed Delivery) Subscriptions. The subscribers can be either external subscribers or internal adapters that support GD.
    M_STREAMING_PROJECT_PUBLISHERS System View [Smart Data Streaming]
    Contains information and statistics about streaming publishers for each SAP HANA streaming project, including external adapters.
    M_STREAMING_PROJECT_SUBSCRIBERS System View [Smart Data Streaming]
    Contains information and statistics about streaming subscribers for each SAP HANA streaming project, including external adapters.
    M_STREAMING_PROJECT_STREAMS System View [Smart Data Streaming]
    Provides information and statistics about each stream in an SAP HANA streaming project.
    M_STREAMING_SCHEMAS System View [Smart Data Streaming]
    Provides a list of the streaming schemas in the SAP HANA streaming cluster
    M_STREAMING_SERVICES System View [Smart Data Streaming]
    Provides a list of logical node names in the SAP HANA streaming cluster.
    Additional Views For Smart Data Streaming
    Views including information pertaining to, but not limited to, SAP HANA smart data streaming.
