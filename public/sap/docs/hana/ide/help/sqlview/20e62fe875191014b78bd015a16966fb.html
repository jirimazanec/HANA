QUARTER Function (Datetime)

Returns the quarter for the specified date.

Syntax
QUARTER (<d>, [, <start_month> ])
Description

Returns the numerical year and quarter of date <d>. The first quarter starts in the month specified by <start_month>. If <start_month> is not specified, then the first quarter is assumed to begin in January.

Example

The following example returns the value 2011-Q4 as the year and quarter for the specified date:

SELECT QUARTER (TO_DATE('2012-01-01', 'YYYY-MM-DD'), 2) "quarter" FROM DUMMY;
