M_ES_CONNECTIONS System View [Dynamic Tiering]

Processes running in SAP HANA dynamic tiering.

Caution This information applies only for the SAP HANA option cited in the title of this topic. You must have purchased the license for it in order to use the information contained in this topic.
Structure
Column name Data type Description
CONNECTION_ID BIGINT Connection ID. This connection ID can join the REMOTE_CONNECTION_ID of M_REMOTE_STATEMENTS.
INTERNAL_CONNECTION_ID BIGINT Internal Connection ID.
USER_ID VARCHAR(255) User ID for the connection.
REQUEST_TYPE VARCHAR(255) Type of the last request.
COMMAND_TYPE VARCHAR(32) Specific type of the command the command is executing.
LOCK_OWNER_CONNECTION_ID BIGINT The connection ID on which this connection is blocked; 0 if not blocked on any connection.
LOCK_OWNER_USER_ID VARCHAR(255) The owner of the blocking connection; NULL if there is no blocking connection.
CURSOR_COUNT INTEGER The number of active cursors in the connection; 0 if no cursors.
THREAD_COUNT INTEGER The number of threads with the connection. At least one thread is started as soon as the connection is opened, so the minimum value is 1.
IDLE_DURATION INTEGER The time in seconds since the last SA request was issued through the connection; in case of no last SA command, the time since ‘01-01-2000’ is displayed.
INTERNAL_IDLE_DURATION INTEGER The time in seconds since the last command was issued through the connection; in case of no command, the time since '01-01-2000' is displayed.
TEMP_TABLE_SIZE BIGINT The size of temporary table space in MBs; 0 if no temporary table space is used.
TEMP_WORKSPACE_SIZE BIGINT The size of temporary workspace in MBs; 0 if no temporary workspace is used.
LAST_REQUEST_TIME TIMESTAMP The time at which the last request for this connection started.
START_TIME TIMESTAMP The time the connection was created.
TRANSACTION_ID BIGINT The transaction ID of the current transaction on the connection. This is the same as the transaction ID in the esserver.trc file by the BeginTxn, CmtTxn, and PostCmtTxn messages, as well as the Txn ID Seq logged when the database is opened.
