HOST_ONE_DAY_FILE_COUNT System View (Embedded Statistics Service)

Information on trace files accumulated per host. The table contains information only for the last 42 days.

Structure

Column name

Data type

Unit

Description

SNAPSHOT_ID

TIMESTAMP

Datetime

Timestamp (UTC) identifying when the statistics service performed a data collection. Data collections scheduled for the same time have the same snapshot ID.

INDEX

VARCHAR(70)

  

Unique identifier for a row of the result set that a particular snapshot of a data collection represents.

SERVER_TIMESTAMP

TIMESTAMP

Datetime

Timestamp (local server time) identifying when a data collection actually occurred on the server.

FILE_TYPE

VARCHAR(6)

String

LOG, TRC, or DUMP

HOST

VARCHAR(64)

  

Host name

NUM_FILES

BIGINT

Number

Number of trace files

SUM_SIZE

Number
