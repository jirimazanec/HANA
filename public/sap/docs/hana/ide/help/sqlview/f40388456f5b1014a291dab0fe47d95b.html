M_MEMORY_RECLAIM_STATISTICS System View

Statistics for reclaiming memory (e.g. defragmentation, unloading of memory objects, ...).

Structure

This view has a resettable counterpart. This means, you can also see the values since the last reset in the dependent view M_MEMORY_RECLAIM_STATISTICS_RESET. To reset the values, use the SQL command: ALTER SYSTEM RESET MONITORING VIEW SYS.M_MEMORY_RECLAIM_STATISTICS_RESET or the management console command: monitor reset -n M_MEMORY_RECLAIM_STATISTICS_RESET
