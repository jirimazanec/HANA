M_DELTA_MERGE_STATISTICS System View

Provides information on table delta merge statistics.

Full description

This view shows table delta merge statistics. It lists the table delta merge events since the indexserver was last restarted. Table delta merges, optimize compression runs, and application merge hints are listed separately.

Boolean

The call success flag. This depends on the field TYPE. HINT is when the application merge hint was accepted or rejected. MERGE/SPARSE means that the delta merge/optimize compression was completed with or without success, indicated with either 'TRUE' or 'FALSE'. For example, a table delta merge call which did not result in a delta merge because the delta was empty, is indicated with 'FALSE'.
