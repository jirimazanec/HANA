M_SERVICE_THREADS System View

Displays detailed information about threads created by services.

Structure

Column name

Data type

Description

HOST

VARCHAR(64)

Specifies the host name.

PORT

INTEGER

Specifies the internal port.

SERVICE_NAME

VARCHAR(32)

Specifies the service name. See M_SERVICE_TYPES for all known service names.

HIERARCHY

VARCHAR(128)

Specifies the thread grouping information. This column contains the connection ID, the update transaction ID, and the transaction ID. The column is empty if the threads are inactive.

CONNECTION_ID

INTEGER

Specifies the connection ID.

THREAD_ID

BIGINT

Specifies the thread ID.

THREAD_TYPE

VARCHAR(128)

Specifies the thread type.

THREAD_METHOD

VARCHAR(256)

Specifies the thread method.

THREAD_DETAIL

NCLOB

Specifies the thread detail.

THREAD_STATE

VARCHAR(32)

Specifies the thread state.

IS_ACTIVE

VARCHAR(5)

Specifies whether the thread is active.

DURATION

BIGINT

Specifies the thread duration.

CALLER

VARCHAR(256)

Specifies the service that called the thread.

CALLING

VARCHAR(256)

Specifies the service called by the thread.

STATEMENT_ID

VARCHAR(20)

Specifies the ID of the statement being executed.

STATEMENT_HASH

VARCHAR(32)

Specifies the MD5 hash value for STATEMENT_STRING.

USER_NAME

NVARCHAR(256)

Specifies the SQL user name.

APPLICATION_NAME

NVARCHAR(256)

Specifies the name of the application.

APPLICATION_USER_NAME

NVARCHAR(256)

Specifies the application user name.

APPLICATION_SOURCE

NVARCHAR(256)

Specifies that the application can define which source file SAP HANA is called from. This value is also displayed in the M_PREPARED_STATEMENTS.APPLICATION_SOURCE system view.

CPU_TIME_SELF

BIGINT

Specifies the active CPU time of the thread.

CPU_TIME_CUMULATIVE

BIGINT

Specifies the active CPU time of the thread and its associated children.

CLIENT_IP

VARCHAR(45)

Specifies the IP of the client machine.

CLIENT_PID

BIGINT

Specifies the client process ID.

STATEMENT_EXECUTION_ID

BIGINT

Specifies the statement execution ID.

TRANSACTION_ID

INTEGER

Specifies the transaction ID.

UPDATE_TRANSACTION_ID

BIGINT

Specifies the update transaction ID.

LOCK_WAIT_COMPONENT

VARCHAR(32)

Specifies the component assigned to the lock.

LOCK_WAIT_NAME

VARCHAR(128)

Specifies the internal name of the lock (can be joined with the STATISTICS_NAME column from, for example, M_MUTEXES).

LOCK_OWNER_THREAD_ID

BIGINT

Specifies the ID of the thread that is holding the lock.

LOCKS_OWNED

VARCHAR(256)

Specifies the locks currently owned by the thread, including the lock type (shared/exclusive). This column does not show all locks that are monitored.

PRIORITY

INTEGER

Specifies the statement priority.

STATEMENT_THREAD_LIMIT

INTEGER

Specifies the statement thread limit.

STATEMENT_MEMORY_LIMIT

INTEGER

Specifies the statement memory limit.

PASSPORT_ROOTCONTEXT_ID

VARCHAR(16)

Specifies the passport root context ID.

PASSPORT_TRANSACTION_ID

VARCHAR(32)

Specifies the passport transaction ID.

PASSPORT_CONNECTION_ID

VARCHAR(16)

Specifies the passport connection ID.

PASSPORT_CONNECTION_COUNTER

BIGINT

Specifies the passport connection counter.

PASSPORT_COMPONENT_NAME

NVARCHAR(32)

Specifies the passport component name.

PASSPORT_ACTION

NVARCHAR(40)

Specifies the passport action.
