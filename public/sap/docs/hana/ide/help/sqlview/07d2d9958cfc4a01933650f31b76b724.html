M_STREAMING_PROJECT_GD_SUBSCRIPTIONS System View [Smart Data Streaming]

Contains information about active and inactive GD (Guaranteed Delivery) Subscriptions. The subscribers can be either external subscribers or internal adapters that support GD.

Caution This information applies only for the SAP HANA option cited in the title of this topic. You must have purchased the license for it in order to use the information contained in this topic.

Only users with a MONITOR role can view the content of this system view. The view is empty for all other users.

Structure
Column name Data type Unit Description
SCHEMA_NAME NVARCHAR(256)   The schema in which the project is deployed.
PROJECT_NAME NVARCHAR(256)   The name given to the project. It corresponds to the APPLICATION_NAME in the M_STREAMING_APPLICATIONS System View for APPLICATION_TYPE "PROJECT" or "HA_PROJECT".
PROJECT_INSTANCE INTEGER   The instance number of the project.. It corresponds to the APPLICATION_INSTANCE in the M_STREAMING_APPLICATIONS System View for APPLICATION_TYPE "PROJECT" or "HA_PROJECT".
GD_KEY NVARCHAR(1024)   The automatically generated key for the GD (Guaranteed Delivery) session.
STREAM_NAME NVARCHAR(256)   The name of the stream being subscribed.
USER_NAME NVARCHAR(256)   The name of the user doing the subscription.
GD_NAME NVARCHAR(256)   The name of this guaranteed delivery session .
GD_SEQUENCE_NUMBER BIGINT   The sequence number of the last event committed from stream_name in this guaranteed delivery session. A value of 0 indicates that no events have been committed.
STREAMING_CONNECTION_ID INTEGER   The unique handle for active external subscribers. For active internal adapters the value is always 0. Inactive subscribers have a value of -1.
LAST_UPDATE_TIME TIMESTAMP   The last time statistics were updated for the GD subscription
