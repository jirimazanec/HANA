M_CONNECTIONS System View

Provides detailed information on connections between a client and a database. Information includes: connection status, client information, connection type, and resource utilization.

Structure

Column name

Data type

Unit

Description

HOST

VARCHAR(64)

  

The host name.

PORT

INTEGER

  

The internal port.

CONNECTION_ID

INTEGER

  

The connection ID.

TRANSACTION_ID

INTEGER

  

The transaction object ID.

START_TIME

TIMESTAMP

  

The connected time.

IDLE_TIME

BIGINT

Millisecond

The time that the connection is unused and idle.

CONNECTION_STATUS

VARCHAR(128)

  

The connection status, which can be either RUNNING or IDLE.

CLIENT_HOST

NVARCHAR(256)

  

The host name of client machine.

CLIENT_IP

VARCHAR(45)

  

The IP of client machine.

CLIENT_PID

BIGINT

  

The client process ID.

USER_NAME

NVARCHAR(256)

  

The user name.

CONNECTION_TYPE

VARCHAR(128)

  

The connection type. Connection types include: Remote, Local, History (remote), History (local).

OWN

VARCHAR(5)

  

The user's own connection. Returns TRUE if it is your own connection and FALSE if it is not.

IS_HISTORY_SAVED

VARCHAR(5)

  

This column is deprecated. Do not use this value.

MEMORY_SIZE_PER_CONNECTION

BIGINT

Byte

Specifies the allocated memory size per connection.

AUTO_COMMIT

VARCHAR(5)

  

The commit mode of the current transaction. Returns TRUE if the current connection is in auto-commit mode and FALSE if it is not.

LAST_ACTION

VARCHAR(128)

  

The last action completed by the current connection. Actions include: ExecuteGroup, CommitTrans, AbortTrans, PrepareStatement, CloseStatement, ExecutePrepared, ExecuteStatement, FetchCursor, CloseCursor, LobGetPiece, LogPutPiece, LobFind, Authenticate, Connect, Disconnect, ExecQidItab, CursorFetchItab, InsertIncompleteItab, AbapStream, TxStartXA, and TxJoinXA.

CURRENT_STATEMENT_ID

VARCHAR(256)

  

The current statement ID.

CURRENT_OPERATOR_NAME

VARCHAR(64)

  

The current operator name.

FETCHED_RECORD_COUNT

BIGINT

  

The sum of the record count fetched by select statements.

AFFECTED_RECORD_COUNT

BIGINT

  

The sum of the record count affected by DML/DDL statements. For example, INSERT/UPDATE/DELETE/REPLACE/CREATE TABLE ... LIKE ... WITH DATA/CREATE TABLE ... AS (SELECT ..)/IMPORT/EXPORT.

SENT_MESSAGE_SIZE

BIGINT

  

The total size of messages sent by the current connection.

SENT_MESSAGE_COUNT

BIGINT

  

The total message count sent by the current connection.

RECEIVED_MESSAGE_SIZE

BIGINT

  

The total size of messages received by the current connection.

RECEIVED_MESSAGE_COUNT

BIGINT

  

The total message count received by the current connection.

CREATOR_THREAD_ID

BIGINT

  

The thread ID that created the current connection.

CREATED_BY

VARCHAR(256)

  

The engine component that created the connections. These include: Session, Planning, Repository, CalcEngine, Authentication, Table Exporter, Loader, LLVM, JSVM, IMS Search API, OLAP Engine, Mergedog, Ping Status, Name Server, Queue Server, SQL Stored Procedure, Authorization, TrexViaDbsl from ABAP, HybridTable Reorganizer, and Session external.

IS_ENCRYPTED

VARCHAR(5)

  

Encrypted: TRUE if the secure communication is enabled (SSL enabled), and FALSE if otherwise.

AUTHENTICATION_METHOD

VARCHAR(48)

  

The name of the authentication method used to authenticate the connection.

END_TIME

TIMESTAMP

  

The time when the connection is closed for history connections.

PARENT_CONNECTION_ID

INTEGER

  

The parent connection ID.

CLIENT_DISTRIBUTION_MODE

VARCHAR(128)

  

The client distribution mode of the current connection (for example, [distribution] client_distribution_mode).

LOGICAL_CONNECTION_ID

INTEGER

  

The logical connection ID in statement routing.

SOURCE_SITE_LOGICAL_CONNECTION_ID

INTEGER

  

The logical connection ID of the origin site.

CURRENT_SCHEMA_NAME

NVARCHAR(256)

  

The current schema name.

CURRENT_THREAD_ID

BIGINT

  

The current executing thread ID.

PRIORITY

TINYINT

  

The user-specified priority.

WORKLOAD_CLASS_NAME

NVARCHAR(256)

  

The name of the effective workload class.

CURRENT_COLLATION_NAME

VARCHAR(16)

  

The name of the current collation.

CLOSE_REASON

VARCHAR(32)

  

The reason for the connection close.
