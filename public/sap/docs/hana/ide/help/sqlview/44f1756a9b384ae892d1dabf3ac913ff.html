STATISTICS_EMAIL_PROCESSING System View (Embedded Statistics Service)

Statistics of emails processed per host. This table contains information only for the last 7 days.

Structure

Column name

Data type

Description

SNAPSHOT_ID TIMESTAMP Time (UTC) at which the alert occurred
ALERT_ID INTEGER Unique check ID
STATE VARCHAR(20) Status of e-mail notification. The following values are possible: New (default), Handled.
SEVERITY TINYINT Highest severity of alert issued by the check at the SNAPSHOT_ID
