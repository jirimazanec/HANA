M_MVCC_SNAPSHOTS System View

Provides detailed snapshot information of the Multiversion Concurrency Control (MVCC) manager.

Structure

Column name

Data type

Unit

Description

HOST

VARCHAR(64)

Specifies the host name.

PORT

INTEGER

Specifies the internal port.

TRANSACTION_ID

BIGINT

Specifies the associated transaction object ID.

MVCC_SNAPSHOT_TIMESTAMP

BIGINT

Specifies the MVCC timestamp of the snapshot.

STATEMENT_ID

BIGINT

Specifies the associated logical statement ID.

RELATED_TABLES

VARCHAR(1024)

Specifies the associated table object IDs.
