TEXT_CONFIGURATIONS System View

Provides information about configurations and other resources for customizing text analysis and text mining.

Structure

Column name

Data type

Description

SCHEMA_NAME

NVARCHAR(256)

The name of the schema containing the resource

NAME

NVARCHAR(256)

The name of the resource

TYPE

VARCHAR (16)

The resource type, such as hdbtextconfig (configuration), textdict (dictionary), textrule (ruleset), textminingconfig

OWNER_NAME

NVARCHAR (256)

The name of the resource owner

LAST_UPDATE_TIME

TIMESTAMP

The timestamp when the resource was created or last updated

DATA

BLOB

The resource content
