Accelerator for SAP ASE


    ALTER DATABASE Statement [Accelerator for SAP ASE]
    Alters an existing database by adding or removing the etsserver service.
    ALTER SYSTEM ALTER CONFIGURATION Statement [Accelerator for SAP ASE]
    Enables the accelerator for SAP ASE service for provisioning to a tenant database.
    ALTER SYSTEM INITIALIZE SERVICE Statement [Accelerator for SAP ASE]
    Creates the remote sources and virtual tables, and imports the delivery unit (DU) for the etsservice service.
    ALTER SYSTEM UNINITIALIZE SERVICE Statement [Accelerator for SAP ASE]
    Removes the remote sources, virtual tables, and the delivery unit (DU) for the etsserver service.
