M_HOST_NETWORK_STATISTICS System View

Provides information about the network statistics of a host. Most columns correspond to the similarly named values reported by netstat -s.

Structure

Column name

Data type

Description

HOST

VARCHAR(64)

Host name the network statistics refer to

TCP_SEGMENTS_RECEIVED

BIGINT

Total number of TCP segments received. Corresponds on Linux to Tcp: Insegs from /proc/net/snmp

TCP_SEGMENTS_SENT_OUT

BIGINT

Total number of TCP segments sent out. Corresponds on Linux to Tcp: OutSegs from /proc/net/snmp

TCP_SEGMENTS_RETRANSMITTED

BIGINT

Number of TCP segments that had to be retransmitted. Corresponds on Linux to Tcp: RetransSegs from /proc/net/snmp

TCP_BAD_SEGMENTS_RECEIVED

BIGINT

Number of TCP segments that were broken upon receive. Corresponds on Linux to Tcp: InErrs from /proc/net/snmp
