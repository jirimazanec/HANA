LDAP_PROVIDERS System View

Lists all LDAP providers.

Structure

Column name

Data type

Description

LDAP_PROVIDER_NAME

NVARCHAR(256)

Name of the LDAP provider

LDAP_PROVIDER_ID

BIGINT

ID of the LDAP provider

CREATE_TIME

TIMESTAMP

Creation time

IS_DEFAULT

VARCHAR(5)

Specifies if the LDAP provider is the default provider: ‘TRUE’/’FALSE’

IS_SSL_USED

VARCHAR(5)

Specifies if the connection to the configured LDAP Server uses SSL: ‘TRUE’/’FALSE’

IS_PROVIDER_ENABLED

VARCHAR(5)

Specifies if the LDAP Provider is enabled for use: ‘TRUE’/’FALSE’

ATTRIBUTE_DN

VARCHAR(256)

The LDAP attribute to provide the LDAP distinguished name of a user

ATTRIBUTE_MEMBER_OF

NVARCHAR(256)

The LDAP attribute that provides the list of LDAP groups that the user is a member of
