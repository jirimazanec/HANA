M_CACHES_RESET System View

Shows aggregated information on caches (since last reset)..

This view contains values accumulated since the last reset of the main view M_CACHES. Refer to M_CACHES for information about the structure and use of this view. In addition to the members mentioned in M_CACHES, this view also contains a timestamp field RESET_TIME, which indicates the last time the data was reset
