JSON Document Store Statements

The following statements enable you to use the SAP HANA database as a JSON document store.

Supported Statements

The following two lists provide links to various topics in this guide that support the JSON Document Store feature. To access the SQL features of JSON Document Store, you must first enable the feature for the database (for example, ALTER DATABASE <database_name> ADD 'docstore').

DDL statements:

  • CREATE COLLECTION Statement (JSON Document Store)

  • DROP COLLECTION Statement (JSON Document Store)

  • RENAME COLLECTION Statement (JSON Document Store)

  • TRUNCATE COLLECTION Statement (JSON Document Store)

DML statements:

  • INSERT Statement (JSON Document Store)

  • SELECT Statement (JSON Document Store)

  • UPDATE Statement (JSON Document Store)

  • DELETE Statement (JSON Document Store)
