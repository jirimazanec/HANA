LAST_VALUE Function (Aggregate)

Returns the value of the last element of a column.

Syntax
LAST_VALUE (<column1> ORDER BY <column2>)
Description

Returns the value of the last element in <column1> as ordered by <column2>.

NULL is returned if the value is NULL or if <column1> is empty.

The output of LAST_VALUE function can be non-deterministic for values that are considered equivalent when compared.

Example

The example below returns the last value in COL1 column when the table is ordered by COL2. The query returns 4.

CREATE TABLE T (COL1 DOUBLE, COL2 DOUBLE);

INSERT INTO T VALUES(1, 1);
INSERT INTO T VALUES(4, 5);
INSERT INTO T VALUES(7, 3);

SELECT LAST_VALUE (COL1 ORDER BY COL2) FROM T;
