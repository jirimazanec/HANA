M_FEATURE_USAGE System View

Provides detailed feature usage statistics.

Structure

Column name

Data type

Description

COMPONENT_NAME

VARCHAR(32)

Component name (see M_FEATURES)

FEATURE_NAME

VARCHAR(64)

Feature name (see M_FEATURES)

IS_DEPRECATED

VARCHAR(5)

Set to TRUE if a feature is deprecated, otherwise it is FALSE.

OBJECT_COUNT

BIGINT(19)

Counter tracking the number of artifacts that belong to the feature (for example, number of existing history tables).

CALL_COUNT

BIGINT(19)

Counter tracking the number of times, the feature was used since the last start of the indexserver (for example, number of accesses to any history table).

LAST_TIMESTAMP

TIMESTAMP(27)

Last point in time when the feature was used.

LAST_CONNECTION_ID

INTEGER(10)

Connection ID of the last feature usage.

LAST_USER_NAME

VARCHAR(256)

DB user of the last feature usage (for example, SAP<SID>).

LAST_APPLICATION_USER_NAME

VARCHAR(256)

Application user of the last feature usage (for example, <D-User>).

LAST_APPLICATION_NAME

VARCHAR(256)

Application name of the last feature usage (e.g. "ABAP:KIT").

LAST_APPLICATION_SOURCE

VARCHAR(256)

Application source name/location of the last feature usage (for example, "CL_RSCRT_RDA_RUN====_").

LAST_STATEMENT_ID

VARCHAR(20)

Statement ID of the last feature usage.

LAST_STATEMENT_HASH

VARCHAR(32)

Statement hash of the last feature usage.
