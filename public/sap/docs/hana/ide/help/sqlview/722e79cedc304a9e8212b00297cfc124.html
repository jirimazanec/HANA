M_TABLE_LOB_STATISTICS System View

Provides information about the aggregated file and packed LOB statistics per host, port, table, partition, and column.

Structure

Column name

Data type

Unit

Description

HOST

VARCHAR(64)

  

The name of the host.

PORT

INTEGER

  

The internal port.

SCHEMA_NAME

NVARCHAR(256)

  

The schema name associated with the LOBs.

TABLE_NAME

NVARCHAR(256)

  

The table name associated with the LOBs.

PART_ID

INTEGER

  

The partition ID: 0 for non-partitioned tables and the number of partitions for partitioned tables.

COLUMN_NAME

NVARCHAR(256)

  

The column name associated with the LOBs.

LOB_STORAGE_TYPE

VARCHAR(7)

  

The type is one of the following:

INPLACE

Small LOBs (completely loaded into main memory).

PACKED

Medium-sized LOBs where the size is between two thresholds (only loaded into main memory when required).

FILE

Large LOBs (individually loaded into main memory on demand).

CONTAINER_ID

BIGINT

  

The container ID of the packed LOB container; NULL for file LOBs.

DISK_SIZE

BIGINT

Byte

The number of bytes used for the LOBs on a disk.

BINARY_SIZE

BIGINT

Byte

The binary size of the LOBs. This value might be less than the DISK_SIZE due to management information and paging.

CHARACTER_SIZE

BIGINT

  

The number of characters in the NCLOBs (only) column. For packed LOBs, this corresponds to the total number of characters of all of the LOBs in the container.

LOB_COUNT

BIGINT

  

The total number of associated LOBs.

READ_COUNT

BIGINT

  

The counter for read operations on the LOBs.
