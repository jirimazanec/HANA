GLOBAL_TABLE_PERSISTENCE_STATISTICS System View (Embedded Statistics Service)

Persistence virtual file summary statistics for tables.

Full description

Only tables where DISK_SIZE > 10,000,000 bytes are listed.

The table contains information only for the last 42 days.
