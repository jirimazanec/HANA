HOST_TASK_OPERATIONS System View (Embedded Statistics Service)

Task operation statistics per host. This table contains information only for the last 7 days.

Structure
Column name Data type Description
CONNECTION_ID INTEGER Connection identifier
TRANSACTION_ID INTEGER Transaction identifier used for the operation execution
TASK_EXECUTION_ID BIGINT Task identifier
OPERATION_NAME NVARCHAR(128) Internal name of operation
CURRENT_OPERATION NVARCHAR(128) Name of operation
OPERATION_TYPE NVARCHAR(128) Type of operation
START_TIME TIMESTAMP Start time of operation
END_TIME TIMESTAMP End time of operation
DURATION BIGINT Execution time of operation (ms)
STATUS VARCHAR(16) Status of the operation: STARTING/RUNNING/FAILED/COMPLETED/CANCELLING/CANCELLED
PROCESSED_RECORDS BIGINT Total number of records processed
OPERATION_PROGRESS_PERCENT BIGINT Operation progress (percent)
HAS_SIDE_EFFECTS VARCHAR(5) 'TRUE' if the operation produces side effect data, else 'FALSE'
