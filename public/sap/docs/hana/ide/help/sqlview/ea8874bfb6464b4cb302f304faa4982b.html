M_WORKLOAD_CAPTURES System View

Provides information about workload captures.

Structure

Column name

Data type

Description

CAPTURE_ID

BIGINT

The unique ID of the captured workload

CAPTURE_NAME

NVARCHAR (256)

The user-specified name of the captured workload

CAPTURE_DESCRIPTION

NVARCHAR (5000)

The user-specified description of the captured workload

CAPTURE_VERSION

INTEGER

The version number of the structure format for the captured workloads or capturing workload

START_TIME

TIMESTAMP

The start timestamp of the capture

UTC_START_TIME

TIMESTAMP

The UTC start timestamp of the capture

END_TIME

TIMESTAMP

The end timestamp of the captured workload

UTC_END_TIME

TIMESTAMP

The UTC end timestamp of the captured workload

IS_FILTER_APPLIED

VARCHAR (5)

The flag that is set to FALSE if the user specifies any non-default value to any filter parameters

SYSTEM_ID

VARCHAR (3)

The system ID in which the capture has been done

DATABASE_NAME

NVARCHAR (256)

The database name in which the capture has been done

STATUS

VARCHAR (16)

The status of the workload captures

IS_LOADED

VARCHAR (5)

The flag that is set to TRUE if the captured data is loaded into the database

PROGRESS

DOUBLE

The current progress of the load

LEVEL

VARCHAR (16)

The level of workload captures

PARAMETERS

NVARCHAR (5000)

The applied parameter's key/value pairs

STATEMENT_DURATION_THRESHOLD

BIGINT

The threshold value for the workloads to be captured

FLUSH_INTERVAL

INTEGER

The interval of flushing out the file output stream

CAPTURE_FILE_NAME

VARCHAR(256)

The file name of the captured workload

CAPTURE_SESSION_COUNT

BIGINT

The number of current or captured logical sessions

CAPTURE_STATEMENT_COUNT

BIGINT

The number of current or captured statements

CAPTURE_FETCH_COUNT

BIGINT

The number current or captured fetch operations

CAPTURE_COMMITTED_TRANSACTION_COUNT

BIGINT

The number of current or captured committed transactions

CAPTURE_SIZE

BIGINT

The size of the captured workload file

ERROR_CODE

INTEGER

The error code generated by the system during capturing

ERROR_MESSAGE

NVARCHAR (5000)

The error message generated by system during capturing
