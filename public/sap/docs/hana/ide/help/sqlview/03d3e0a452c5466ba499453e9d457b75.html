STREAMING_PROJECTS_STATISTICS System View (Embedded Statistics Service)

Provides detailed data for smart data streaming projects. The table contains information only for the last 3 days.

Structure

Column name

Data type

Description

SNAPSHOT_ID TIMESTAMP Timestamp ID (UTC)
SERVER_TIMESTAMP TIMESTAMP Processing Timestamp (local time)
INDEX NVARCHAR(524) Index
SCHEMA_NAME NVARCHAR(256) Schema in which the project is deployed
PROJECT_NAME NVARCHAR(256) Name given to the project. Corresponds to the APPLICATION_NAME in M_STREAMING_APPLICATIONS
PROJECT_INSTANCE INTEGER Instance number of the project. It corresponds to the APPLICATION_INSTANCE in M_STREAMING_APPLICATIONS
CPU_SYSTEM_UTILIZATION REAL Total system CPU usage, as a percentage, by the project since the last update. Values are in the range 0 to 100
CPU_USER_UTILIZATION REAL Total user CPU usage, as a percentage, by the project since the last update. Values are in the range 0 to 100
CPU_UTILIZATION REAL Total CPU usage, as a percentage, by the project since the last update. Values are in the range 0 to 100
RESIDENT_MEMORY_SIZE BIGINT Total amount of resident memory used by the project in bytes
THREAD_COUNT INTEGER Total number of threads used by the project at the time of the update
VIRTUAL_MEMORY_SIZE BIGINT Total amount of virtual memory used by the project in bytes
