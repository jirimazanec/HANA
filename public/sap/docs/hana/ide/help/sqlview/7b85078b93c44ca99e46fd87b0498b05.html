STREAMING_PROJECT_STREAMS_STATISTICS System View (Embedded Statistics Service)

Provides detailed data for the streams of smart data streaming projects. The table contains information only for the last 3 days.

Structure

Column name

Data type

Description

SNAPSHOT_ID TIMESTAMP Timestamp ID (UTC)
SERVER_TIMESTAMP TIMESTAMP Processing Timestamp (local time)
INDEX NVARCHAR(781) Index
SCHEMA_NAME NVARCHAR(256) Schema in which the project is deployed
PROJECT_NAME NVARCHAR(256) Name given to the project. Corresponds to the APPLICATION_NAME in M_STREAMING_APPLICATIONS
PROJECT_INSTANCE INTEGER Instance number of the project. It corresponds to the APPLICATION_INSTANCE in M_STREAMING_APPLICATIONS
STREAM_NAME NVARCHAR(256) Name of the stream
CPU_SYSTEM_UTILIZATION REAL System CPU usage for the stream, as a percentage of all CPU cores on the machine
CPU_USER_UTILIZATION REAL User CPU usage for the stream, as a percentage of all CPU cores on the machine
CPU_UTILIZATION REAL Total CPU usage for the stream, as a percentage of all CPU cores on the machine
ROW_THROUGHPUT REAL Stream's performance, in transactions per second, since the last update
TRANSACTION_THROUGHPUT REAL Stream's performance, in transactions per second, since the last update
