Full description

Use '/' as root path. For deeper paths use PATH+'/'+NAME. If NAME contains '/' you have to enclose it in CHR(1), For example:

SELECT * FROM M_TOPOLOGY_TREE WHERE PATH='/index/'||CHR(1)||'SYSTEM:A/B'||CHR(1)||'#15046'

To change topology tree values use ALTER SYSTEM ALTER CONFIGURATION ('topology.ini','system') SET (PATH,NAME)=VALUE To delete values use ALTER SYSTEM ALTER CONFIGURATION ('topology.ini','system') UNSET (PATH,NAME)
