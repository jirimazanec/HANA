M_CS_LOG_REPLAY_QUEUE_STATISTICS System View

Provides information about column store log replay queue statistics.

Structure

Column name

Data type

Unit

Description

HOST

VARCHAR(64)

  

The name of the host.

PORT

INTEGER

  

The internal port.

VOLUME_ID

INTEGER

  

The persistence volume ID.

LOG_REPLAY_QUEUE_ID

INTEGER

  

The ID of the recovery queue.

TOTAL_CS_LOG_RECORD_COUNT

BIGINT

  

The total number of redo records that were processed in this queue.

TOTAL_DML_LOG_RECORD_COUNT

BIGINT

TOTAL_DDL_LOG_RECORD_COUNT

BIGINT

  

The total number of column store DDL redo records that were processed in this queue.

LAST_CS_LOG_RECORD_TYPE

VARCHAR(40)

  

The last column store log record type in this queue.

LAST_LOG_REPLAY_POSITION

BIGINT

  

The last log replay position that was processed in this queue.

LAST_SAVEPOINT_POSITION

BIGINT

  

The last savepoint position that was passed in this queue.

TOTAL_SAVEPOINT_COUNT

BIGINT

  

The total number of savepoints that were passsed in this queue.

ACTIVE_TABLE_COUNT

BIGINT

  

The number of tables that are currently handled in this queue.

BIGINT

  

The number of delta merges that are currently ongoing for tables that are handled in this queue.

TOTAL_DELTA_MERGE_COUNT

BIGINT

  

The total number of delta merges that were started on this queue.

TOTAL_SUCCESSFUL_DELTA_MERGE_COUNT

BIGINT

The total number of successful merges.

TOTAL_DELTA_FUSION_COUNT

BIGINT

The total number of delta fusions that were made in this queue.

ACTIVE_OPTIMIZE_COMPRESSION_COUNT

BIGINT

The number of optimize compressions that are currently ongoing for tables that are handled in this queue.

TOTAL_OPTIMIZE_COMPRESSION_COUNT

BIGINT

The total number of successful optimize compressions.

TOTAL_SUCCESSFUL_OPTIMIZE_COMPRESSION_COUNT

BIGINT

The total number of successful optimize compressions.

TOTAL_DML_CONTEXT_CREATE_COUNT

BIGINT

The number of expensive creations of DML contexts.
