M_LOAD_HISTORY_SERVICE System View

Lists service-specific load history KPIs.

Structure

Column name

Data type

Unit

Description

HOST

VARCHAR(64)

The host name.

PORT

INTEGER

The internal port.

TIME

TIMESTAMP

The KPI collection timestamp.

CPU

BIGINT

The CPU used by the service.

SYSTEM_CPU

BIGINT

The OS Kernel/System CPU used by the service.

MEMORY_USED

BIGINT

The memory used by the service.

MEMORY_ALLOCATION_LIMIT

BIGINT

The memory allocation limit for the service.

HANDLE_COUNT

BIGINT

The number of open handles.

PING_TIME

BIGINT

The duration of the service ping request (THREAD_METHOD=__nsWatchdog). This request includes the time it takes to measure the values shown in this view.

SWAP_IN

BIGINT

The bytes read from the swap by the service (column 12(majflt) in /proc/<pid>/stat * sysconf(_SC_PAGESIZE)).

CONNECTION_COUNT

BIGINT

The number of open SQL connections.

TRANSACTION_COUNT

BIGINT

The number of open SQL transactions.

BLOCKED_TRANSACTION_COUNT

BIGINT

The number of blocked SQL transactions.

STATEMENT_COUNT

BIGINT

The number of finished SQL statements.

COMMIT_ID_RANGE

BIGINT

The range between the newest and oldest active commit IDs.

PENDING_SESSION_COUNT

BIGINT

The number of pending requests.

MVCC_VERSION_COUNT

BIGINT

The number of active MVCC versions.

RECORD_LOCK_COUNT

BIGINT

The number of acquired record locks.

BIGINT

The number of read requests (SELECT).

CS_WRITE_COUNT

BIGINT

The number of write requests (INSERT, UPDATE, DELETE).

CS_MERGE_COUNT

BIGINT

The number of merge requests.

CS_UNLOAD_COUNT

BIGINT

The number of column unloads.

ACTIVE_THREAD_COUNT

BIGINT

The number of active threads.

WAITING_THREAD_COUNT

BIGINT

The number of waiting threads.

TOTAL_THREAD_COUNT

BIGINT

The total number of threads.

ACTIVE_SQL_EXECUTOR_COUNT

BIGINT

The number of active SQLExecutors.

WAITING_SQL_EXECUTOR_COUNT

BIGINT

The number of waiting SQLExecutors.

TOTAL_SQL_EXECUTOR_COUNT

BIGINT

The total number of SQLExecutors.

DATA_WRITE_SIZE

BIGINT

The bytes written to the data area.

DATA_WRITE_TIME

BIGINT

The time used for writing to the data area.

LOG_WRITE_SIZE

BIGINT

The bytes written to the log area.

BIGINT

The time used for writing to the log area.

BIGINT

BIGINT

BIGINT

BIGINT

BIGINT
