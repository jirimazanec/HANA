X509_USER_MAPPINGS System View

Shows the X.509 certificates that are known for each user.

Structure

Column name

Data type

Description

USER_NAME

NVARCHAR(256)

The user name

SUBJECT_NAME

NVARCHAR(512)

The subject of the X.509 certificate of the user

ISSUER_NAME

NVARCHAR(512)

The issuer of the X.509 certificate of the user
