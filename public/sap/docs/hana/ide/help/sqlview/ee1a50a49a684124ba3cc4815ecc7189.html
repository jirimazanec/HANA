M_ENCRYPTION_OVERVIEW System View

Reports the encryption status for all data at rest where encryption is supported.

Structure

Column name

Data type

Description

SCOPE

VARCHAR(16)

Specifies whether the scope includes PERSISTENCE (data volumes), LOG (redo log), or BACKUP

IS_ENCRYPTION_ACTIVE

VARCHAR(5)

Returns TRUE if the encryption for the scope is currently active; otherwise FALSE

LAST_CHANGE_TIME

TIMESTAMP

Returns the latest timestamp, in the server's local time, when the status was changed
