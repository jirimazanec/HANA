ALTER SYSTEM STOP PERFTRACE Statement (System Management)

Stops an active performance trace.

Note Performance tracing is only to be used in conjunction with SAP Support personnel. The collected perftrace data cannot be analyzed by an end user.
Syntax
ALTER SYSTEM STOP PERFTRACE
Description

After stopping the trace, collect and save the performance trace data with the ALTER SYSTEM SAVE PERFTRACE statement.

Example

Stop an active performance trace. 

ALTER SYSTEM STOP PERFTRACE
