HOST_SERVICE_THREAD_SAMPLES_STATEMENTS System View (Embedded Statistics Service)

Provides data collection information for snapshots, including information about threads.

Structure

Column name

Description

SNAPSHOT_ID

Displays the timestamp (UTC) identifying when the statistics service performed a data collection. Data collections scheduled for the same time have the same snapshot ID.

SERVER_TIMESTAMP

Displays the timestamp (local server time) identifying when a data collection actually occurred on the server.

INDEX

Specifies the unique identifier for a row of the result set that a particular snapshot of a data collection represents.

APPLICATION_NAME

Specifies the name of the application.

APPLICATION_SOURCE

An application can define which source file SAP HANA is called from. Usage is up to the application. This value is also displayed in M_PREPARED_STATEMENTS.APPLICATION_SOURCE.

APPLICATION_USER_NAME

Specifies the application user name.

CPU_TIME

Specifies the active CPU time of the thread and associated children.

DURATION

Specifies the duration of the operation.

FIRST_TIMESTAMP

Specifies the first timestamp of the record.

HOST

Specifies the host name.

LAST_TIMESTAMP

Specifies the last timestamp of the record.

PORT

Specifies the internal port.

STATEMENT_EXECUTION_ID

Specifies the statement execution ID.

STATEMENT_HASH

Specifies the unique identifier for an SQL string.

STATEMENT_ID

Specifies the statement ID.

THREAD_AVG

Specifies the average number of threads during execution.

THREAD_MAX

Specifies the maximum number of threads during execution.

THREAD_MIN

Specifies the minimum number of threads during execution.

THREAD_SUM

Specifies the sum of distinct threads during execution.

USER_NAME

Specifies the SQL user name.
