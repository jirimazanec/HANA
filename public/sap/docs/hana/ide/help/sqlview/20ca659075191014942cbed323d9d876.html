Full description

This view has a resettable counterpart. This means you can also see the values since the last reset in the dependent view M_VOLUME_IO_RETRY_STATISTICS_RESET. To reset the values, use the SQL command: ALTER SYSTEM RESET MONITORING VIEW SYS.M_VOLUME_IO_RETRY_STATISTICS_RESET or the management console command: monitor reset -n M_VOLUME_IO_RETRY_STATISTICS_RESET.

This view shows retry and error I/O statistics for various buffer sizes. Each buffer size is a maximum value, so a buffer size of 4k actually means <= 4k and a buffer size of 16k means 4k < buffer size <= 16k. For overall I/O performance, see TOTAL_READ/WRITE_SIZE and TOTAL_IO_TIME in the M_VOLUME_IO_TOTAL_STATISTICS system view.
