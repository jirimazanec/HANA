M_JOIN_TRANSLATION_TABLES System View

Provides column store join engine translation tables statistics.

Structure
Column name Data type Description
HOST VARCHAR(64) Host name
PORT INTEGER Internal port
SCHEMA_NAME1 NVARCHAR(256) First schema name
TABLE_NAME1 NVARCHAR(256) First table name
COLUMN_NAME1 NVARCHAR(256) First table's column name
PART_ID1 INTEGER First table partition ID. 0 for non-partitioned tables, 1 through number of partitions for partitioned tables
VERSION1 BIGINT Version information of first table
SCHEMA_NAME2 NVARCHAR(256) Second schema name
TABLE_NAME2 NVARCHAR(256) Second table name
COLUMN_NAME2 NVARCHAR(256) Second table's column name
PART_ID2 INTEGER Second table partition ID. 0 for non-partitioned tables, 1 through number of partitions for partitioned tables
VERSION2 BIGINT Version information of second table
IMPLEMENTATION_TYPE NVARCHAR(8) Implementation type, for example, Vector, HASHMAP
IMPLEMENTATION_TYPE_CHANGE_TIME TIMESTAMP Last change of implementation type
IMPLEMENTATION_TYPE_CHANGE_COUNT INTEGER Number of implementation type changes
VALUE_ID_COUNT1 BIGINT Number of translated value IDs of first table
VALUE_ID_COUNT2 BIGINT Number of translated value IDs of second table
TRANSLATION_TABLE_MEMORY_SIZE BIGINT Memory used by translation table
ESTIMATED_HASHMAP_MEMORY_SIZE BIGINT Estimated Memory used by HASHMAP implementation
ESTIMATED_VECTOR_MEMORY_SIZE BIGINT Estimated Memory used by vector implementation
LAST_USED_COUNT BIGINT Counter to support LRU cache replacement algorithm
