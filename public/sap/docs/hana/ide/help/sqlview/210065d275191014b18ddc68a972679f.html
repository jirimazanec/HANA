TABLE_COLUMNS_ODBC System View

Provides information about available table columns.

Structure

Column name

Data type

Description

SCHEMA_NAME

NVARCHAR(256)

The schema name

TABLE_NAME

NVARCHAR(256)

The table name

TABLE_OID

BIGINT

The object ID of the table

COLUMN_NAME

NVARCHAR(256)

The name of the column

DATA_TYPE_ID

SMALLINT

The SQL data type ID of the column

DATA_TYPE_NAME

VARCHAR(16)

The SQL data type name of the column

COLUMN_SIZE

INTEGER

The total number of characters required to display the value when it is converted to characters

BUFFER_LENGTH

INTEGER

The length in bytes required to transfer the value

DECIMAL_DIGITS

INTEGER

The total number of significant digits to the right of the decimal point

NUM_PREC_RADIX

INTEGER

For numeric data types either 10 or 2. Specifies how to interpret columns COLUMN_SIZE and DECIMAL_DIGITS

NULLABLE

VARCHAR(5)

Specifies whether the column is allowed to accept null value: 'TRUE', 'FALSE'

COLUMN_DEF

NVARCHAR(5000)

The default value of the column

SQL_DATA_TYPE

SMALLINT

The SQL data type ID of the column

SQL_DATETIME_SUB

SMALLINT

The subtype code for datetime data types

CHAR_OCTET_LENGTH

INTEGER

The maximum length in bytes of a character or binary data type column

POSITION

INTEGER

The ordinal position of the column in a record

COMMENTS

NVARCHAR(5000)

The description for this column
