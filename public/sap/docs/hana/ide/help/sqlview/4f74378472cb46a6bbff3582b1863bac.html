M_DATA_STATISTICS System View

Provides an overview of the current state of data statistics objects, including their current content and properties, as seen and used by the consumers of data statistics (for example, the SQL Optimizer).

Structure

Column name

Data type

Description

DATA_STATISTICS_OID

BIGINT

Data statistics object ID. For example, 3209.

DATA_STATISTICS_TYPE

VARCHAR(12)

Data statistics object type. For example, HISTOGRAM.

DATA_STATISTICS_SCHEMA_NAME

NVARCHAR(256)

Data statistics object schema

DATA_STATISTICS_NAME

NVARCHAR(256)

Data statistics object name

DATA_SOURCE_SCHEMA_NAME

NVARCHAR(256)

Schema name of the data source object

DATA_SOURCE_OBJECT_NAME

NVARCHAR(256)

Name of the data source object

DATA_SOURCE_COLUMN_NAMES

NVARCHAR(5000)

List of column names of the data source

DATA_SOURCE_STORAGE_TYPE

VARCHAR(8)

Source storage type of the data source

DATA_SOURCE_PART_ID

INTEGER

Partition ID of the data source

LAST_REFRESH_TIME

TIMESTAMP

Time of the last refresh of the data statistics object

LAST_REFRESH_REASON

VARCHAR(19)

Reason of the last refresh of the data statistics object

DATA_STATISTICS_CONTENT

NCLOB (max size 2Gb)

Content and properties of the data statistics object as of the last refresh (JSON format). The content section is empty for SIMPLE, SKETCH, and RECORD COUNT; it contains bucket boundaries and counts for HISTOGRAM and lists of values and frequencies for TOPK. The content is similar to exported data for data statistics objects.

The properties section contains properties relevant to the data statistics of the specified type, and are set during the last refresh. The properties section, when present, may include some of the following properties:

MEMORY

Specifies the memory size used by the data statistics object

MEMORY PERCENT

Specifies the memory size used by the data statistics object, as a percentage of the data source size

DISTINCT COUNT

Specifies the number of unique values in the data source

NULL COUNT

Specifies the number of NULL values in the data source

BUCKETS

Specifies the number of buckets in the data statistics object

QERROR

Specifies the value of the qerror parameter used for building the data statistics object

QTHETA

Specifies the value of the qtheta parameter used for building the data statistics object

ACCURACY

Specifies the accuracy parameter used for building the data statistics object

PREFIX BITS

Specifies the prefix bits parameter used for building the data statistics object

MIN VALUE

Specifies the minimum value in the data source

MAX VALUE

Specifies the maximum value in the data source

MAX_ROWID

Specifies the maximum rowID for the data statistics object

MIN MAX IS VALID
Specifies whether MIN/MAX values are valid
SAMPLE SIZE

Specifies the size of the SAMPLE data statistics object

SAMPLE SIZE PERCENT

Specifies the maximum size of the SAMPLE data statistics object as a percentage of the data source size
