M_DATABASES System View

Provides information about all databases in the system. The full content of this view is only accessible from the system database.

Structure

Column name

Data type

Description

DATABASE_NAME

NVARCHAR (256)

Database name

DESCRIPTION

NVARCHAR (2000)

Database description

ACTIVE_STATUS

VARCHAR (16)

Database status

ACTIVE_STATUS_DETAILS

VARCHAR (128)

Database status details. Possible values include: Stopped by user, Stopped due to broken recovery, Stopped by takeover

OS_USER

NVARCHAR(256)

Database isolation operation system user

OS_GROUP

NVARCHAR(256)

Database isolation operation system group
