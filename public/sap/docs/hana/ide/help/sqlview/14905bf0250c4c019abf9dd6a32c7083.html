M_CS_COLUMNS_PERSISTENCE System View

Provides column persistence information for column tables.

Structure

Column name

Data type

Description

HOST

VARCHAR(64)

Specifies the host name.

PORT

INTEGER

Specifies the internal port.

SCHEMA_NAME

NVARCHAR(256)

Specifies the schema name.

TABLE_NAME

NVARCHAR(256)

Specifies the table name.

PART_ID

INTEGER

Specifies the table partition ID. Returns the following:

  • For partitioned tables, the part ID is equal to the sequential number of the partition, starting at 1.
  • In the case of replicated tables, the part ID is 1 for the original table and subsequent part IDs are assigned to replica tables.
  • The part ID is 0 for tables that are not partitioned.
  • A part ID value of -1 indicates that a modfication of the table schema is currently in progress.

COLUMN_ID

INTEGER

Specifies the numeric column ID.

COLUMN_NAME

NVARCHAR(256)

Specifies the column name.

PERSISTENCE_TYPE

VARCHAR(16)

Specifies the type of column persistence. This can be specified as: SINGLE, PAGED, VIRTUAL_FILE, or VIRTUAL_PAGED.

MAIN_PHYSICAL_SIZE

BIGINT

Specifies the storage size, in bytes, used by the column.
