SERIES_TABLES System View

Provides series definitions for tables.

Structure

Column name

Data type

Description

SCHEMA_NAME

NVARCHAR(256)

The schema name

TABLE_NAME

NVARCHAR(256)

The series table name

PERIOD_TYPE_ID

TINYINT

The SQL data type ID of the period columns

PERIOD_TYPE_NAME

VARCHAR(16)

The SQL data type name of the period columns

IS_EQUIDISTANT

VARCHAR(5)

The series table has equidistant elements: TRUE, FALSE

IS_INSTANT

VARCHAR(5)

The series represents instants rather than a period with a start/end: TRUE, FALSE

INCREMENT_BY

NVARCHAR(51)

The delta value for an equidistant series or NULL for a non-equidistant series. When PERIOD_TYPE is a numeric type, a string representing the number. For datetime, a string of the form INTERVAL <number-units> where units is one of YEAR, MONTH, DAY, HOUR, MINUTE, SECOND

IS_MISSING_ALLOWED

VARCHAR(5)

Indicates whether missing values are allowed in a non-equidistant series: TRUE, FALSE, or NULL for a non-equidistant series

PERIOD_START_COLUMN

NVARCHAR(256)

The starting series column name, or NULL if it is IMPLIED or an INSTANT series

PERIOD_END_COLUMN

NVARCHAR(256)

The ending series column name, or NULL if it is IMPLIED or an INSTANT series

PERIOD_INSTANT_COLUMN

NVARCHAR(256)

The instant series column name, or NULL for an INTERVAL series

MIN_VALUE

NVARCHAR(5000)

The minimum value allowed in the series columns

MAX_VALUE

NVARCHAR(5000)

The maximum value allowed in the series columns

NVARCHAR(256)

Reserved

CALENDAR_TABLE_NAME

NVARCHAR(256)

Reserved

CALENDAR_TABLE_OID

BIGINT

Reserved
