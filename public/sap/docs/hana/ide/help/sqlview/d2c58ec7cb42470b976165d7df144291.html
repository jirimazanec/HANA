APPLICATION_ENCRYPTION_KEYS System View

Provides information about encryption keys used by applications.

Structure

Column name

Data type

Unit

Description

COMPONENT

NVARCHAR(20)

  

Specifies the component that the key belongs to

SUB_COMPONENT

NVARCHAR(512)

  

Specifies the subcomponent that the key belongs to

CREATOR

NVARCHAR(256)

  

Specifies the database user or the internal (sub)component that created the key

CREATE_TIMESTAMP

TIMESTAMP

  

Specifies the timestamp when the key was created

IS_CURRENT

VARCHAR(5)

  

Returns TRUE if the key is used by the subcomponent for newly encrypted data, FALSE otherwise
