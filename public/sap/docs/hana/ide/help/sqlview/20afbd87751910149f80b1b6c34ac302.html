M_EXTRACTORS System View

Structure

Column name

Data type

Description

NVARCHAR(256)

Name of the schema where the DSO tables are generated

NVARCHAR(256)

Name of the DSO specific status table

CLIENT

NVARCHAR(3)

The ID of the client

STATUS

VARCHAR(5)

General status overview saying OK or ERROR

REQUEST_ID

INTEGER

SID of the request used for activation

ERROR_MESSAGE

NVARCHAR(5000)

The message in case of an error

PHASE

VARCHAR(32)

The phase of the last running operation

TIMESTAMP

TIMESTAMP

Timestamp of the operation phase
