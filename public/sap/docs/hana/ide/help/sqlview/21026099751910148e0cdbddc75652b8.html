USERS System View

Lists all users.

Structure

Column name

Data type

Description

USER_NAME

NVARCHAR(256)

The name of the user

USER_ID

BIGINT

The ID of the user

USER_MODE

VARCHAR(8)

The mode of the user: LOCAL, GLOBAL, or EXTERNAL

EXTERNAL_IDENTITY

NVARCHAR(256)

The external identity of the user

CREATOR

NVARCHAR(256)

The creator of the user

CREATE_TIME

TIMESTAMP

The creation time

VALID_FROM

TIMESTAMP

The start time of user's validity

VALID_UNTIL

TIMESTAMP

The end time of user's validity

LAST_SUCCESSFUL_CONNECT

TIMESTAMP

The time of the last successful connect of the user

LAST_INVALID_CONNECT_ATTEMPT

TIMESTAMP

The time of the last invalid connect attempt

INVALID_CONNECT_ATTEMPTS

INTEGER

The number of invalid connect attempts since last successful connect

ADMIN_GIVEN_PASSWORD

VARCHAR(5)

Specifies if the password was given by administrator or by the user himself: TRUE or FALSE

LAST_PASSWORD_CHANGE_TIME

TIMESTAMP

The time of last password change

PASSWORD_CHANGE_NEEDED

VARCHAR(5)

Specifies if the user is forced to change his password himself: TRUE or FALSE

IS_PASSWORD_LIFETIME_CHECK_ENABLED

VARCHAR(5)

Specifies if the password-lifetime will be checked for this user: TRUE or FALSE

USER_DEACTIVATED

VARCHAR(5)

Specifies if user is deactivated: TRUE or FALSE

DEACTIVATION_TIME

TIMESTAMP

The time given with explicit deactivation command for this user

IS_PASSWORD_ENABLED

VARCHAR(5)

Specifies whether authentication using a password is enabled: TRUE or FALSE

IS_KERBEROS_ENABLED

VARCHAR(5)

Specifies whether authentication using KERBEROS is enabled: TRUE or FALSE

IS_SAML_ENABLED

VARCHAR(5)

Specifies whether authentication using SAML is enabled: TRUE or FALSE

IS_X509_ENABLED

VARCHAR(5)

Specifies whether authentication using an X.509 certificate is enabled: TRUE or FALSE

IS_SAP_LOGON_TICKET_ENABLED

VARCHAR(5)

Specifies whether authentication using an SAP logon ticket is enabled: TRUE or FALSE

IS_SAP_ASSERTION_TICKET_ENABLED

VARCHAR(5)

Specifies whether authentication using an SAP assertion ticket is enabled: TRUE or FALSE

IS_RESTRICTED

VARCHAR(5)

Specifies whether the user was created with option RESTRICTED: TRUE or FALSE

IS_CLIENT_CONNECT_ENABLED

VARCHAR(5)

Specifies whether the user is allowed to connect outside of applications: TRUE or FALSE

HAS_REMOTE_USERS

VARCHAR(5)

Specifies whether the user has a remote identity mapping: TRUE or FALSE

PASSWORD_CHANGE_TIME

TIMESTAMP

Deprecated: The time of the last password change, NULL, if password-lifetime is not checked for this user

AUTHORIZATION_MODE

VARCHAR(5)

Authorization mode of the user: LOCAL, LDAP

IS_JWT_ENABLED

VARCHAR(5)

Specifies whether authentication using JWT is enabled: TRUE or FALSE
