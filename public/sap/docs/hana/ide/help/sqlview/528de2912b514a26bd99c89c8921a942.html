M_JOIN_DATA_STATISTICS System View

Provides column store join engine join statistics.

Structure

Column name

Data type

Description

HOST

VARCHAR(64)

Host name

PORT

INTEGER

Internal port

SCHEMA_NAME1

NVARCHAR(256)

Specifies the schema name of the first table

TABLE_NAME1

NVARCHAR(256)

Specifies the name of the first table

COLUMN_NAME1

NVARCHAR(256)

Specifies the column name of the first table

SCHEMA_NAME2

NVARCHAR(256)

Specifies the schema name of the second table

TABLE_NAME2

NVARCHAR(256)

Specifies the name of the second table

COLUMN_NAME2

NVARCHAR(256)

Specifies the column name of the second table

LAST_REFRESH_TIME

TIMESTAMP

Specifies the time when the statistics was last refreshed.

RECORD_COUNT1

BIGINT

Specifies the number of rows of the first table

COUNT1

BIGINT

Specifies the number of not Null values of the column of the first table

DISTINCT_COUNT1

BIGINT

Specifies the number of unique not Null values of the column of the first table

VALUE_COUNTS1

NVARCHAR(1000)

Comma separated sorted list of unique column value counts. An incomplete output of the list will be truncated by asterisk sign

VALUE_COUNTS_FREQUENCIES1

NVARCHAR(1000)

Comma separated list of frequencies of first table's column values with same count. Any frequency refers to that value count with corresponding position in the list of value counts. An incomplete output of the list will be truncated by asterisk sign

MATCHING_VALUES_COUNTS1

NVARCHAR(1000)

Comma separated list of counts of first table's column values which find a join partner. Any matching values count refers to that value count with corresponding position in the list of value counts. An incomplete output of the list will be truncated by asterisk sign

MEMORY_SIZE

BIGINT

Specifies the memory size used by the join data statistics object
