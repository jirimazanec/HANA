VIRTUAL_TABLE_PARAMETERS System View

Provides a list of parameters of the virtual tables that refer to column views in a remote SAP HANA database.

Structure

Column name

Data type

 Unit

Description

SCHEMA_NAME

NVARCHAR(256)

  

The schema of the virtual table

OBJECT_NAME

NVARCHAR(256)

  

The name of the virtual table

PARAMETER_NAME

NVARCHAR(256)

  

The name of the parameter

IS_MANDATORY

VARCHAR(5)

  

Indicates if a parameter is mandatory or not. Possible values are TRUE and FALSE. If TRUE, the value of the parameter should be provided either by setting the default value or through the SQL query. If FALSE, setting the value of the parameter is optional

DEFAULT_VALUE

NVARCHAR(256)

  

The default value of the parameter. This column is set to an empty string when the parameter has no default value
