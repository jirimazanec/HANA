CURRENT_TIMESTAMP Function (Datetime)

Returns the current local system timestamp information.

Syntax
CURRENT_TIMESTAMP
Description

Returns the current local system timestamp information.

Use UTC times instead of local timestamps. The use of local times or conversion between local time zones might require additional handling in the application code.

Example

The following example returns the current timestamp:

SELECT CURRENT_TIMESTAMP "current timestamp" FROM DUMMY;
