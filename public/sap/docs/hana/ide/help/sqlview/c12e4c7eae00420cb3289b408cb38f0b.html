Additional System View [Dynamic Tiering]

Views including information pertaining to, but not limited to, SAP HANA dynamic tiering.

View Information Displayed
GRANTED_PRIVILEGES System View Privileges granted to users and roles.
GRANTED_ROLES System View Roles granted to users or other roles.
INDEXES System View Indexes on tables.
INDEX_COLUMNS System View Columns of indexes.
M_BACKUP_CATALOG System View Common data for all backup catalog entries.
M_BACKUP_CATALOG_FILES System View Location information of all backup catalog entries.
M_CONNECTIONS System View Detailed information on connections between a client and database. Information includes connection status, client information, connection type, resource utilization.
M_DISKS System View Disk configuration and utilization of the host machine.
M_EXPORT_BINARY_STATUS System View Export status information for current session.
M_FEATURES System View All supported features.
M_HISTORY_INDEX_LAST_COMMIT_ID System View Last commit ID of history index for each session.
M_HOST_INFORMATION System View Host information such as machine, OS configuration.
M_HOST_RESOURCE_UTILIZATION System View Host resource utilization, CPU time is in milliseconds and added across all cores since system start.
M_IMPORT_BINARY_STATUS System View Import status information for current session.
M_INIFILES All configuration files.
M_INIFILE_CONTENTS System View Configuration information from inifiles.
M_LANDSCAPE_HOST_CONFIGURATION System View Host roles in a distributed landscape.
M_MERGED_TRACES System View Contains the merged content of the server trace files for all of the SAP HANA services.
M_REMOTE_STATEMENTS System View Detailed information on executed remote queries. Information includes query status, number of fetched rows.
M_SERVICES System View Status of all services.
M_SERVICE_MEMORY System View Detailed information on memory utilization by services.
M_SERVICE_STATISTICS System View Statistics on active services.
M_SERVICE_TYPES System View Service types.
M_SYSTEM_LIMITS System View System limits information.
M_TABLES System View Information on row and column tables
M_TRACEFILE_CONTENTS System View SAP HANA information from trace files.
M_TRACEFILES System View All trace files.
M_VOLUMES System View Volumes used by SAP HANA servers.
M_VOLUME_FILES System View Information about volume files.
M_VOLUME_SIZES System View Volume sizes used by SAP HANA servers.
ROLES System View Shows available roles.
TABLES System View Available tables.
TABLE_COLUMNS System View Available table columns.
TRANSACTION_HISTORY System View Committed transactions and their users.
USERS System View All users.
