TIMEZONES System View

Provides information about the timezones which are available together with their originating data set.

Structure

Column name

Data type

Description

TIMEZONE_NAME

NVARCHAR(64)

Gives the name of the respective timezone that can be used as parameter to give the source respective destination timezone for UTCTOLOCAL and LOCALTOUTC.

TIMEZONE_DATASET

NVARCHAR(8)

Specifies the data set in which the timezones definition is located. It can take the values sap or platform as they are allowed for the dataset parameter to UTCTOLOCAL/LOCALTOUTC. The value of this column can thus be directly used as an input for the respective UTCTOLOCAL/LOCALTOUTC parameter.
