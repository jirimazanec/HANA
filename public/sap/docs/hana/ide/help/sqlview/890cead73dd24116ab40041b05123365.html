M_CS_MVCC System View

Provides column store MVCC information.

Structure
Column name Data type Description
SCHEMA_NAME NVARCHAR(256) Schema name
TABLE_NAME NVARCHAR(256) Table name
PART_ID INTEGER Partition id
IS_HISTORY VARCHAR(5) Flag indicating if partition is a history partition or not: TRUE, FALSE
CTS_BLOCK_COUNT BIGINT Number of blocks used to store creation timestamps (CTS)
DTS_BLOCK_COUNT BIGINT Number of blocks used to store deletion timestamps (DTS)
CTS_MEMORY_SIZE BIGINT Memory size used to store CTS blocks
DTS_MEMORY_SIZE BIGINT Memory size used to store DTS blocks
