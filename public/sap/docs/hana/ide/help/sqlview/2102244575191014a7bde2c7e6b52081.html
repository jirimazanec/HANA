USER_PARAMETERS System View

Lists all parameters and their values, which have been assigned to users in the system (using CREATE USER ... SET PARAMETER or ALTER USER ... SET PARAMETER).

Structure

Column name

Data type

Description

USER_NAME

NVARCHAR(256)

The name of the user

PARAMETER

NVARCHAR(256)

The parameter name: CLIENT

VALUE

NVARCHAR(5000)

The parameter value
