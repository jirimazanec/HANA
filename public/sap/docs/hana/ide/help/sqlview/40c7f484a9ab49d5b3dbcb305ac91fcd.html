ENCRYPTION_ROOT_KEYS System View

Provides information about root keys.

Structure

Column name

Data type

Unit

Description

ROOT_KEY_TYPE

VARCHAR(16)

  

The usage type of the root key: PERSISTENCE, DPAPI, LOG, BACKUP

ROOT_KEY_VERSION

INTEGER

  

The version of the root key

CREATE_TIMESTAMP

TIMESTAMP

  

The creation time of the root key (UTC)

IS_CONSISTENT

VARCHAR(5)

  

Returns TRUE if the key is consistent between persistence and the SSFS file, and FALSE otherwise

RESET_COUNT

INTEGER

  

The number of consistency resets

ROOT_KEY_STATUS

VARCHAR(11)

  

The state of the root key: ACTIVE, PREACTIVE, or DEACTIVATED
