INVALID_CONNECT_ATTEMPTS System View

Provides the number of invalid connect attempts for a user between two successful connections.

Structure

Column name

Data type

Description

NVARCHAR(256)

Name of the user

SUCCESSFUL_CONNECT_TIME

TIMESTAMP

Time of the valid connect attempt

INVALID_CONNECT_ATTEMPTS

BIGINT

Number of invalid connect attempts for this user between last successful connect and given time
