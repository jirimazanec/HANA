Full description

This view shows information on which node contains the persistence parts of a table. This includes the assigned node as visible from the view M_TABLE_LOCATIONS but also includes other nodes that still contain some persistence of the table. This could occur if a table is moved and not yet merged. In this case the old node will still contain some persistence content for the table beside the currently assigned node.
