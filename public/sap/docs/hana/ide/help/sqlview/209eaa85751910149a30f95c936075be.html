Data Manipulation Statements

The following statements enable you to manipulate data within the SAP HANA database.

Supported Statements

  • DELETE Statement (Data Manipulation)

  • EXPLAIN PLAN Statement (Data Manipulation)

  • INSERT Statement (Data Manipulation)

  • LOAD Statement (Data Manipulation)

  • MERGE DELTA Statement (Data Manipulation)

  • MERGE INTO Statement (Data Manipulation)

  • REPLACE | UPSERT Statement (Data Manipulation)

  • SELECT Statement (Data Manipulation)

  • TRUNCATE TABLE Statement (Data Manipulation)

  • UNLOAD Statement (Data Manipulation)

  • UPDATE Statement (Data Manipulation)
