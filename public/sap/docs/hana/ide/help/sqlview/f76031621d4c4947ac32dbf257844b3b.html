M_TRANS_TOKENS System View

Provides information about all active transaction tokens.

Structure

Column name

Data type

Description

HOST

VARCHAR(64)

The host name

PORT

INTEGER

The internal port

TRANS_TOKEN_ID

BIGINT

The transaction token ID

PRIMARY_TRANS_TOKEN_ID

BIGINT

The primary transaction token ID for the shipped transaction token. If transaction token is local-only, then this has same value as TRANS_TOKEN_ID

TRANSACTION_ID

INTEGER

The associated transaction object ID

LOGICAL_CONNECTION_ID

INTEGER

The associated logical connection ID

STATEMENT_ID

BIGINT

The associated logical statement ID

MVCC_SNAPSHOT_TIMESTAMP

BIGINT

The MVCC timestamp that is held by the transaction token

MVCC_SNAPSHOT_SCOPE

VARCHAR(8)

The MVCC snapshot scope : TABLE or GLOBAL

ES_SNAPSHOT_COUNT

BIGINT

The number of extended storage's snapshots

ES_SNAPSHOTS

VARCHAR(1024)

A list of volume and snapshot IDs for all extended storage snapshots
