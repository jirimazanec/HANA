M_REMOTE_STATEMENTS System View

Displays detailed information about executed remote queries. This information includes the query status and the number of fetched rows.

Data type

Unit

Description

Specifies the ID of the remote connection

REMOTE_SOURCE_NAME

NVARCHAR(256)

  

Specifies the remote source name

START_TIME

TIMESTAMP

  

Specifies the statement start time

END_TIME

TIMESTAMP

  

Specifies the statement end time

FETCHED_RECORD_COUNT

Counter

VARCHAR(16)
