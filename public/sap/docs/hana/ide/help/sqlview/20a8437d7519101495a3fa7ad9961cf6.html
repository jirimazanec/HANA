M_BACKUP_CATALOG System View

Provides common data for all backup catalog entries.

Structure

Column name

Data type

Description

ENTRY_ID

BIGINT

Specifies the unique ID of a backup catalog entry.

ENTRY_TYPE_NAME

VARCHAR(64)

Specifies the classification of backup catalog entries. The following types are supported: complete data backup, data snapshot, log backup, log missing, differential, and incremental backups

BACKUP_ID

BIGINT

Specifies the unique ID of a data backup or a log backup, respectively. All backup files of a single data backup share the same BACKUP_ID.

SYS_START_TIME

TIMESTAMP

Specifies the start time given in server local time.

UTC_START_TIME

TIMESTAMP

Specifies the start time given in UTC.

SYS_END_TIME

TIMESTAMP

Specifies the stop time given in server local time.

UTC_END_TIME

TIMESTAMP

Specifies the stop time given in UTC.

STATE_NAME

VARCHAR(64)

Specifies the result of the corresponding action: successful, failed, running, cancel pending, or canceled.

COMMENT

VARCHAR(256)

Specifies any additional information.

MESSAGE

VARCHAR(512)

Specifies any additional information.

SYSTEM_ID

VARCHAR(3)

Specifies the system identifier (SID) of the SAP HANA database.

ENCRYPTION_ROOT_KEY_HASH

VARCHAR(64)

Specifies the key used, if any, for encrypting the backup.

SOURCE_DATABASE_NAME

NVARCHAR(256)

Specifies the name of the database that creates the backup.
