M_CS_LOB_SPACE_RECLAIMS System View

Provides information regarding executed LOB garbage collection runs.

Structure

Column name

Data type

Unit

Description

HOST

VARCHAR(64)

  

The name of the host.

PORT

INTEGER

  

The internal port.

VOLUME_ID

INTEGER

  

The persistence volume ID.

COLLECTION_SCOPE

VARCHAR(8)

  

The scope of the garbage collection run on either VOLUME or TABLE.

SCHEMA_NAME

NVARCHAR(256)

  

The schema name of the scanned table if the scope is TABLE.

TABLE_NAME

NVARCHAR(256)

  

The name of the scanned table if the scope is TABLE.

START_TIME

TIMESTAMP

  

The start TIMESTAMP of the garbage collection run.

END_TIME

TIMESTAMP

  

The end TIMESTAMP of the garbage collection run.

SCANNED_COLUMNS_COUNT

BIGINT

 Counter

The number of overall scanned columns.

REMOVED_FILE_LOBS_COUNT

BIGINT

 Counter

The number of removed file LOBs.

REMOVED_PACKED_LOBS_COUNT

BIGINT

 Counter

The number of removed packed LOBs.

ERROR_CODE

INTEGER

  

The first error code if the execution fails. More details are available in the ERROR_MESSAGE field.

ERROR_MESSAGE

NVARCHAR(2000)

  

The detailed error message.
