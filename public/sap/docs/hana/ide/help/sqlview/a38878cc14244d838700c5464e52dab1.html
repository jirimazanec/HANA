M_ES_DELTA_MERGE_STATISTICS System View [Dynamic Tiering]

History of internal and user driven merge operations for delta enabled extended tables.

Caution This information applies only for the SAP HANA option cited in the title of this topic. You must have purchased the license for it in order to use the information contained in this topic.
Structure
Column name Data type Description
SCHEMA_NAME NVARCHAR(255) Schema name.
TABLE_NAME NVARCHAR(255) Table name.
START_TIME TIMESTAMP Time that the delta merge started.
END_TIME TIMESTAMP Time that the delta merge ended.
STATUS VARCHAR(16) Status of the merge operation: 'STARTED', 'COMPLETED', 'FAILED'
ERROR_CODE INTEGER SQL code of the merge after completion.
ERROR_TEXT NVARCHAR(255) Additional error information.
MOTIVATION VARCHAR(16) Cause of the merger trigger: 'DML', 'DDL', 'SHUTDOWN', 'USER'
IS_BLOCKING VARCHAR(5) Specify if the merge operation was blocking: 'TRUE', 'FALSE'
INSERT_RECORD_COUNT BIGINT Number of rows that were inserted as a result of the merge.
UPDATE_RECORD_COUNT BIGINT Number of rows that were inserted as a result of the merge.
DELETE_RECORD_COUNT BIGINT Number of rows that were inserted as a result of the merge.
UNCOMMITTED_RECORD_COUNT BIGINT Number of rows that were uncommitted at the time of the merge.
