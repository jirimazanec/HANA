FUNCTIONS System View

Provides information about available functions.

Structure

Column name

Data type

Description

SCHEMA_NAME

NVARCHAR(256)

Specifies the schema name of the function

FUNCTION_NAME

NVARCHAR(256)

Specifies the name of the function

FUNCTION_OID

BIGINT

Specifies the object ID of the function

SQL_SECURITY

VARCHAR(7)

Specifies the SQL security setting of the function: DEFINER, INVOKER

DEFAULT_SCHEMA_NAME

NVARCHAR(256)

Specifies the schema name of the unqualified objects in the function

INPUT_PARAMETER_COUNT

INTEGER

Specifies the input type parameter count

RETURN_VALUE_COUNT

INTEGER

Specifies the return value type parameter count

IS_UNICODE

VARCHAR(5)

Specifies whether the function contains Unicode: TRUE, FALSE

DEFINITION

NCLOB

Specifies the query string of the function

FUNCTION_TYPE

VARCHAR(10)

Specifies the type of function

FUNCTION_USAGE_TYPE

VARCHAR(9)

Specifies the usage type of the function: SCALAR, TABLE, AGGREGATE, WINDOW

IS_VALID

VARCHAR(5)

Specifies whether the function is valid or not. This value becomes FALSE when its base objects are changed or dropped: TRUE, FALSE

IS_HEADER_ONLY

VARCHAR(5)

Specifies whether the function is a header-only function or not: TRUE, FALSE

IS_DETERMINISTIC

VARCHAR(5)

Specifies whether the function is deterministic or not: TRUE, FALSE

OWNER_NAME

NVARCHAR(256)

Specifies the name of the owner of the function

CREATE_TIME

TIMESTAMP
