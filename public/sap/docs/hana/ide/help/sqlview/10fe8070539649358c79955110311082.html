SET PSE Statement (System Management)

Sets a purpose for a PSE store.

Syntax
SET PSE <pse_name> PURPOSE <purpose>
Syntax Elements
<pse_name>

Specifies the name of the PSE store.

<pse_name> ::= <identifier>
PURPOSE <purpose_name>
<purpose_name> ::= <identifier>

The PSE store is assigned one of the following purposes:

  • JWT

  • SAML

  • SAP LOGON

  • SSL

  • X509

  • JWT

Description

This command can be executed by users with the REFERENCES privilege on the specified PSE store. In addition, the user must have the privilege required by the purpose. For authentication purposes, this is the TRUST ADMIN privilege. For SSL, this is the system privilege SSL ADMIN.

JWT can be used if a PSE store with purpose JWT is created and contains a valid X509 certificate to validate the JWT.

Examples

Set the purpose SAP LOGON for the examplepse PSE store: 

SET PSE examplepse PURPOSE SAP LOGON
