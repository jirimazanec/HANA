M_SERVICE_NETWORK_METHOD_IO System View

Displays the number of calls and amount of data that is sent and received.

Structure

Column name

Data type

Unit

Description

SENDER_HOST

VARCHAR(64)

Specifies the host name of the sender.

SENDER_PORT

INTEGER

Specifies the host port of the sender.

RECEIVER_HOST

NVARCHAR(256)

Specifies the host name of the receiver.

RECEIVER_PORT

NVARCHAR(256)

Specifies the host port of the receiver.

THREAD_METHOD

NVARCHAR(256)

Specifies the originating method name.

OPERATOR

NVARCHAR(256)

Specifies additional details.

REQUEST_COUNT

BIGINT

Counter

Specifies the total number of requests for the method.

SEND_SIZE

BIGINT

Byte

Specifies the total number of bytes sent.

RECEIVE_SIZE

BIGINT

Byte

Specifies the total number of bytes received.

LAST_UPDATE_TIME

TIMESTAMP

Specifies the last updated timestamp.
