STREAMING_PROJECT_CONNECTIONS_STATISTICS System View (Embedded Statistics Service)

Provides detailed data for the connections of smart data streaming projects. The table contains information only for the last 3 days.

Structure

Column name

Data type

Description

SNAPSHOT_ID TIMESTAMP Timestamp (UTC) identifying when the statistics service performed a data collection. Data collections scheduled for the same time have the same snapshot ID.
SERVER_TIMESTAMP TIMESTAMP Timestamp (local server time) identifying when a data collection actually occurred on the server.
INDEX NVARCHAR(524) Index
SCHEMA_NAME NVARCHAR(256) Schema in which the project is deployed
PROJECT_NAME NVARCHAR(256) Name given to the project. Corresponds to the APPLICATION_NAME in M_STREAMING_APPLICATIONS
PROJECT_INSTANCE INTEGER Instance number of the project. It corresponds to the APPLICATION_INSTANCE in M_STREAMING_APPLICATIONS
TOTAL_PUB_COUNT BIGINT Total number of publishers for a streaming project
TOTAL_PUB_ROW_THROUGHPUT REAL Total number of publisher throughput for a streaming project, in rows per second
TOTAL_PUB_TRANSACTION_THROUGHPUT REAL Total number of publisher throughput for a streaming project, in transactions per second
TOTAL_SUB_COUNT BIGINT Total number of subscribers for a streaming project
TOTAL_SUB_ROW_THROUGHPUT REAL Total number of subscriber throughput for a streaming project, in rows per second
TOTAL_SUB_TRANSACTION_THROUGHPUT REAL Total number of subscriber throughput for a streaming project, in transactions per second
TOTAL_CONN_COUNT BIGINT Total number of subscribers and publishers for a streaming project
