M_ES_OVERVIEW System View [Dynamic Tiering]

Status information of SAP HANA dynamic tiering as named value pair.

Caution This information applies only for the SAP HANA option cited in the title of this topic. You must have purchased the license for it in order to use the information contained in this topic.
Structure
Column name Data type Description
CATEGORY VARCHAR(32) Describes the main category type for which the parameters and values are shown. The following categories are reported:
  • GENERAL
  • BUFFERS
  • MEMORY
  • STORAGE
  • DELTA
  • BACKUP
  • VERSIONING
NAME VARCHAR(255) Name of the parameter of which the information is provided in value column.
VALUE VARCHAR(255) Value of the parameter.
