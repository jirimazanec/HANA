M_STATISTICS_LASTVALUES System View

Provides an in-memory version of _SYS_STATISTICS.STATISTICS_LASTVALUES.

Structure
     
HOST VARCHAR(64) Host name
PORT INTEGER Port name
NAME NVARCHAR(128) Value name
INDEX NVARCHAR(1024) Index of value
REACHED_AT TIMESTAMP Value reached at
VALUE NVARCHAR(5000) Last value
Full description

The Statistics Server when running has a view _SYS_STATISTICS.STATISTICS_LASTVALUES on which this view is based.

Note The new Embedded Statistics Service (ESS) does not use the view _SYS_STATISTICS.STATISTICS_LASTVALUES, which means this view is therefore empty.
