M_COLLATIONS System View

Provides information about SQL collations. This view is for internal use only.

Structure

Column name

Data type

Description

COLLATION_NAME

VARCHAR(16)

Specifies the collation name.

COLLATION_LANGUAGE

VARCHAR(32)

Specifies the language.

COLLATION_DESCRIPTION

NVARCHAR(1024)

Specifies the description.
