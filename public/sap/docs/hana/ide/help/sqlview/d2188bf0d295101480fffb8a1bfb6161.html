M_SQL_CLIENT_NETWORK_IO System View

Provides the client and server elapsed time as well as message sizes for client network messages.

Structure

Column name

Data type

Description

HOST

VARCHAR(64)

Host name

PORT

INTEGER

Internal port

CLIENT_HOST

NVARCHAR(256)

The host of the client machine, also found in M_CONNECTIONS.CLIENT_HOST

CONNECTION_ID

INTEGER

The connection ID of the physical server connection

MESSAGE_ID

BIGINT

The client message ID on the physical server connection

CLIENT_DURATION

BIGINT

Elapsed time on client side including send, server execution and receive

SERVER_DURATION

BIGINT

Execution time on server side. Transport time is the delta between CLIENT_DURATION and SERVER_DURATION

SERVER_RECEIVED_TIME

TIMESTAMP

Time when message was received on server side and begin of execution

RECEIVED_MESSAGE_SIZE

BIGINT

Size of messages received on server side

SEND_MESSAGE_SIZE

BIGINT

Size of messages sent to client side
