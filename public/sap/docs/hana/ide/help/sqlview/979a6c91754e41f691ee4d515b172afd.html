STATISTICS_EMAILRECIPIENTS System View (Embedded Statistics Service)

Statistics alerts email recipients per host. This table contains information only for the last 7 days.

Structure

Column name

Data type

Description

ALERT_ID INTEGER Unique check ID
RECIPIENT NVARCHAR(1024) Recipient's e-mail address
FREQUENCY NVARCHAR(256) Specifies how often an e-mail is sent when the check issues an alert. The following values are possible: onchange, always. Onchange (default) means that one e-mail is sent when the alert occurs for the first time and one when it stops. Always means that e-mails are sent continuously while the alert is current.
