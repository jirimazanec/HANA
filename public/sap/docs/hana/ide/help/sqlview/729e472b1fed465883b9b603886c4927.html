M_DSO_OPERATIONS System View

Provides information about data store object (DSO) operations.

Structure

Column name

Data type

Unit

Description

HOST

VARCHAR(64)

Specifies the host name.

PORT

INTEGER

Specifies the internal port.

OPERATION

VARCHAR(32)

Specifies the operation: ACTIVATION or ROLLBACK.

USAGE_MODE

VARCHAR(32)

Specifies the usage mode. This clarifies which variant of DSO is used:

  • ADSO for advanced DSOs
  • ODSO for classic DSOs with persistent change log tables
  • CDSO for CDS-based native DSOs
  • IMO for older in-memory optimized DSOs (not recommended)

ACTIVATION_IDS

NVARCHAR(1024)

Specifies the activations IDs.

TARGET_SCHEMA_NAME

NVARCHAR(256)

Specifies the schema name of the target table.

TARGET_TABLE_NAME

NVARCHAR(256)

Specifies the name of the target table.

START_TIME

TIMESTAMP

Specifies the start time of the operation.

END_TIME

TIMESTAMP

Specifies the end time of the operation.

ERROR_CODE

INTEGER

Specifies the last error code in the case of a failure; otherwise specifies 0.

ERROR_MESSAGE

NVARCHAR(2000)

Specifies the last error message in the case of a failure; otherwise, specifies "no error".

SOURCE_RECORD_COUNT

BIGINT

Counter

Specifies the number of source table rows activated.

INSERT_RECORD_COUNT

BIGINT

Counter

Specifies the number of rows inserted into the target table.

DELETE_RECORD_COUNT

BIGINT

Counter

Specifies the number of rows deleted from the target table.

UPDATE_RECORD_COUNT

BIGINT

Counter

Specifies the number of target table rows that were updated.

CHANGE_LOG_RECORD_COUNT

BIGINT

Counter

Specifies the number of rows inserted into the change log.

PARAMETERS

NCLOB(2147483647)

Specifies the call parameters.

CONFIGURATION

NCLOB(2147483647)

Specifies the non-default, DSO-specific ini file settings used by the operation.

USER_NAME

NVARCHAR(256)

Specifies the user name.

APPLICATION_USER_NAME

NVARCHAR(256)

Specifies the application user name.
