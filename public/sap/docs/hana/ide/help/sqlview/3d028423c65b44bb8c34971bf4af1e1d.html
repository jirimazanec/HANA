PROCEDURE_PARAMETER_COLUMNS System View

Lists available columns of table parameters of stored procedures.

Structure

Column name

Data type

Description

SCHEMA_NAME

NVARCHAR(256)

The schema name of the stored procedure

PROCEDURE_NAME

NVARCHAR(256)

The same of the stored procedure

PROCEDURE_OID

BIGINT

The object ID of the stored procedure

PARAMETER_NAME

NVARCHAR(256)

The parameter name

PARAMETER_POSITION

INTEGER

The ordinal position of the parameter

COLUMN_NAME

NVARCHAR(256)

The name of the column in the table parameter

POSITION

INTEGER

The ordinal position of the column in the table parameter

DATA_TYPE_NAME

VARCHAR(16)

The SQL data type name of the column

LENGTH

INTEGER

The number of chars for char types, number of max digits for numeric types; number of chars for datetime types, number of bytes for LOB types

SCALE

INTEGER

For numeric types: the maximum number of digits to the right of the decimal point; for time and timestamp: the decimal digits are defined as the number of digits to the right of the decimal point in the second's component of the data

IS_NULLABLE

VARCHAR(5)

Specifies whether the column is allowed to accept null values: TRUE or FALSE
