M_CACHES System View

Provides aggregated information on caches.

This view has a resettable counterpart. This means, you can also see the values since the last reset in the dependent view M_CACHES_RESET. To reset the values, use the SQL command: ALTER SYSTEM RESET MONITORING VIEW SYS.M_CACHES_RESET or the management console command: monitor reset -n M_CACHES_RESET
