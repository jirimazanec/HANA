AUDIT_LOG System View

Provides information about audit records.

Structure

Column name

Data type

Description

TIMESTAMP

TIMESTAMP

Specifies the time that the event occurred

HOST

VARCHAR(64)

Specifies the name of the host where the event occurred

PORT

INTEGER

Specifies the port number

SERVICE_NAME

VARCHAR(32)

Specifies the name of the service

CONNECTION_ID

INTEGER

Specifies the ID of the connection

CLIENT_HOST

NVARCHAR(256)

Specifies the IP of the client host

CLIENT_IP

VARCHAR(45)

Specifies the IP of the client application

CLIENT_PID

BIGINT

Specifies the PID of the client process

CLIENT_PORT

INTEGER

Specifies the port of the client process

USER_NAME

NVARCHAR(256)

Specifies the name of the user connected to the database

APPLICATION_USER_NAME

NVARCHAR(256)

Specifies the name of the application user

XS_APPLICATION_USER_NAME

NVARCHAR(256)

Specifies the name of the XS application user

AUDIT_POLICY_NAME

NVARCHAR(256)

Specifies the name of the Audit Policy hit

EVENT_STATUS

VARCHAR(32)

Specifies whether the event was successful or not

EVENT_LEVEL

VARCHAR(16)

Specifies the severity level of the event

EVENT_ACTION

VARCHAR(64)

Specifies the action performed by the audit event

SCHEMA_NAME

NVARCHAR(256)

Specifies the name of schema

OBJECT_NAME

NVARCHAR(256)

Specifies the name of object

PRIVILEGE_NAME

NVARCHAR(256)

Specifies the name of the privilege granted

ROLE_SCHEMA_NAME

NVARCHAR(256)

Specifies the schema name of the granted role

ROLE_NAME

NVARCHAR(256)

Specifies the name of the granted role

GRANTEE_SCHEMA_NAME

NVARCHAR(256)

Specifies the schema name of the grantee in GRANT/REVOKE statements

GRANTEE

NVARCHAR(256)

Specifies the grantee in GRANT/REVOKE statements

GRANTABLE

VARCHAR(16)

Specifies whether the privilege/role being granted is grantable or not

FILE_NAME

VARCHAR(256)

Specifies the configuration file that was changed

SECTION

VARCHAR(128)

Specifies the configuration that was changed

KEY

VARCHAR(128)

Specifies the attribute being changed

PREV_VALUE

VARCHAR(5000)

Specifies the old value of the attribute

VALUE

VARCHAR(5000)

Specifies the new value of the attribute

STATEMENT_STRING

NCLOB

Specifies the SQL statement which caused the event

COMMENT

VARCHAR(5000)

Specifies any extra information on the event

ORIGIN_DATABASE_NAME

NVARCHAR(256)

Specifies the origin database name on cross database queries

ORIGIN_USER_NAME

NVARCHAR(256)

Specifies the origin user name on cross database queries
