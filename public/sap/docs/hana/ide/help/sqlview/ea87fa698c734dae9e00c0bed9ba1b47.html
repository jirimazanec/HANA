CDS_ARTIFACT_NAMES System View

Provides CDS artifact names.

Structure

Column name

Data type

Description

SCHEMA_NAME

NVARCHAR(256)

Schema name

ARTIFACT_NAME

NVARCHAR(127)

Artifact name

ARTIFACT_KIND

VARCHAR(32)

The artifact kind is one of the following enumeration values: ANNOTATION, ARRAY_TYPE, ASSOCIATION, CONTEXT, DERIVED_TYPE, ENTITY, ENUM, STRUCTURED_TYPE, VIEW

For more information, see the SAP HANA Core Data Services (CDS) Reference Guide.
