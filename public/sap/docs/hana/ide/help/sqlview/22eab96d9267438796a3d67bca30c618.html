HOST_HEAP_ALLOCATORS System View (Embedded Statistics Service)

Memory allocator statistics per host. Only allocators where INCLUSIVE_SIZE_IN_USE > 1,000,000,000 bytes are listed. The table contains information only for the last 42 days.
