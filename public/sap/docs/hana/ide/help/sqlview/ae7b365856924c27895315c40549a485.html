RESULT_CACHE System View

Provides information about objects available to use the result cache.

Structure

Column name

Data type

Description

SCHEMA_NAME

NVARCHAR(256)

The schema name

OBJECT_NAME

NVARCHAR(256)

The object name

OBJECT_TYPE

VARCHAR(32)

The object type: VIEW or FUNCTION

OBJECT_OID

BIGINT

The object ID

CACHE_TYPE

VARCHAR(40)

Indicates which type of cache is enabled for the object: FULL, PARTIAL (SELECTED COLUMNS), PARTIAL (FILTERED), PARTIAL (SELECTED COLUMNS, FILTERED)

CACHE_RETENTION

INT

The cache refresh interval in minutes

CACHE_FILTER

NCLOB

The filter condition specified to reduce the cache size

IS_CACHE_FORCED

VARCHAR(5)

Indicates whether the cached is forced or if cachability is checked: TRUE or FALSE
