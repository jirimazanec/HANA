STATISTICS_ALERT_THRESHOLDS System View (Embedded Statistics Service)

Statistics alerts thresholds per host. This table contains information only for the last 7 days.

Structure

Column name

Data type

Description

ALERT_ID INTEGER Unique ID of check
SEVERITY TINYINT Severity of the alert issued when the threshold value is reached. The following values are possible: 1 (info), 2 (warning1), 3 (warning2), 4 (warning3) , 5 (error).
UNIT NVARCHAR(256) Unit of the threshold value (for example, millisecond, percent, and so on). This value is freely definable and used for information only.
DEFAULT_VALUE NVARCHAR(256) Default threshold value. This value cannot be changed. It is used when there is no value specified in the CURRENT_VALUE column.
CURRENT_VALUE NVARCHAR(256) User-defined threshold value. This value overrides the value in the DEFAULT_VALUE column.
REACHED_AT TIMESTAMP Time (UTC) at which the threshold value was reached
