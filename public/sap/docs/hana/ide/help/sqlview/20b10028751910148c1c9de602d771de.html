M_HOST_INFORMATION System View

Provides host information such as machine and OS configuration.

Column name

Data type

Description

VARCHAR(64)

VARCHAR(256)

Value

Full description

This view contains some host specific information. The following describes the contents of the individual lines of the monitor view (VALUE), as identified by KEY:. The values are updated once per minute. Most values require the INIFILE ADMIN privilege to view, as noted in the table.
