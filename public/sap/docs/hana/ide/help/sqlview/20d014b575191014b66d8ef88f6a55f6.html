ALTER INDEX Statement (Data Definition)

Rebuilds an index.

Syntax
ALTER INDEX <index_name> REBUILD
Syntax Elements
<index_name>

Specifies the name of the index to be rebuilt with an optional schema name.

<index_name> ::= [<schema_name>.]<identifier>

<schema_name> ::= <unicode_name>
Description

The ALTER INDEX statement rebuilds an index.

Example

Create table A and an index i on column b of table A.

CREATE TABLE A (a INT, b NVARCHAR(10), c NVARCHAR(20));
 CREATE INDEX i ON A(b);

Rebuild index i.

ALTER INDEX i REBUILD;
