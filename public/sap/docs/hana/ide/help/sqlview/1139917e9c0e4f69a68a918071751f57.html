M_XB_MESSAGING_CONNECTIONS System View

Provides information about existing message connections. This view is for internal use only.

Structure

Column name

Data type

Unit

Description

HOST

VARCHAR(64)

Specifies the host name.

PORT

INTEGER

Specifies the internal port.

USER_NAME

NVARCHAR(256)

Specifies the user name.

DESTINATION_ID

VARCHAR(20)

Specifies the destination ID.

DESTINATION_TYPE

VARCHAR(20)

Specifies the destination type.

REMOTE_USER_NAME

NVARCHAR(256)

Specifies the remote logon user.

CONNECTION_STATE

VARCHAR(16)

Specifies the connection status. The status is specified as: CONNECTING, CONNECTED, DISCONNECTED, or ERROR.

SUBSCRIPTION_COUNT

BIGINT

Specifies the number of subscriptions.

RECEIVED_MESSAGE_COUNT

BIGINT

Counter

Specifies the number of messages received.

SENT_MESSAGE_COUNT

BIGINT

Counter

Specifies the number of messages sent.

RECEIVED_MESSAGE_SIZE

BIGINT

Byte

Specifies the number of bytes received.

SENT_MESSAGE_SIZE

BIGINT

Byte

Specifies the number of bytes sent.

ERROR_COUNT

BIGINT

Counter

Specifies the number of errors.

LAST_ERROR_DESCRIPTION

NVARCHAR(512)

Specifies the last error description.
