CDS_ANNOTATION_ASSIGNMENTS System View

Provides CDS annotation assignments.

Structure

Column name

Data type

Description

SCHEMA_NAME

NVARCHAR(256)

Specifies the schema name.

ARTIFACT_NAME

NVARCHAR(127)

Specifies the artifact name.

COMPONENT_NAME

NVARCHAR(127)

Specifies the component name.

ANNOTATION_NAME

NVARCHAR(127)

Specifies the annotation name.

EXTENSION_PACKAGE_NAME

NVARCHAR(127)

Specifies the extension package name.

FORMAT

NVARCHAR(127)

Specifies the format of the annotation value.

VALUE

NCLOB(2147483647)

Specifies the value.

VALIDATION_RESULT

NVARCHAR(127)

Specifies the validation result of the annotation value.

DEFINITION_SCHEMA_NAME

NVARCHAR(256)

Specifies the schema name of the annotation definition.

DEFINITION_NAME

NVARCHAR(127)

Specifies the name of the annotation definition.

For more information, see the SAP HANA Core Data Services (CDS) Reference Guide.
