TABLES System View

Provides information about available tables.

Structure

Column name

Data type

Description

SCHEMA_NAME

NVARCHAR(256)

Specifies the schema name.

TABLE_NAME

NVARCHAR(256)

Specifies the table name.

TABLE_OID

BIGINT

Specifies the object ID of the table.

COMMENTS

NVARCHAR(5000)

Specifies the table description.

FIXED_PART_SIZE

SMALLINT

Specifies the fixed part size of the table.

IS_LOGGED

VARCHAR(5)

Specifies whether or not logging was on for the table at last restart time: TRUE or FALSE.

IS_SYSTEM_TABLE

VARCHAR(5)

Deprecated.

IS_COLUMN_TABLE

VARCHAR(5)

Specifies whether the table is a column table: TRUE or FALSE.

TABLE_TYPE

VARCHAR(16)

Specifies the type of the table: ROW, COLUMN, HYBRID, VIRTUAL, EXTENDED, HYBRID EXTENDED, and COLLECTION.

IS_INSERT_ONLY

VARCHAR(5)

Specifies whether the table is an insert only table: TRUE, FALSE.

IS_TENANT_SHARED_DATA

VARCHAR(5)

Specifies whether the table can be shared among other instances: TRUE, FALSE.

IS_TENANT_SHARED_METADATA

VARCHAR(5)

Specifies whether the table is a global table: TRUE, FALSE.

SESSION_TYPE

VARCHAR(7)

Specifies the session type: NONE, SIMPLE, or HISTORY. For HISTORY, time travel is possible.

IS_TEMPORARY

VARCHAR(5)

Specifies whether the table is a temporary table: TRUE or FALSE.

TEMPORARY_TABLE_TYPE

VARCHAR(10)

Specifies the temporary table type.

COMMIT_ACTION

VARCHAR(8)

Specifies the setting for on commit deletion for global temporary tables.

IS_USER_DEFINED_TYPE

VARCHAR(5)

Returns TRUE if this is a user defined table type, and FALSE otherwise.

HAS_PRIMARY_KEY

VARCHAR(5)

Returns TRUE if the table has a primary key, and FALSE otherwise. Only valid for column store tables.

PARTITION_SPEC

NCLOB

Returns the detail specification of table partitioning. Only valid for column store tables.

USES_EXTKEY

VARCHAR(5)

Returns TRUE if the table uses an external key, and FALSE otherwise. Only valid for column store tables.

AUTO_MERGE_ON

VARCHAR(5)

Returns TRUE if automatic delta table merge is activated for the table, and FALSE otherwise. Only valid for column store tables.

USES_DIMFN_CACHE

VARCHAR(5)

Returns TRUE if the table uses the DimFunctionCache feature, and FALSE otherwise. Only valid for column store tables.

IS_PUBLIC

VARCHAR(5)

Returns TRUE if the table is public, and FALSE otherwise. Only valid for column store tables.

AUTO_OPTIMIZE_COMPRESSION_ON

VARCHAR(5)

Returns TRUE if automatic optimize compression is activated for the table, and FALSE otherwise. Only valid for column store tables.

COMPRESSED_EXTKEY

VARCHAR(5)

Returns TRUE if the table uses a compressed external key, and FALSE otherwise. Only valid for column store tables.

HAS_TEXT_FIELDS

VARCHAR(5)

Returns TRUE if the table has at least one column of type TEXT, and FALSE otherwise. Only valid for column store tables.

USES_QUEUE_TABLE

VARCHAR(5)

Returns TRUE if the table uses a queue table, and FALSE otherwise. Only valid for column store tables.

IS_PRELOAD

VARCHAR(5)

Returns TRUE if the table uses preloading, and FALSE otherwise. Only valid for column store tables. Preload information can be NULL in case of row store, virtual columns or tables on extended storage.

IS_PARTIAL_PRELOAD

VARCHAR(5)

Returns TRUE if the table uses partial preloading, and FALSE otherwise. Only valid for column store tables. Preload information can be NULL in the case of row store, virtual columns, or tables on extended storage.

UNLOAD_PRIORITY

TINYINT

Specifies the unload priority of the table: 0 - 9. 0 means not unloadable, 1 means latest unload, and 9 means earliest unload.

HAS_SCHEMA_FLEXIBILITY

VARCHAR(5)

Returns TRUE if the table has schema flexibility, and FALSE otherwise. Only valid for column store tables.

IS_REPLICA

VARCHAR(5)

Returns TRUE if the table is a replica, and FALSE otherwise.

HAS_STRUCTURED_PRIVILEGE_CHECK

VARCHAR(5)

Returns TRUE if the table is registered for a structured privilege check, and FALSE otherwise.

IS_SERIES_TABLE

VARCHAR(5)

Returns TRUE if the table is a series table, and FALSE otherwise.

ROW_ORDER_TYPE

VARCHAR(8)

Returns the row order type. Can be either NULL or BY VALUE.

CREATE_TIME

TIMESTAMP

Specifies the creation time of the table.
