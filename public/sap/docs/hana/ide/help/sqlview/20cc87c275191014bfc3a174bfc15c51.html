PROCEDURES System View

Provides information about available stored procedures.

Structure

Column name

Data type

Description

SCHEMA_NAME

NVARCHAR(256)

Specifies the schema name of the stored procedure

PROCEDURE_NAME

NVARCHAR(256)

Specifies the name of the stored procedure

PROCEDURE_OID

BIGINT

Specifies the object ID of the stored procedure

SQL_SECURITY

VARCHAR(7)

Specifies the SQL security setting of the stored procedure: DEFINER or INVOKER

DEFAULT_SCHEMA_NAME

NVARCHAR(256)

Specifies the schema name of the unqualified objects in the procedure

INPUT_PARAMETER_COUNT

INTEGER

Specifies the input type parameter count

OUTPUT_PARAMETER_COUNT

INTEGER

Specifies the output type parameter count

INOUT_PARAMETER_COUNT

INTEGER

Specifies the in/out type parameter count

RESULT_SET_COUNT

INTEGER

Specifies the result set count

IS_UNICODE

VARCHAR(5)

Specifies whether the stored procedure contains Unicode: TRUE or FALSE

DEFINITION

NCLOB

Specifies the query string of the stored procedure

PROCEDURE_TYPE

VARCHAR(10)

Specifies the type of the stored procedure

READ_ONLY

VARCHAR(5)

Specifies whether the procedure is read only or not: TRUE or FALSE

IS_VALID

VARCHAR(5)

Specifies whether the procedure is valid or not. This becomes FALSE when its base objects are changed or dropped: TRUE or FALSE

IS_HEADER_ONLY

VARCHAR(5)

Specifies whether the procedure is a header only procedure or not: TRUE or FALSE

HAS_TRANSACTION_CONTROL_STATEMENTS

VARCHAR(5)

Specifies whether the procedure has transaction control statements or not: TRUE or FALSE

OWNER_NAME

NVARCHAR(256)

Specifies the name of the owner of the stored procedure

CREATE_TIME

TIMESTAMP

Specifies the creation time of the procedure
