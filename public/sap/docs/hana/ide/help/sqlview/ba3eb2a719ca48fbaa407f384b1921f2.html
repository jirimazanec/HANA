M_XS_SESSIONS System View

Provides information about sessions running in SAP HANA XS.

Structure
Column name Data type Description
HOST VARCHAR(64) The host name
PORT INTEGER The internal port
XS_SESSION_ID BIGINT The identifier of XS session running on this host
USER_NAME NVARCHAR(256) The user name
REQUEST_COUNT INTEGER The number of HTTP requests that invoked this XS session
AUTHENTICATION_METHOD VARCHAR(48) The method used for authenticating this XS session
IS_DEBUGGED VARCHAR(5) The value is TRUE if a debugger is currently attached to this XS session, and FALSE otherwise
REFERENCE_COUNT INTEGER The current number of references to this XS session
START_TIME TIMESTAMP The point in time when this XS session started
LAST_REQUEST_TIME TIMESTAMP The point in time of last activity in the XS session
