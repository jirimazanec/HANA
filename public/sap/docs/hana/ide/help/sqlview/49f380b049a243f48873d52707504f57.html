JWT_USER_MAPPINGS System View

Lists all of the user-JWT mappings configured in the SAP HANA database.

Structure

Column name

Data type

Description

USER_NAME

NVARCHAR(256)

The SAP HANA user name

JWT_PROVIDER_NAME

NVARCHAR(256)

The name of the JWT provider

EXTERNAL_IDENTITY

NVARCHAR(256)

The name of the user known to the JWT provider
