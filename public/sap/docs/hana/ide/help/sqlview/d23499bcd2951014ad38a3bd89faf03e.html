ST_SPATIAL_REFERENCE_SYSTEMS System View

Provides information about spatial reference systems.

Structure

Column name

Data type

Description

OWNER_NAME

NVARCHAR(256)

The owner name of this spatial reference system (SRS)

SRS_ID

INTEGER

The numeric identifier (SRID) of this spatial reference system

SRS_NAME

NVARCHAR(256)

The name of this spatial reference system

ROUND_EARTH

VARCHAR(7)

Indicates whether the SRS type is ROUND EARTH (TRUE) or PLANAR (FALSE)

AXIS_ORDER

VARCHAR(12)

Describes how the database server interprets points with regards to latitude and longitude: x/y/z/m, long/lat/z/m, lat/long/z/m

SNAP_TO_GRID

DOUBLE

Defines the size of the grid when performing calculations

TOLERANCE

DOUBLE

Defines the precision to use when comparing points

SEMI_MAJOR_AXIS

DOUBLE

The distance from the center of the ellipsoid to the equator for a ROUND EARTH SRS

SEMI_MINOR_AXIS

DOUBLE

The distance from the center of the ellipsoid to the poles for a ROUND EARTH SRS

INV_FLATTENING

DOUBLE

The inverse flattening used for the ellipsoid in a ROUND EARTH SRS

MIN_X

DOUBLE

The minimum x value allowed in coordinates

MAX_X

DOUBLE

The maximum x value allowed in coordinates

MIN_Y

DOUBLE

The minimum y value allowed in coordinates

MAX_Y

DOUBLE

The maximum y value allowed in coordinates

MIN_Z

DOUBLE

The minimum z value allowed in coordinates

MAX_Z

DOUBLE

Maximum z value allowed in coordinates

ORGANIZATION

NVARCHAR(256)

Name of the organization that created the coordinate system used by this spatial reference system

ORGANIZATION_COORDSYS_ID

INTEGER

The ID given to the coordinate system by the organization that created it

SRS_TYPE

VARCHAR(11)

The type of SRS as defined by the SQL/MM standard: GEOGRAPHIC, PROJECTED, ENGINEERING

LINEAR_UNIT_OF_MEASURE

NVARCHAR(256)

The linear unit of measure used by this spatial reference system

ANGULAR_UNIT_OF_MEASURE

NVARCHAR(256)

The angular unit of measure used by this spatial reference system

POLYGON_FORMAT

VARCHAR(16)

The orientation of the rings in a polygon: COUNTERCLOCKWISE, CLOCKWISE, EVENODD

STORAGE_FORMAT

VARCHAR(8)

Indicates whether the data is stored in normalized format (INTERNAL), unnormalized format (ORIGINAL), or both (MIXED)

DEFINITION

NVARCHAR(256)

The WKT definition of this spatial reference system in the format defined by the OGC standard

TRANSFORM_DEFINITION

NVARCHAR(256)

The transform definition settings for use when transforming data from this SRS to another
