M_OBJECT_LOCK_STATISTICS System View

Lock contention statistics - lock wait count, wait time, failed count for each objects.

Structure

This view has a resettable counterpart. This means, you can also see the values since the last reset in the dependent view M_OBJECT_LOCK_STATISTICS_RESET. To reset the values, use the SQL command: ALTER SYSTEM RESET MONITORING VIEW SYS.M_OBJECT_LOCK_STATISTICS_RESET or the management console command: monitor reset -n M_OBJECT_LOCK_STATISTICS_RESET.
