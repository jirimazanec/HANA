M_TRANSACTIONS System View

Provides information about all transactions created by users or the database.

Structure

Column name

Data type

Unit

Description

HOST

VARCHAR(64)

  

The host name

PORT

INTEGER

  

The internal port

CONNECTION_ID

INTEGER

  

The connection ID

TRANSACTION_ID

INTEGER

  

The transaction object ID (this number is reused after a transaction is closed)

VOLUME_ID

INTEGER

  

The volume ID where the transaction has started

PRIMARY_TRANSACTION_ID

INTEGER

  

The primary transaction object ID for the external transaction

TRANSACTION_SEQUENCE

INTEGER

  

The transaction sequence number in a transaction

UPDATE_TRANSACTION_ID

BIGINT

  

The write transaction ID (this number is ever increasing)

TRANSACTION_STATUS

VARCHAR(128)

  

The transaction status; INACTIVE, ACTIVE, PRECOMMITTED, ABORTING, PARTIAL_ABORTING, or ACTIVE_PREPARE_COMMIT

TRANSACTION_TYPE

VARCHAR(128)

  

The transaction type; USER, VERSION GARBAGE COLLECTION, DDL VERSION GARBAGE COLLECTION, INTERNAL, or EXTERNAL

START_TIME

TIMESTAMP

  

The start time

END_TIME

TIMESTAMP

  

The end time

START_MVCC_TIMESTAMP

BIGINT

  

The internal MVCC timestamp of the transaction start time

END_MVCC_TIMESTAMP

BIGINT

  

The internal MVCC timestamp of the transaction end time

EXECUTED_STATEMENT_COUNT

INTEGER

  

The number of executed statements in the transaction

CREATED_VERSION_COUNT

INTEGER

  

The number of created versions in the transaction

ALLOCATED_VERSION_SIZE

BIGINT

Byte

INTEGER

  

The number of acquired locks in the transaction

LOCK_WAIT_COUNT

INTEGER

  

The number of lock waits in the transaction

DOUBLE

Second

Byte

The current statement ID

CURRENT_STATEMENT_SEQUENCE

INTEGER

  

The sequence number of the currently executed statement

ALLOCATED_MEMORY_SIZE

BIGINT

Byte

The memory size occupied by the transaction

ACQUIRED_METALOCK_INDEX

INTEGER

  

The index of the acquired metalock

LOG_PARTITION_ID

SMALLINT

  

The log partition ID of the transaction

REDO_LOG_AMOUNT

BIGINT

  

The size of the redo log amount generated by the transaction

UNDO_LOG_AMOUNT

BIGINT

  

The size of the undo log amount generated by the transaction

MIN_MVCC_SNAPSHOT_TIMESTAMP

BIGINT

  

The logical timestamp of the first executed statement in the transaction

LAST_COMMIT_ID

BIGINT

  

The last commit ID of the transaction

ACTIVE_STATEMENT_COUNT

INTEGER

  

The number of opened cursors in the transaction

ISOLATION_LEVEL

The isolation level of each transaction: READ UNCOMMITTED, READ COMMITTED, REPEATABLE READ, or SERIALIZABLE

VARCHAR(5)

VARCHAR(5)
